# Beyond Formulas

《数学基础笔记》是一份面向数据科学、机器学习及相关方向的中文数学笔记。它关心的不只是怎样使用公式，也关心公式为什么成立、什么时候失效，以及计算结果是否可信。

## 内容

笔记从数学语言出发，逐步覆盖微积分、线性代数、离散数学、概率论、随机过程、信息论、数值分析、凸优化、矩阵微积分、数理统计与实分析支撑，并延伸到动力系统、控制、强化学习和综合应用。

写作遵循几个基本原则：

- 先说明问题与动机，再引入概念；
- 用反例和认知冲突暴露直觉的边界；
- 明确结论成立所需的条件及其失效情形；
- 把数学工具放回建模、计算和判断的完整过程。

## 编译

项目使用 `ctexbook`，推荐通过 XeLaTeX 编译：

```powershell
xelatex main.tex
xelatex main.tex
```

需要连续编译两次，以稳定目录和交叉引用。完整 TeX 发行版需包含 CTeX、AMS 数学宏包、`longtable`、`enumitem`、`tocloft` 和 `hyperref` 等常用组件。

## 目录结构

```text
main.tex       全书入口
tex/           正文、单元导读与全局数学宏
tex/macros.tex 常用数学符号宏
```

各学科在 `tex/` 下按 Part 组织；每个内容单元包含一个 `00-MOC.tex` 导读文件及若干独立小节。学习顺序可从书中的“学习路线”开始选择。
