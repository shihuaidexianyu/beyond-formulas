# Beyond Formulas

《Beyond Formulas》是一套以数学为主轴、面向建模、计算、机器学习与决策的长期知识框架。它关心的不只是怎样使用公式，也关心公式为什么成立、什么时候失效、计算结果是否可信，以及不同学科怎样连接成可以持续扩展的体系。

它不宣称穷尽未来可能需要的全部知识，而是提供一张稳定的结构图：新概念应当放在哪里，它依赖哪些前提，解决什么问题，又会在哪些边界上失效。数学语言、概率统计、数值计算、优化、机器学习、深度学习、动力系统、控制与强化学习因此不是彼此孤立的课程，而是同一套判断能力的不同层次。

## 内容

笔记从数学语言出发，逐步覆盖微积分、线性代数、离散数学、概率论、随机过程、信息论、数值分析、凸优化、矩阵微积分、数理统计与实分析支撑，并延伸到机器学习、深度学习、动力系统、控制和强化学习。

写作遵循几个基本原则：

- 先说明问题与动机，再引入概念；
- 用反例和认知冲突暴露直觉的边界；
- 明确结论成立所需的条件及其失效情形；
- 把数学工具放回建模、计算和判断的完整过程。

## 编译

项目使用 `ctexbook`，推荐通过 XeLaTeX 编译：

```powershell
xelatex main.tex
xelatex main.tex
```

需要连续编译两次，以稳定目录和交叉引用。完整 TeX 发行版需包含 CTeX、AMS 数学宏包、`longtable`、`enumitem`、`tocloft` 和 `hyperref` 等常用组件。

## 目录结构

```text
main.tex       全书入口
tex/           正文、单元导读与全局数学宏
tex/macros.tex 常用数学符号宏
reference-materials/ 可选的本地参考资料库（由 .gitignore 排除）
```

各学科在 `tex/` 下按 Part 组织；每个内容单元包含一个 `00-MOC.tex` 导读文件及若干独立小节。学习顺序可从书中的“学习路线”开始选择。参考原件若保存在本地，应统一放入 `reference-materials/`，避免散落在项目根目录，也不会被提交到仓库。
