\section{数据科学的数学基础}

数据科学中的数学问题大致来自四层。第一层是怎样准确表达对象和论证；第二层是怎样描述变化、空间与不确定性；第三层是怎样从有限数据求解、估计和验证；最后一层才是把这些工具组合成模型。各学科因此不是并排的孤岛，而是在回答同一个问题的不同部分。

第一次使用这套笔记，可以先读``怎样选择学习路线''。它给出完整基础、机器学习、概率统计和定向查缺四种进入方式，不要求每位读者按同一顺序从头读到尾。

\subsection{表达、证明与极限}

数学语言处理命题、集合、函数、量词和证明。它的作用不是增加术语，而是让``对所有''\,``存在''\,``如果\ldots\ldots 那么\ldots\ldots''不再含混。

实分析支撑解释完备性、收敛、连续性与极限交换。它适合与微积分交错学习：先凭直觉使用极限，再回来理解哪些条件让直觉可靠。

微积分研究连续变化与累积，离散数学研究计数、递推、图和整数结构。二者分别提供连续与离散世界的基本推理方式。

\subsection{空间、随机性与信息}

线性代数把方程、几何、变换和数据矩阵放进同一空间语言。最小二乘、PCA、Markov 链和神经网络都在反复使用其中的子空间、谱与分解。

概率论从随机试验进入条件化、期望和极限定理；随机过程让随机变量沿时间相互依赖；信息论则用 entropy、KL 与 mutual information 衡量不确定性、差异和可传递信息。

数理统计把概率模型用于有限数据，区分估计对象、抽样波动、置信陈述、检验错误与模型选择。

\subsection{计算、优化与学习}

数值分析研究实数公式进入有限精度计算后发生什么，重点是误差、conditioning、稳定性和可验证算法。

凸优化解释怎样把目标与约束写成可求解问题，何时局部条件已经足够，何时对偶变量揭示了约束代价。

矩阵微积分把微分、梯度、Jacobian、Hessian 和反向传播统一到线性近似与伴随映射上。

综合应用把这些工具放回完整建模过程：表示、概率假设、损失、优化、数值计算、统计评估和部署边界必须同时成立。

\subsection{按当前问题直接进入}

\begin{itemize}
\tightlist
\item
  看不懂量词、证明或定义边界：从数学语言开始。
\item
  极限、级数、微分或积分缺乏直觉：进入微积分，并在需要严格性时回到实分析。
\item
  矩阵公式很多却看不见几何：从线性代数的方程、子空间与正交性开始。
\item
  条件概率、期望或统计推断混在一起：先学概率论，再进入数理统计。
\item
  算法有公式却出现发散、NaN 或不稳定：进入数值分析。
\item
  想理解正则化、对偶或梯度算法：进入凸优化与矩阵微积分。
\item
  想把数学真正用于回归、隐变量、采样或神经网络：进入综合应用，但遇到缺口时沿链接返回基础章节。
\end{itemize}

课程、教材和公开讲义的统一入口在``参考资料''部分。
