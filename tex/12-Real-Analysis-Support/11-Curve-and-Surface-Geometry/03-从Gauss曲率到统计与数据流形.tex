\section{从 Gauss 曲率到统计与数据流形}
\label{sec:12-Real-Analysis-Support/11-Curve-and-Surface-Geometry/03}

\begin{quote}
\textbf{先抓住这一点：}Gauss 曲率虽可由主曲率相乘得到，却完全能从曲面内部度量恢复。迁移到统计与数据流形时，首先要说明采用的度量和流形假设，而不是只借用“曲率”一词。
\end{quote}

经典微分几何的终点恰好是本书两条主线的起点：Fisher 度量让参数族成为黎曼流形，流形假设让数据成为流形附近的点云。两者都可以问同一个几何问题：这个流形弯不弯，曲率告诉我们什么。

\subsection{统计流形的曲率：可区分性的全局行为}

信息几何单元（Part 12）把 Fisher 信息提升为黎曼度量，自然梯度成为这个几何里的最速下降。度量一旦给定，曲率随之确定。对一维指数族，Fisher 度量 \(g=I(\theta)\) 的 Gauss 曲率可以用本节公式直接计算；对多元指数族，对偶联络（Amari 的 \(\alpha\)-联络）给出比 Levi-Civita 联络更贴切的曲率概念。

曲率的统计含义是：Fisher 度量局部告诉``哪个方向难区分''，曲率告诉``这种区分结构能否全局拉平''。指数族的统计流形在对偶联络下是平坦的（曲率为零）——这正是 Legendre 对偶、自然参数与期望参数两套``平直坐标''存在的几何原因。非指数族（如混合模型）曲率非零：不存在全局平直坐标，局部线性化（Fisher 度量看成常数）的误差随范围增长。这解释了为什么混合模型的似然面有多个局部极大、为什么 EM 的收敛依赖初值——不只是算法问题，也是几何问题。

\subsection{数据流形的曲率：局部线性近似的误差}

流形假设（Part 13）说数据躺在低维流形附近，降维方法在切空间中做局部线性近似。近似的误差由曲率控制：切空间是流形的一阶近似，二阶偏差正是形状算子的大小。具体地说，点 \(p\) 处沿切方向 \(w\) 偏离切空间的距离，首项是 \(\frac12 II(w,w)t^2\)。

这给出三条可操作的分界线：

\begin{itemize}
\tightlist
\item
  \textbf{邻域半径}：局部 PCA、LLE 的邻域必须小于曲率半径 \(1/\kappa\) 量级，否则切空间近似失真。高曲率区域需要更小的邻域、更多的样本。
\item
  \textbf{测地距离估计}：Isomap 用邻接图最短路近似测地线。曲率大处，图最短路偏离测地线最多（弦与弧的差距随曲率增长）——采样密度必须随曲率自适应。
\item
  \textbf{内蕴维数与外在维数}：曲率非零的流形不能全局嵌入与其维数相同的欧氏空间（\(K\ne0\) 曲面不能摊平）。数据流形若内蕴弯曲，任何保持内蕴距离的降维都必然有失真——这与``一张地图画不下整个地球''是同一个定理。
\end{itemize}

\subsection{一个可算的完整例子：单位球面}

把本节与上节的概念在 \(S^2\) 上算一遍，作为参照系：

\begin{itemize}
\tightlist
\item
  第一基本形式 \(I=\diag(1,\sin^2\theta)\)（取 \(R=1\)）；
\item
  第二基本形式 \(II=-I\)，形状算子是 \(-\mathrm{id}\)，主曲率 \(\kappa_1=\kappa_2=-1\)；
\item
  Gauss 曲率 \(K=1\) 处处为正，平均曲率 \(H=-1\)；
\item
  测地线是大圆，两条经线在极点相交——正曲率让起初平行的测地线汇聚；负曲率（马鞍面）则让它们发散。
\end{itemize}

测地线的汇聚与发散是曲率最直观的几何效应，也是 Jacobi 场与截面曲率（黎曼几何的下一步）要精确化的现象。本书停在这里：知道``正曲率汇聚、负曲率发散、零曲率平行''已经足够解读统计与数据流形中的局部几何。

\subsection{边界清单}

\begin{itemize}
\tightlist
\item
  曲率控制的是二阶效应。一阶上切空间总是好的；问题只在于邻域多大时二阶项不可忽略。
\item
  Theorema Egregium 说的是 Gauss 曲率内蕴，不是说外在形状无关。平均曲率、主曲率的方向仍依赖嵌入；最小曲面与等温参数化是外在问题。
\item
  统计流形的曲率依赖联络的选择：Levi-Civita 联络与 \(\alpha\)-联络给出不同曲率，统计意义也不同。说``统计流形是弯的''必须指明在哪个联络下。
\item
  数据流形的曲率估计对噪声敏感：二阶量比一阶量（切空间）需要多一个数量级的样本。低样本时先假设低曲率，比强行估计更稳健。
\end{itemize}

\subsection{本单元的落点}

经典微分几何给本书留下三件工具：曲率是``局部线性近似误差''的度量（数据流形），Theorema Egregium 是``内蕴结构不依赖外在表示''的范例（统计流形与流形假设的共同前提），测地线汇聚发散是曲率最直接的可观测效应。联络、Jacobi 场与示性类属于黎曼几何本身；本书需要的几何到此为止——它足够让``这个流形是弯的''从比喻变成可计算的句子。
