\section{条件期望与线性回归是两种投影}
\label{sec:12-Real-Analysis-Support/07-Lp-Spaces-and-Function-Geometry/04}

\begin{quote}
\textbf{先抓住这一点：}条件期望投影到所有由给定信息可测的平方可积函数上，线性回归只投影到特征的线性张成空间。前者的投影空间通常更大。
\end{quote}

``条件期望是对信息 \(\sigma\)-代数的投影''曾用``在每个信息事件上积分匹配''定义条件期望。在平方可积的范围内，这个定义还有一个更直观的几何身份：条件期望就是 \(L^2\) 中的正交投影。同时，最小二乘线性回归也是投影，但投到的是另一个小得多的子空间。把两者放在同一幅图里，``预测''的两个层次就分开了。

\subsection{\texorpdfstring{条件期望是向 \(\mathcal G\) 可测函数的投影}{条件期望是向 G 可测函数的投影}}

设 \(X\in L^2(\Omega,\mathcal F,P)\)，\(\mathcal G\subseteq\mathcal F\) 是子 \(\sigma\)-代数。所有 \(\mathcal G\)-可测且平方可积的随机变量组成 \(L^2\) 的闭线性子空间，记为 \(L^2(\mathcal G)\)。把 \(X\) 向它投影，得到唯一的 \(\E[X\mid\mathcal G]\)，满足

\[
\E\bigl[(X-\E[X\mid\mathcal G])Z\bigr]=0
\qquad\text{对所有 }Z\in L^2(\mathcal G).
\]

取 \(Z=\mathbf1_G\)（\(G\in\mathcal G\)），上式正是条件期望的定义性质

\[
\int_G X\,dP=\int_G\E[X\mid\mathcal G]\,dP.
\]

两个定义在 \(L^2\) 中完全等价：积分匹配是逐事件的说法，正交性是几何的说法。投影身份立刻白送三条性质：

\begin{itemize}
\tightlist
\item
  \textbf{最佳均方预测}：在所有能用信息 \(\mathcal G\) 表达的预测中，条件期望使 \(\E[(X-\widehat X)^2]\) 最小——``知道 \(\mathcal G\) 后对 \(X\) 的最好猜测''有了唯一答案；
\item
  \textbf{塔式性质}：若 \(\mathcal H\subseteq\mathcal G\)，向更小空间的投影可以由两级投影复合：\(\E[\E[X\mid\mathcal G]\mid\mathcal H]=\E[X\mid\mathcal H]\)——先粗后细的投影次序无关，这是嵌套子空间的直接性质；
\item
  \textbf{正交分解}：\(X=\E[X\mid\mathcal G]+(X-\E[X\mid\mathcal G])\) 把随机变量拆成``信息能解释的部分''与``与信息正交的残差''，方差相应分解——这正是方差分析（ANOVA）与``已解释/未解释方差''的几何原型。
\end{itemize}

\subsection{线性回归投到的是小得多的子空间}

条件期望 \(\E[Y\mid X]\) 是\(X\) 的所有可测函数中的最佳均方预测；线性回归只允许仿射函数。设 \((X,Y)\) 平方可积，考虑子空间

\[
M=\{a+bX:a,b\in\R\}.
\]

向 \(M\) 投影 \(Y\)，正交条件取 \(Z=1\) 与 \(Z=X\) 两个基，得到正规方程

\[
\E[Y]=a+b\E X,
\qquad
\E[XY]=a\E X+b\E[X^2],
\]

解出

\[
b=\frac{\Cov(X,Y)}{\Var(X)},
\qquad
a=\E Y-b\E X.
\]

这就是一元最小二乘的斜率与截距——\hyperref[sec:13-Machine-Learning/02-Linear-Models/01]{``线性回归从投影到概率模型''}中由列空间几何得到的同一对量，现在出现在 Hilbert 空间投影中。

两种投影的关系一目了然：

\[
\{\text{仿射函数}\}
\subseteq
\{\text{可测函数}\},
\]

所以条件期望的均方误差不会超过线性回归。线性回归是``在直线里找最好''，条件期望是``在所有函数里找最好''；后者的曲线可以任意弯曲，前者只能是一条直线。

\subsection{Gaussian 情形两个投影重合}

若 \((X,Y)\) 联合 Gaussian，条件期望恰好是仿射函数：

\[
\E[Y\mid X]
=\E Y+\frac{\Cov(X,Y)}{\Var(X)}(X-\E X).
\]

验证只需代入联合密度（或利用联合 Gaussian 的条件分布公式）。此时最佳均方预测与最佳线性预测相同——不是线性方法变强了，而是 Gaussian 世界里所有条件结构本来就是线性的。这解释了为什么``概率论''与``数理统计''中，相关与回归在 Gaussian 假设下配合得天衣无缝，也提示了外推的边界：离开 Gaussian 族，条件期望一般是曲线，线性投影只在它自己的子空间里最优。

\subsection{\texorpdfstring{投影视角统一了\(L^2\) 统计的半壁江山}{投影视角统一了 L2 统计的半壁江山}}

\begin{itemize}
\tightlist
\item
  条件期望：向 \(\mathcal G\)-可测函数投影；
\item
  线性回归：向 \(\{1,X\}\) 张成的空间投影；
\item
  多元回归：向设计矩阵列空间投影（Part 03 与 Part 13 的有限维版本）；
\item
  Fourier 逼近：向三角多项式空间投影（上一节）；
\item
  方差分解与 ANOVA：投影的 Pythagoras 定理。
\end{itemize}

它们共用同一个定理：闭子空间上存在唯一正交投影。每换一个子空间，就换一种``最好''的含义。选择子空间——选特征、选信息集、选基——才是这些方法的真正建模决定；投影本身只是随后必然发生的事。

\subsection{边界清单}

\begin{itemize}
\tightlist
\item
  条件期望的投影身份要求 \(X\in L^2\)；只有 \(L^1\) 时，定义仍由积分匹配给出，``最佳均方预测''的说法不成立。
\item
  投影要求子空间闭；\(\{1,X\}\) 这类有限维空间自动闭，\(L^2(\mathcal G)\) 的闭性来自 \(L^2\) 极限保持可测性。
\item
  正交是 \(L^2\) 的内积正交，不是独立；零相关只排除线性关系（Gaussian 例外）。
\item
  线性回归的最优性是均方意义下的；绝对误差、分位数等其他损失对应不同几何，不由正交投影描述。
\end{itemize}

函数空间的三层结构至此齐了：度量（\(L^p\) 范数）、完备性（Banach）、内积（Hilbert）。这套语言足够支撑本书后面对象——从 Fourier 分析、RKHS 到条件期望——中所有``把函数当点''的推理。若继续深入泛函分析（对偶空间、算子谱、弱拓扑），这些概念仍是地基，但那已超出本书为概率与学习服务的支撑范围。
