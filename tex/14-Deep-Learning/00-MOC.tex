\section{深度学习：让表示、梯度与数据结构在复合函数中相遇}

深度学习并没有创造一套脱离数学基础的新语言。线性层调用线性映射，反向传播调用链式法则与伴随映射，训练调用随机优化，卷积写入平移结构，注意力用矩阵乘法构造内容相关的加权平均，VAE 则把潜变量模型、变分下界与 Monte Carlo 梯度接到一起。

真正的新困难来自这些工具被放进很深的复合系统：局部上正确的运算可能在深度中放大或消失；数学上等价的表达式可能在浮点数中产生完全不同的结果；网络能够插值训练数据，也不说明它保留了部署所需的结构。

本 Part 按``表示---目标与梯度---训练---结构化架构---生成模型''推进：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  表示与逼近研究非线性和深度改变了什么函数族，并划清通用逼近定理的边界；
\item
  损失与反向传播把统计目标、计算图、VJP 和梯度检查接成一条链；
\item
  优化与训练讨论 mini-batch 噪声、动量、自适应缩放、初始化和数值稳定；
\item
  卷积网络把局部性与平移等变性写入参数共享；
\item
  注意力让每个位置依据内容选择信息来源，同时暴露 mask、轴与二次复杂度；
\item
  变分自编码器用可训练近似后验解决潜变量模型中的不可积 posterior。
\end{enumerate}

阅读时始终区分四类证据：函数类能否表示，梯度是否算对，优化是否到达合适区域，独立数据是否支持泛化。训练曲线只直接回答其中一小部分。

\subsection{本质视角：深度学习的低维本质}

深度学习不只是``多层神经网络''，而是逐层提取数据低维结构的过程。其低维本质体现在：

\begin{itemize}
\tightlist
\item
  \textbf{表示学习}：每一层学习数据的一种表示，深层表示更抽象、更任务相关——这是逐层逼近数据的低维结构；
\item
  \textbf{特征学习}：网络自动学习与任务相关的特征，而非手工设计——这是从数据中发现低维结构；
\item
  \textbf{函数空间视角}：无限宽网络等价于核方法（NTK）或分布演化（mean-field）——这是深度学习的函数空间本质；
\item
  \textbf{泛化的隐式偏置}：过参数化网络倾向于找到简单解（低秩、稀疏、平滑），这是泛化的低维解释。
\end{itemize}

理解这些低维本质，就是理解为什么深度学习有效（它捕捉了数据的低维结构）、什么时候会失效（结构假设不成立或优化失败）、以及失效时应该回到哪里（重新检查数据结构、优化算法或正则化）。
