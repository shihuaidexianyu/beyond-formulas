深度学习并没有创造一套脱离数学基础的新语言。线性层调用线性映射，反向传播调用链式法则与伴随映射，训练调用随机优化，卷积写入平移结构，注意力用矩阵乘法构造内容相关的加权平均，VAE 则把潜变量模型、变分下界与 Monte Carlo 梯度接到一起。

真正的新困难来自这些工具被放进很深的复合系统：局部上正确的运算可能在深度中放大或消失；数学上等价的表达式可能在浮点数中产生完全不同的结果；网络能够插值训练数据，也不说明它保留了部署所需的结构。

本 Part 按``表示---目标与梯度---训练---结构化架构---生成模型''推进：

\begin{enumerate}
\tightlist
\item
  表示与逼近研究非线性和深度改变了什么函数族，并划清通用逼近定理的边界；
\item
  损失与反向传播把统计目标、计算图、VJP 和梯度检查接成一条链；
\item
  优化与训练讨论 mini-batch 噪声、动量、自适应缩放、初始化和数值稳定；
\item
  卷积网络把局部性与平移等变性写入参数共享；
\item
  注意力让每个位置依据内容选择信息来源，同时暴露 mask、轴与二次复杂度；
\item
  变分自编码器用可训练近似后验解决潜变量模型中的不可积 posterior。
\end{enumerate}

阅读时始终区分四类证据：函数类能否表示，梯度是否算对，优化是否到达合适区域，独立数据是否支持泛化。训练曲线只直接回答其中一小部分。

\subsection{怎么读这一章}

深度网络的成功没有一个对所有任务都成立的单一解释。低维结构、局部性、对称性、组合性和优化的隐式偏置都可能发挥作用；它们是需要由数据与实验检验的候选机制，不是从网络层数自动推出的事实。

\begin{itemize}
\tightlist
\item
  \textbf{表示学习}：中间层会形成适合当前目标的表示，但``更深''不自动等于``更抽象''，表示是否有用要由干预、迁移或下游任务检验；
\item
  \textbf{特征学习}：参数更新会改变特征本身；这区别于固定核，但学到的特征可能利用真实结构，也可能利用数据捷径；
\item
  \textbf{函数空间视角}：特定无限宽缩放分别导向 NTK 或 mean-field 极限；这些极限解释某些训练制度，不等同于所有有限宽网络；
\item
  \textbf{隐式偏置}：优化器、初始化和参数化会偏好某些解；``简单''必须由具体范数、边际、秩或函数空间复杂度定义，不能只凭训练成功倒推。
\end{itemize}

这些视角放在最后作为选读，是因为它们需要把理论极限、经验现象和因果解释分开。它们能提出可检验假设，却不能单独证明某个模型为什么在当前数据上有效。
