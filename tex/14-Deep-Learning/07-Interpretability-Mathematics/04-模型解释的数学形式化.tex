\section{模型解释的数学形式化}

模型解释不是``讲故事''，而是数学测量：SHAP、LIME、积分梯度等方法测量模型对输入的依赖，但它们的数学本质不同，适用范围不同，失效模式也不同。理解这些差异，是理解什么时候可以用哪种解释方法、什么时候会失效的关键。

\subsection{SHAP：Shapley 值的博弈论本质}

SHAP（SHapley Additive exPlanations）基于 Shapley 值，来自合作博弈论。把特征看作玩家，模型预测看作收益，Shapley 值测量每个特征对预测的边际贡献：

\[
\phi_i=\sum_{S\subseteq N\setminus\{i\}}\frac{|S|!(n-|S|-1)!}{n!}\left[f(S\cup\{i\})-f(S)\right],
\]

其中 $N$ 是所有特征，$S$ 是特征子集，$f(S)$ 是用子集 $S$ 的预测。

SHAP 的数学本质是：特征重要性是平均边际贡献，平均在所有可能的特征子集上。这解释了为什么 SHAP 满足对称性、虚拟性、可加性等公理——这些是 Shapley 值的公理，不是人为选择的。

SHAP 的失效是：计算复杂（需要 $2^n$ 个子集），需要近似；解释依赖参考分布（特征缺失时的填充方式），不同参考给出不同解释。

\subsection{LIME：局部线性逼近}

LIME（Local Interpretable Model-agnostic Explanations）在数据点附近用线性模型逼近复杂模型。具体地，在点 $x$ 附近采样扰动点 $x'$，用简单模型（如线性回归）拟合 $f(x')$，权重按距离衰减。

LIME 的数学本质是：局部 Taylor 展开。在光滑点，复杂模型可以局部线性逼近，LIME 估计这个线性模型的系数。

LIME 的失效是：线性逼近只在光滑点有效，在决策边界附近失效；解释依赖扰动分布，不同扰动给出不同解释；解释可能不稳定（同一点多次运行给出不同结果）。

\subsection{积分梯度：路径积分}

积分梯度（Integrated Gradients）测量特征对预测的贡献，通过从基线 $x_0$ 到输入 $x$ 的路径积分：

\[
\text{IG}_i(x)=(x_i-x_{0,i})\int_0^1\frac{\partial f(x_0+\alpha(x-x_0))}{\partial x_i}d\alpha.
\]

积分梯度的数学本质是：特征重要性是沿路径的梯度积分，而不是点梯度。这解释了点梯度为零（饱和区域）时，积分梯度仍可能非零——它累积了路径上的梯度。

积分梯度的失效是：依赖基线选择，不同基线给出不同解释；路径是直线，不反映数据流形；梯度可能噪声大，需要平滑。

\subsection{解释方法的统一视角}

这些解释方法可以统一为：测量模型对输入的依赖，但测量方式不同：

\begin{itemize}
\tightlist
\item
  SHAP 测量平均边际贡献（博弈论视角）；
\item
  LIME 测量局部线性逼近（Taylor 展开视角）；
\item
  积分梯度测量路径积分（微积分视角）。
\end{itemize}

统一视角揭示了它们的差异：SHAP 全局但计算复杂，LIME 局部但可能不稳定，积分梯度路径依赖但计算高效。选择解释方法，就是选择测量方式，没有普适最优。

\subsection{解释的边界}

模型解释不是万能的。以下情况需要警惕：

\begin{itemize}
\tightlist
\item
  \textbf{解释与真实因果无关}：解释测量相关性，不是因果性（Part 12 的因果结构）；
\item
  \textbf{解释可能误导}：解释可能符合人类直觉，但不反映模型的真实计算；
\item
  \textbf{解释可能被操纵}：对抗样本可以操纵解释，让错误预测看起来合理。
\end{itemize}

解释的失效不是``解释方法失效''，而是``解释与真实计算的差距''。理解失效模式，就是理解解释的边界。

\subsection{落点}

模型解释的数学形式化把``解释''从直觉提升为测量：SHAP 测量平均边际贡献，LIME 测量局部线性逼近，积分梯度测量路径积分。理解这些，就是理解什么时候可以用哪种解释方法、什么时候会失效、以及失效时应该回到哪里。这是从``相信解释''到``理解解释''的关键一步。
