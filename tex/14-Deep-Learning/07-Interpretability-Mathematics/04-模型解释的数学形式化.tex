\section{模型解释的数学形式化}
\label{sec:14-Deep-Learning/07-Interpretability-Mathematics/04}

问``哪个特征重要''还没有定义一个数学问题：重要是相对哪个输出、哪个基线、哪类扰动和哪个数据分布？SHAP、LIME 与积分梯度给出不同定义，因此即使都计算正确，也可能产生不同答案。

\begin{quote}
\textbf{先抓住这一点：}解释方法首先定义一个被测量量，然后才估计它。比较解释前必须先比较问题定义，而不能只比较热力图是否符合直觉。
\end{quote}

\subsection{SHAP 依赖特征缺失的语义}

Shapley 值把特征视为合作博弈中的参与者。给定集合函数 \(v(S)\)，特征 \(i\) 的归因是它加入不同子集时边际贡献的加权平均：

\[
\phi_i
=\sum_{S\subseteq N\setminus\{i\}}
\frac{|S|!(n-|S|-1)!}{n!}
\bigl[v(S\cup\{i\})-v(S)\bigr].
\]

真正困难是定义 \(v(S)\)：模型原本需要全部特征，只提供子集 \(S\) 时其余特征怎样处理？使用边缘分布、条件分布或某个背景样本，会回答不同问题；相关特征下差异尤其明显。

对线性模型，在特征独立并采用相应背景期望等附加条件下，归因可化成 \(w_i(x_i-\E X_i)\)。没有这些条件时，相关结构会重新分配贡献，不能把该公式当作所有线性模型的通式。

\subsection{LIME 拟合的是扰动分布下的局部替身}

LIME 在目标点附近生成扰动样本，并用距离权重拟合简单模型。它解释的是：在选定扰动机制、核宽和替身模型类下，哪个简单函数最接近原模型。

若模型光滑、邻域足够小且采样覆盖合理，线性替身可能接近一阶局部行为；一般情况下它不等同于 Taylor 展开。图像遮挡、离散特征采样或离开数据流形的扰动，都可能让替身主要解释人为生成的样本。

因此报告 LIME 结果时，应同时报告邻域定义、随机性、局部拟合误差和多次运行稳定性。

\subsection{积分梯度把基线差异沿路径分配}

给定基线 \(x_0\) 和到输入 \(x\) 的直线路径，积分梯度定义为

\[
\operatorname{IG}_i(x)
=(x_i-x_{0,i})
\int_0^1
\frac{\partial f(x_0+\alpha(x-x_0))}{\partial x_i}
\,d\alpha.
\]

在适当正则性下，各坐标归因之和等于 \(f(x)-f(x_0)\)。这项完备性说明它怎样分配相对基线的输出差，却没有指定基线是否合理，也没有保证直线路径位于数据支持集附近。

更换基线或路径可能显著改变归因。饱和区中的点梯度接近零时，路径积分能恢复沿途变化，但这不使其自动成为因果贡献。

\subsection{解释质量需要独立评价}

不同解释至少应检查四类性质：

\begin{itemize}
\tightlist
\item 忠实度：解释是否预测了模型在受控扰动下的输出变化；
\item 稳定性：输入和随机种子轻微变化时，解释是否无理由剧烈改变；
\item 敏感性：模型确实依赖的特征被改变时，解释能否响应；
\item 适用范围：结论针对单个样本、局部邻域还是总体分布。
\end{itemize}

删除高归因特征后性能下降也不是万能检验，因为删除会制造分布外输入；更可靠的评估应让扰动尽量符合数据生成约束，并设置已知机制的合成对照。

\subsection{归因不等于因果或机制}

这些方法主要描述给定模型的输入依赖。它们不直接回答改变现实因素会怎样，也不能证明模型内部采用了人类赋予热力图的语义。若目标是因果解释，需要干预假设；若目标是机制理解，需要内部激活或回路的干预证据。

选择解释方法时，先写清被解释输出、参考对象、允许扰动和使用结论的人。数学形式化的价值不是让解释显得客观，而是让不同答案的来源可见、让解释可以被检验和推翻。
