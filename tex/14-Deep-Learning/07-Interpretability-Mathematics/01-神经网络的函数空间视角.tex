\section{神经网络的函数空间视角}
\label{sec:14-Deep-Learning/07-Interpretability-Mathematics/01}

同一个网络函数可以由许多不同参数表示，参数距离也未必对应预测距离。若问题关心网络在输入上的行为，与其只追踪参数，不如研究训练过程中函数 \(f_{\theta_t}\) 怎样变化。NTK 与 mean-field 是两种做到这一点的极限制度。

\begin{quote}
\textbf{先抓住这一点：}无限宽不是一个唯一极限。参数缩放、学习率和训练时间怎样随宽度变化，决定极限更像固定核线性化，还是可移动的神经元分布。
\end{quote}

\subsection{NTK 描述初始化附近的函数线性化}

在参数初始化 \(\theta_0\) 附近，一阶展开给出

\[
f_\theta(x)
\approx f_{\theta_0}(x)
\!+\nabla_\theta f_{\theta_0}(x)^{\trans}(\theta-\theta_0).
\]

相应的神经正切核（neural tangent kernel, NTK）是

\[
K_{\theta_0}(x,x')
=\nabla_\theta f_{\theta_0}(x)^{\trans}
\nabla_\theta f_{\theta_0}(x').
\]

在适当宽度与缩放下，训练期间参数相对初始化移动很小，核也近似不变。平方损失下的函数值动力学因而接近核梯度流。这个结论解释的是特定的近初始化制度，不是说原始有限网络的非凸目标已经变成凸函数。

NTK 基线很有诊断价值：若完整网络与固定 NTK 的预测和学习曲线接近，当前任务可能主要利用初始化附近的线性化；若差异明显，特征移动可能不可忽略。

\subsection{mean-field 追踪神经元分布}

对两层网络采用另一种参数缩放时，可以把每个神经元看成粒子，并用参数经验分布

\[
\rho_m=\frac1m\sum_{i=1}^m\delta_{(a_i,w_i)}
\]

描述整体。宽度增大后，在适当条件下，有限粒子系统趋向分布 \(\rho_t\) 的演化；某些梯度下降极限可写成 Wasserstein 梯度流。

这里的函数由整个位于参数空间的分布产生。分布在训练中移动，因此表示可以变化。它与 NTK 不是``一个允许学习率大、一个要求学习率小''这么简单，而是参数化、缩放和时间极限共同不同。

\subsection{两种极限分别能告诉我们什么}

\begin{itemize}
\tightlist
\item NTK 提供可计算的核基线、局部训练动力学和部分收敛分析；
\item mean-field 把大量神经元的协同变化转成分布动力学；
\item 两者都能揭示某些宽度效应，但都需要明确架构、初始化、缩放和损失；
\item 有限宽、有限数据和实际优化器可能处在两种理想化之外。
\end{itemize}

它们不能单独解释泛化。训练动力学、函数复杂度、数据分布和评价协议仍需分别分析。``极限中可训练''也不等于``有限网络在新分布上可靠''。

\subsection{怎样把理论视角变成检验}

实践中可以比较网络宽度变化后的学习曲线，测量经验 NTK 在训练前后的漂移，比较只训练最后一层、固定 NTK 与完整端到端训练，并检查结论对初始化和学习率缩放是否稳定。

这些对照能判断某个理论极限是否近似当前训练制度。它们比直接宣称``网络本质上是核''或``网络本质上是分布演化''更可靠：函数空间视角的价值在于提出可反驳的近似，而不是提供一个包办所有现象的比喻。
