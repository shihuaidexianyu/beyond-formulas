\section{高维数据的低维结构检验}
\label{sec:14-Deep-Learning/07-Interpretability-Mathematics/03}

``数据位于低维流形附近''经常被用来解释高维学习为何可能，但它首先是一个需要操作化的假设。更容易混淆的是，随机投影能够降维，并不意味着数据本身具有低内在维度。

\begin{quote}
\textbf{先抓住这一点：}低维可以指有限点集可压缩、局部切空间维数较小，或任务只依赖少数方向。三种陈述需要不同证据，不能由一张二维图互相推出。
\end{quote}

\subsection{JL 引理不要求流形假设}

\begin{theorem}[Johnson--Lindenstrauss 引理的含义]
对 \(n\) 个欧氏空间中的点，存在到

\[
k=O\!\left(\varepsilon^{-2}\log n\right)
\]

维空间的映射，使所有点对距离都保持在原距离的 \(1\pm\varepsilon\) 倍内。随机 Gaussian 或适当稀疏投影以高概率实现这种保持。
\end{theorem}

这里的目标维数依赖点数和允许失真，不依赖原始维数，也不假设这些点来自低维流形。JL 说明有限距离表可以被近似保存；它没有识别生成数据的潜变量，也不保证新样本、密度或拓扑结构同样被保存。

因此，随机投影后的模型表现不变，只能说明当前任务对这次压缩较稳健。它不是``数据内在维度等于 \(k\)''的证明。

\subsection{流形假设需要局部和尺度信息}

若数据在某个尺度上接近 \(d_{\mathrm{int}}\) 维光滑流形，那么足够小邻域中的变化应主要落在约 \(d_{\mathrm{int}}\) 个切向方向。常见诊断包括：

\begin{itemize}
\tightlist
\item 局部 PCA：观察邻域 covariance 的谱是否在某个位置稳定衰减；
\item 最近邻尺度：检查邻域半径随样本量变化是否与某个维数模型一致；
\item 相关维数或参与率：用距离计数或谱集中构造有效维数摘要；
\item 多尺度重复：改变邻域大小、度量和预处理，检查估计是否稳定。
\end{itemize}

这些方法估计的是特定尺度、度量和噪声模型下的有效维数。邻域太小会被采样稀疏性支配，太大又会把曲率和不同分支混在一起。

\subsection{一个最小的反例检查}

设信号位于二维平面，但每个坐标都叠加小幅独立高维噪声。很小尺度上，最近邻差异可能主要由噪声决定，局部 PCA 看起来接近高维；较大尺度上，两个信号方向才占主导。问``数据到底几维''没有脱离尺度的唯一答案。

反过来，一条弯曲得很复杂的一维曲线虽然内在维数为一，仍可能需要大量样本才能覆盖，也可能具有高度振荡的分类边界。低维不自动等于容易学习。

\subsection{任务维数与数据维数不同}

预测目标可能只依赖数据变化中的少数方向，即使完整数据分布并不低维。可以通过充分降维、受控特征删除、跨环境稳定性或表示干预，检查哪些方向对任务必要。只看无监督重构方差，可能保留高变异但与任务无关的因素。

深度网络也不会自动发现真实低维结构。卷积和注意力写入的是局部共享、内容读取等结构偏置；训练是否获得低维、可迁移且符合机制的表示，需要独立诊断。

\subsection{可以相信什么}

若多种维数估计在合理尺度上稳定，低维模型在留出数据上保持距离、重构或预测，并且结论对度量和噪声处理不敏感，就获得了支持低维近似的证据。仍应把结论限定为当前数据范围、尺度和任务，而不是宣称世界存在唯一的低维真相。

下一篇讨论模型归因时会遇到同样原则：先定义要解释的对象，再说明估计对哪些扰动稳定；可视化和单一分数只能提供线索。
