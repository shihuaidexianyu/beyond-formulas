\section{高维数据的低维结构检验}

流形假设说：高维数据躺在低维流形上。但这是假设，需要检验。随机投影、JL 引理、以及流形假设的统计检验，提供了验证数据是否真的低维的工具。理解这些工具，是理解什么时候可以用低维方法、什么时候需要高维方法的关键。

\subsection{随机投影：JL 引理}

Part 11 的 JL 引理说：高维数据可以随机投影到低维空间，保持距离结构。具体地，$n$ 个点在 $\R^d$ 中，可以投影到 $\R^k$，其中 $k=O(\varepsilon^{-2}\log n)$，使得所有距离在 $(1\pm\varepsilon)$ 倍内保持。

JL 引理的关键是：投影维度 $k$ 只依赖点数 $n$，不依赖原始维度 $d$。这意味着高维数据可以在低维空间中处理，只要关心的是距离结构，不是具体坐标。

JL 引理解释了为什么随机投影有效：不是``压缩信息''，而是``保持距离''。高维数据的距离结构可以用低维表示捕捉，这是许多降维方法（随机投影、局部敏感哈希）的理论基础。

\subsection{流形假设的统计检验}

流形假设说：数据躺在低维流形 $M$ 上。检验这个假设的统计方法包括：

\begin{itemize}
\tightlist
\item
  \textbf{局部 PCA}：在数据点邻域内做 PCA，若大部分方差由少数主成分解释，则局部维度低；
\item
  \textbf{最近邻距离}：若数据低维，最近邻距离随样本量增加而快速减小（$k$-近邻距离 $\sim n^{-1/d_{\text{int}}}$，其中 $d_{\text{int}}$ 是内在维度）；
\item
  \textbf{相关维度}：Grassberger-Procaccia 算法估计吸引子的维度，通过点对距离的对数-对数图的斜率。
\end{itemize}

这些检验不是证明，而是证据：它们提供数据低维的统计证据，但不排除高维噪声或局部高维结构。

\subsection{低维结构的失效}

低维结构检验可能失效：

\begin{itemize}
\tightlist
\item
  \textbf{高维噪声}：数据是低维信号加高维噪声，局部 PCA 可能被噪声主导；
\item
  \textbf{多尺度结构}：数据在不同尺度上有不同维度（如分形），单一维度估计失效；
\item
  \textbf{非流形结构}：数据有奇异点、自交、或多个分支，流形假设不成立。
\end{itemize}

失效不是``检验失效''，而是``假设失效''。理解失效模式，就是理解低维方法的边界。

\subsection{低维结构在深度学习中的角色}

深度学习自动利用低维结构：卷积网络利用图像的局部结构（平移不变性），注意力网络利用序列的长程依赖。这些结构不是预先固定的，而是从数据中学习的。

理解低维结构检验，就是理解深度学习什么时候有效：若数据真的有低维结构，深度学习可以学习这个结构；若数据没有低维结构，深度学习可能没有优势，甚至不如简单方法。

\subsection{落点}

高维数据的低维结构检验提供了验证流形假设的工具：随机投影保持距离，局部 PCA 估计局部维度，最近邻距离估计内在维度。理解这些，就是理解什么时候可以用低维方法、什么时候需要高维方法。这是从``假设数据低维''到``检验数据是否真的低维''的关键一步。
