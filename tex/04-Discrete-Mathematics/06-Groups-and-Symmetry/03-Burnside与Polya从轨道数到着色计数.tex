\section{Burnside 与 Pólya：从轨道数到着色计数}

\subsection{直接数图案会重复计算对称版本}

给圆环上的珠子着色时，把整条项链旋转后得到的图案通常不算新品种。先生成全部着色再凭眼睛去重，很快会失控。群作用已经给出正确对象：所有着色组成集合 \(X\)，允许的旋转或翻折组成群 \(G\)，真正不同的图案是轨道 \(X/G\)。

对 \(g\in G\)，记

\[
\operatorname{Fix}(g)=\set{x\in X:g\cdot x=x}.
\]

Burnside 引理给出轨道数

\[
|X/G|=\frac1{|G|}\sum_{g\in G}|\operatorname{Fix}(g)|.
\]

它看似在平均“不动着色数”，实质是双重计数集合 \(\set{(g,x):g\cdot x=x}\)：先固定 \(g\) 得右侧求和，先固定 \(x\) 则每个轨道总共贡献 \(|G|\)。

\subsection{四颗双色珠子的完整计数}

只把旋转视为相同，四个位置由循环群 \(C_4\) 作用。恒等旋转固定全部 \(2^4=16\) 种着色；旋转 \(90^\circ\) 或 \(270^\circ\) 要求四颗同色，各固定 \(2\) 种；旋转 \(180^\circ\) 把位置分成两个二循环，固定 \(2^2=4\) 种。因此

\[
|X/C_4|=\frac{16+2+4+2}{4}=6.
\]

若还允许翻折，作用群应换成二面体群。是否把镜像当作同一种，不是计算细节，而是“什么算相同”的建模决定。

\subsection{循环结构让固定点数可以机械读取}

一个位置置换若分成 \(c_k(g)\) 个长度 \(k\) 的循环，要被 \(g\) 固定，同一循环上的位置必须同色。Pólya 方法先把群的循环结构汇总为 cycle index

\[
Z_G(a_1,a_2,\ldots)
=\frac1{|G|}\sum_{g\in G}
\prod_{k\ge1}a_k^{c_k(g)}.
\]

若有 \(q\) 种颜色且只数总数，代入 \(a_k=q\) 即得轨道数。若还要追踪每种颜色用了多少次，可把 \(a_k\) 换成颜色权重的 \(k\) 次幂之和，从而得到带系数信息的生成函数。

四位置旋转群的 cycle index 为

\[
Z_{C_4}=\frac14\left(a_1^4+a_2^2+2a_4\right),
\]

代入 \(a_k=2\) 正好得到 \(6\)。

\subsection{公式不会替我们选择对称群}

瓷砖纹样、分子异构体和重复排班都可能使用 Pólya 计数，但允许的操作不同：有的允许旋转不允许翻面，有的标签本身不可交换，有的颜色库存受限。群选错，后面的平均即使算得精确也回答了另一个问题。Burnside 与 Pólya 的价值不只是少算几步，而是把“忽略哪些差异”写成一个可审查的群作用。
