最优控制把动力学与代价都当成已知；强化学习拿走这份奢侈。智能体只能反复观察当前状态、选择动作、收到奖励并到达下一状态，再从这些局部交互中学会一套长期策略。困难不只是``没有公式''：当前动作会改变未来能看到什么数据，学习者既是统计者，也是数据生成过程的一部分。

本单元沿着“先把问题写对，再把方程变成算法，再处理函数逼近与分布错配，最后让策略本身可微”的顺序展开：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  MDP 把决策问题写成随机过程：用状态、动作、转移、奖励与折扣率刻画交互，说明 Markov 性质究竟要求状态保留什么，并区分策略的表现与单步奖励。
\item
  Bellman 方程与值迭代、策略迭代：从最优策略尾部仍最优推出递归，用压缩映射解释值迭代为何收敛，再把策略评估与策略改进拆成可交替执行的两步。
\item
  Monte Carlo 与随机逼近：把完整回报看成价值的随机样本，从样本平均进入增量更新、Robbins--Monro 步长与 Dvoretzky 型收敛机制，再讨论持续探索。
\item
  从动态规划到 Q 学习：用一次样本转移替代 Bellman 期望，比较 Sarsa、Expected Sarsa 与表格 Q-learning，并说明其收敛条件为何不能直接搬到函数空间。
\item
  线性价值函数逼近：从加权价值回归进入 semi-gradient TD，推导期望更新与投影 Bellman 方程，区分价值误差、Bellman 误差与投影固定点。
\item
  Deadly Triad：用二维线性例子展示函数逼近、bootstrap 与 off-policy 数据如何让期望更新本身发散。
\item
  DQN 的稳定化机制：分别解释经验回放、目标网络、Double DQN 与稳健损失减弱哪一种耦合，以及它们没有提供什么保证。
\item
  Off-policy 校正：从轨迹概率比推导 ordinary、self-normalized 与 per-decision importance sampling，讨论支持条件、权重退化和截断偏差。
\item
  策略梯度直接优化动作分布：从轨迹概率推出 log-derivative 估计，再用 reward-to-go 与 baseline 降低方差。
\item
  Actor--Critic 在偏差与方差之间搭桥：用学习到的价值函数近似 advantage，理解多步目标、双时间尺度与离策略修正。
\end{enumerate}

从因果角度看，策略会在每个时刻主动改变动作分配机制，进而改变未来状态分布；仅对既有数据做条件化无法表达这种干预。Machine Learning Part 的``因果推断：预测世界不等于改变世界''讨论静态干预、可识别性与重叠；本单元的 off-policy 评价和策略学习把同一问题推进到序贯场景，其中历史依赖、动态混杂和支持缺口会反复出现。

若只做一次行动，数理统计中的``统计决策与可信预测：模型给出概率之后怎样行动''可以用 posterior risk、minimax、拒绝成本与容量约束选择规则。强化学习增加的核心困难是：行动会创造下一步状态和数据，眼前风险最小不必等于累计回报最大。

十篇共享一套记号：状态 \(s\)，动作 \(a\)，奖励 \(r\)，折扣率 \(\gamma\)，状态值 \(V^\pi\)，动作值 \(Q^\pi\)。本单元聚焦离散时间和折扣回报；有限时域、平均回报、部分可观测与连续时间问题需要相应改写，不能只换符号。
