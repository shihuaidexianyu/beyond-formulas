\section{风险、收益与投资组合}

投资组合选择展示了凸优化最吸引人也最危险的一面：数学模型清楚、求解全局可靠，但输入的期望收益与协方差很难准确估计。算法可以精确解决模型，却不能替模型承担预测风险。

设 \(w_i\) 是资产 \(i\) 的资金权重，\(R\) 是各资产收益率组成的随机向量，\(\mu\) 是其期望（预期收益向量），\(\Sigma\succeq0\) 是其协方差（\(\succeq0\) 表示半正定）。组合的预期收益与方差分别是

\[
\mu^Tw,
\qquad
w^T\Sigma w.
\]

预算约束通常为

\[
\mathbf1^Tw=1.
\]

若不允许做空，还加入 \(w\ge0\)。

\subsection{均值---方差模型是凸二次规划}

一种形式是给定风险厌恶参数 \(\gamma>0\)，最小化

\[
\frac{\gamma}{2}w^T\Sigma w-\mu^Tw
\]

并满足预算与持仓约束。由于 \(\Sigma\succeq0\)，目标凸；线性约束保持可行域凸。

也可以固定目标收益：

\[
\begin{aligned}
\min_w\quad&w^T\Sigma w\\
\text{s.t.}\quad&
\mu^Tw\ge r,\\
&\mathbf1^Tw=1,\\
&w\in C.
\end{aligned}
\]

其中 \(C\) 概括其余持仓约束，例如不允许做空时的 \(w\ge0\)。随着 \(r\) 改变，最优风险形成有效前沿。低于前沿的组合在相同风险下收益更差，或在相同收益下风险更高。

\(\gamma\) 与 \(r\) 是两种表达偏好的方式。在强对偶与活跃约束条件下，它们可通过 Lagrange 乘子关联，但不存在脱离数据的固定换算公式（第一节中惩罚参数 \(\lambda\) 与约束半径 \(t\) 的对应与此同理）。

\subsection{分散化来自协方差，而不只是资产数量}

若两项资产高度正相关，同时持有它们未必显著降低风险；若相关性较低或为负，组合方差可能小于各单项风险。二次型展开为

\[
w^T\Sigma w = \sum_i w_i^2\sigma_i^2
+2\sum_{i< j}w_iw_j\Cov(R_i,R_j).
\]
交叉协方差决定了分散化效果。逐元素非负或方差都小，不足以判断整个组合；\(\Sigma\) 必须整体半正定，才能保证任何权重方向的方差非负。

样本协方差在资产多、历史短时可能病态或秩亏。收缩估计、因子模型和 PSD 投影能改善稳定性，但它们改变了统计假设，不能只作为数值修复。

\subsection{现实约束怎样保持凸性}

许多交易规则仍可保持凸：

\begin{itemize}
\item
  单项权重上下界 \(l_i\le w_i\le u_i\)；
\item
  行业暴露等线性约束；
\item
  总杠杆 \(\lVert w\rVert_1\le L\)；
\item
  与当前持仓 \(w^{\mathrm{old}}\) 的换手限制

  \[
  \lVert w-w^{\mathrm{old}}\rVert_1\le T;
  \]
\item
  线性或分段线性交易成本；
\item
  对不确定收益集合取最坏情形后的鲁棒线性项。
\end{itemize}

固定交易费、最小交易手数和``最多持有 \(k\) 项资产''引入离散变量，通常破坏纯凸性。可以使用混合整数模型或凸代理，但应明确问题已经改变。

\subsection{影子价格帮助理解瓶颈}

若目标收益约束的乘子很大，稍微提高要求就会显著增加最小风险；若某个持仓上限乘子为零，局部放宽它不会改善目标。对偶变量能回答哪些规则真正限制了当前组合。

但影子价格依赖输入估计与当前活跃集合。市场环境变化、预期收益更新或交易成本模型改变后，价格也会变化。把一个乘子当作长期经济常数会超出模型允许的解释。

\subsection{最危险的误差常来自输入，而非求解器}

期望收益 \(\mu\) 的估计噪声通常很大，而最优权重对它可能高度敏感。优化器会把微小的估计优势放大成极端持仓，这就是``误差最大化器''现象。

因此实际流程应包括：

\begin{itemize}
\tightlist
\item
  对 \(\mu\) 与 \(\Sigma\) 做样本外估计和稳定化；
\item
  加入持仓、杠杆与换手约束；
\item
  进行情景分析和参数扰动，而不只看单一路径；
\item
  把交易成本、流动性和估计窗口纳入回测；
\item
  使用滚动样本外评估，避免未来信息泄漏；
\item
  同时报告预测不确定性与优化 gap。
\end{itemize}

一个原始---对偶间隙为 \(10^{-10}\) 的组合，只能说明给定模型几乎被精确求解；若 \(\mu\) 的误差远大于模型中的收益差，更多求解精度没有决策价值。
