\section{对偶间隙何时闭合}

弱对偶只保证

\[
d^\star\le p^\star.
\]

两者之差

\[
p^\star-d^\star
\]

称为最优对偶间隙。若间隙为零，就有强对偶：

\[
d^\star=p^\star.
\]

强对偶的意义远超过``两个问题答案相同''。它说原问题的最优值可以被一个全局下界完全认证；找到原始解的同时，也能找到一个解释``为什么不可能更好''的证书。

\subsection{凸性为什么还不够}

对于线性规划，在适当可行与有界条件下，强对偶表现得非常稳健。一般凸问题仍可能因定义域边界、缺乏内部可行点或闭性问题出现病态行为。于是需要约束资格条件，排除那些让几何分离失效的退化情形。

病态确实会发生。在定义域 \(y>0\) 上考虑：最小化 \(e^{-x}\)，约束 \(x^2/y\le0\)。这是凸问题（\(x^2/y\) 是``二次除线性''函数，在 \(y>0\) 上凸，可由 Hessian 半正定直接验证）。可行点必须满足 \(x=0\)，故 \(p^\star=1\)。但对任意 \(\lambda\ge0\)，固定 \(x\) 让 \(y\to\infty\)，惩罚项 \(\lambda x^2/y\) 就趋于零，于是

\[
g(\lambda)=\inf_{x,\,y>0}\left(e^{-x}+\lambda\,\frac{x^2}{y}\right)
=\inf_x e^{-x}=0,
\qquad d^\star=0<1=p^\star.
\]

问题凸，间隙却为正；原因在于不存在任何严格可行点，可行域只勉强贴在约束边界上。这正是约束资格要排除的情形。

最常用的充分条件是 Slater 条件。对凸问题

\[
\begin{aligned}
\min_x\quad&f_0(x)\\
\text{s.t.}\quad&f_i(x)\le0,\\
&Ax=b,
\end{aligned}
\]

若存在一个位于共同定义域相对内部（即把定义域放在其仿射包中考察时的内部）的点 \(\tilde x\)，满足

\[
f_i(\tilde x)<0
\quad\text{对所有非仿射不等式},
\qquad
A\tilde x=b,
\]

则在常见闭性条件下强对偶成立，并且对偶最优通常能够取得。

入门时常用更强而简单的版本：存在满足所有等式、且让全部不等式严格成立的点。对纯仿射不等式可以作更细的相对内部处理；关键直觉是可行域不能只勉强贴在约束边界上而没有任何内部余量。

\subsection{严格可行点在几何上做了什么}

对偶理论可以通过一个由目标与约束值组成的凸集合来证明。若最优目标以下的点不属于该集合，分离定理应当给出一个支持超平面；其法向量正是 Lagrange 乘子。

Slater 条件确保这个集合在相关方向上具有足够内部，使分离超平面不会退化为无法归一化的形式。换句话说，它让``存在一个线性下界证书''的几何直觉成为严格结论。

这也是为什么约束资格与凸性承担不同任务：凸性保证集合形状没有凹陷，约束资格保证最优点附近的约束描述没有病态退化。

\subsection{Slater 是充分条件，不是必要定义}

某个问题不满足 Slater，并不自动意味着强对偶失败。很多边界问题仍然具有零间隙，只是不能由这一条定理保证。反过来，原问题非凸也可能偶然出现零对偶间隙。

因此正确推理是：

\begin{itemize}
\tightlist
\item
  验证凸性与 Slater，可得强对偶保障；
\item
  Slater 失败时，需要其他定理、重新表述或直接分析；
\item
  不能把``没有找到严格可行点''直接写成``必有对偶间隙''；
\item
  也不能从一次数值实验中原始值与对偶值接近，就反推所有理论条件已满足。
\end{itemize}

\subsection{原始与对偶的可行、最优、取得要分开}

\(p^\star=d^\star\) 是最优值关系，不自动说明两边都存在最优点。可能出现最优值有限但某一侧只由序列逼近。实际求解时还要分别检查：

\begin{itemize}
\tightlist
\item
  原始可行点是否存在；
\item
  原始最优值是否取得；
\item
  对偶可行乘子是否存在；
\item
  对偶最优值是否取得；
\item
  间隙是否为零。
\end{itemize}

在满足 Slater 的凸问题中，这些性质较为良好，但仍应按定理条件陈述，而不是把``强对偶''扩写成所有良性结论。

\subsection{一个可计算的最优性区间}

假设某算法给出原始可行点 \(\hat x\)，目标值为 \(10.4\)；同时给出对偶可行乘子，满足

\[
g(\hat\lambda,\hat\nu)=10.1.
\]

无论强对偶是否已经证明，弱对偶都给出

\[
10.1\le p^\star\le10.4.
\]

因此 \(\hat x\) 的绝对次优度至多 \(0.3\)。这是比``迭代变化已经很小''更有解释力的停止证据。若还知道强对偶，并且上下界最终相遇，就同时认证两侧最优。

对尺度很大的目标，常使用相对间隙，但分母的选取必须避免最优值接近零时失真；软件报告的 gap 定义也应查看文档，而不是只看一个默认百分比。

\subsubsection{留给读者}

判断下列问题是否满足简单版 Slater 条件：

\[
\min_x x^2
\quad\text{subject to}\quad x^2\le0.
\]

它的可行域只有 \(x=0\)。Slater 失败是否能直接推出强对偶失败？请实际写出对偶问题再回答，并说明这个例子为何能区分``充分条件''和``必要条件''。
