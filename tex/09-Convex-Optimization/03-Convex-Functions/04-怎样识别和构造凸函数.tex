\section{怎样识别和构造凸函数}

实际模型中的目标很少只是一个裸露的 \(x^2\) 或 \(e^x\)。它们由求和、最大值、复合、边缘化和变量变换组合而成。若每次都回到凸函数定义，会让证明冗长且容易出错。更有效的方法是先认识一组可靠的凸函数原子，再使用经过证明的保凸规则。

\subsection{非负加权和与逐点上确界}

若 \(f_i\) 都凸，且 \(\alpha_i\ge0\)，那么

\[
f(x)=\sum_i\alpha_i f_i(x)
\]

仍凸。这直接来自弦不等式逐项相加。经验风险

\[
\frac1n\sum_{i=1}^n\ell_i(w)+\lambda R(w)
\]

因此可以通过凸损失与凸正则项构造。

凸函数族的逐点上确界

\[
f(x)=\sup_{\alpha} f_\alpha(x)
\]

也是凸函数，因为其上境图是所有上境图的交——``弦在图像上方意味着什么''一节对逐点最大值用过的正是这个论证。有限最大值、支持函数

\[
\sigma_C(y)=\sup_{x\in C}y^Tx
\]

以及最大特征值函数都属于这一结构。

逐点下确界一般不保凸；两只碗取较低的一只，可能产生两个分离谷底。但若对一部分变量做最小化且联合函数凸，

\[
g(x)=\inf_y f(x,y),
\]

在适当定义域条件下 \(g\) 仍凸。其上境图可由联合上境图投影得到。消去辅助变量和构造 value function 时经常使用这条规则。

\subsection{仿射复合总是安全的}

若 \(f\) 凸，则

\[
g(x)=f(Ax+b)
\]

凸。因为仿射映射保存凸组合：

\[
A(\theta x+(1-\theta)y)+b = \theta(Ax+b)+(1-\theta)(Ay+b).
\]
最小二乘

\[
\lVert Ax-b\rVert_2^2
\]

、范数残差 \(\lVert Ax-b\rVert\) 和 logistic 损失中的线性预测量都依赖这条规则。

一般复合则需要单调性条件。若外层 \(h\) 凸且逐坐标非减，内层 \(g_i\) 凸，那么

\[
h(g_1(x),\ldots,g_k(x))
\]

凸。若某个坐标上外层非增，对应内层通常需要凹。忽略单调方向是复合证明中最常见的错误。

例如 \(e^{g(x)}\) 在 \(g\) 凸时仍凸，因为指数既凸又递增；但 \(g(x)^2\) 不能仅因平方函数凸就断言凸，因为平方在负半轴上递减。若已知 \(g(x)\ge0\)，这条复合才安全。

\subsection{Log-sum-exp：从稳定计算到凸模型}

函数

\[
f(x)=\log\sum_{i=1}^n e^{x_i}
\]

在数值分析中因稳定概率计算出现，在凸分析中则是最大值的光滑近似。其梯度是 softmax 概率向量 \(p\)，即 \(p_i=e^{x_i}\big/\sum_j e^{x_j}\)，Hessian 为

\[
\nabla^2 f(x) = \diag(p)-pp^T.
\]
对任意 \(v\)，

\[
v^T\nabla^2f(x)v = \sum_i p_i v_i^2-\left(\sum_i p_i v_i\right)^2
=\Var_p(v_i)\ge0.
\]
右端可以读作``以概率 \(p_i\) 取值 \(v_i\)''的随机变量的方差；这里的等号是恒等式而非类比，非负性因此来自方差本身。所以 LSE 凸。这个证明还说明 Hessian 沿 \(\mathbf1\) 方向为零，因为给所有 \(x_i\) 加同一常数只会使函数整体增加该常数，softmax 不变。

二分类 logistic 损失

\[
\ell(t)=\log(1+e^{-t})
\]

是 LSE 与仿射映射的组合，因此凸。令 \(t=y_iw^Tx_i\) 后，对样本求和仍然凸。这种从原子与规则构造模型的思路，比每次重新算 Hessian 更清晰。

\subsection{透视函数把比例关系变成凸结构}

若 \(f\) 凸，其透视函数定义为

\[
g(x,t)=t f(x/t),
\qquad t>0.
\]

透视保持凸性。以 \(f(x)=x^2\) 为例，

\[
g(x,t)=\frac{x^2}{t},
\qquad t>0,
\]

是凸函数，尽管从分式外观看不直观。二次除线性、分式模型和若干锥约束都由此而来。

证明可以从上境图的线性变换入手，也可以直接追踪带不同 \(t\) 权重的凸组合。重要的是辨认``同质化''结构，而不是把分母出现变量一概视为非凸。

\subsection{保凸规则是一种建模纪律}

Disciplined Convex Programming 要求表达式由已知曲率与单调性的原子按合法规则组合。这样，系统可以在求解前验证凸性并转换为标准锥形式；软件验证合法性，但不会替用户完成建模判断。

这套纪律有意保守：有些函数数学上凸，却无法由现有规则直接识别。此时应寻找等价表达、补充原子或给出独立证明，而不是关闭检查后希望求解器自行处理。

同样，以下外观判断都不可靠：

\begin{itemize}
\tightlist
\item
  两个正凸函数的乘积不一定凸；
\item
  两个凸函数的差不一定凸；
\item
  凸函数的等值约束一般不凸；
\item
  可逆的非线性变量变换不一定保凸；
\item
  Hessian 在若干采样点半正定不能证明全域凸。
\end{itemize}

\subsubsection{留给读者}

证明函数

\[
f(x)=\max_{1\le i\le m}(a_i^Tx+b_i)
\]

凸，并分别从``逐点最大值''\,``上境图是半空间的交''\,``次梯度是活跃斜率的凸包''三个角度解释它。其中第三个角度在``切平面为何是全局下界''一节中已经算过一个具体实例。三个角度各自为后续哪类问题提供方便？
