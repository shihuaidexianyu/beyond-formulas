\section{变量变换揭示隐藏凸性}
\label{sec:09-Convex-Optimization/04-Convex-Problems/03}

\begin{quote}
\textbf{先抓住这一点：}一个问题写成原始变量时可能显得非凸，换成对数、比例或新的辅助变量后，熟悉的凸结构才会出现。变量变换不是装饰，它会改变我们能否看见并利用全局结构。
\end{quote}

有些问题在原始坐标中不满足凸优化的标准形式，却能通过一一对应的变量变换显露凸性。几何规划是最典型的例子：它处理正变量的幂乘积与正项之和，原式看似非线性，取对数后却变成仿射函数与 log-sum-exp。

\subsection{单项式与正项式}

在几何规划中，变量必须满足 \(x_i>0\)。单项式定义为

\[
g(x)=c x_1^{a_1}\cdots x_n^{a_n},
\qquad c>0,
\]

其中指数 \(a_i\) 可以是任意实数。正项式是若干单项式之和：

\[
f(x)=\sum_{k=1}^K
c_kx_1^{a_{1k}}\cdots x_n^{a_{nk}},
\qquad c_k>0.
\]

标准几何规划写成：最小化一个正项式，满足正项式不等式 \(f_i(x)\le1\) 和单项式等式 \(g_j(x)=1\)。

这些函数在原变量 \(x\) 中不一定凸。例如 \(xy\) 的 Hessian 是
\(\begin{bmatrix}0&1\\1&0\end{bmatrix}\)，特征值为 \(1\) 与 \(-1\)，不定。几何规划的凸性不来自原坐标，而来自对数变换。

\subsection{对数坐标把乘法变成加法}

令

\[
y_i=\log x_i,
\qquad x_i=e^{y_i}.
\]

单项式的对数变为

\[
\log g(e^y) = \log c+a^{\trans}y,
\]
是仿射函数。正项式的对数变为

\[
\log f(e^y) = \log\sum_{k=1}^K
\exp(a_k^{\trans}y+\log c_k),
\]
也就是仿射函数的 log-sum-exp，因此凸（log-sum-exp 的凸性在上一单元的保凸运算中已建立）。

因为对数严格递增，约束

\[
f(x)\le1
\]

等价于

\[
\log f(e^y)\le0.
\]

单项式等式则变成仿射等式。目标也可以取对数而不改变最优解排序，于是整个几何规划变成 \(y\) 中的凸优化问题。

\subsection{一个面积最大的盒子}

考虑正数 \(x,y\)，在资源约束

\[
x+y\le10
\]

下最大化面积 \(xy\)。等价地最小化单项式

\[
(xy)^{-1},
\]

并把约束写成正项式

\[
\frac{x}{10}+\frac{y}{10}\le1.
\]

令 \(u=\log x,v=\log y\)。目标的对数为

\[
-u-v,
\]

约束变成

\[
\log(e^u+e^v)\le\log10.
\]

这是凸问题。最优点也可以直接验证：由 \(x+y\le10\) 与算术---几何平均不等式，

\[
xy\le\Bigl(\frac{x+y}{2}\Bigr)^2\le25,
\]

等号当且仅当 \(x=y=5\)。下一单元的 KKT 条件会在对数变换后的问题中给出同一结论。

原问题``最大化乘积''看起来不像凸最小化；但正变量、幂乘积和正项式结构表明，它具有隐藏凸性。

\subsection{变换必须一一对应并保存顺序}

并非任何换元都能把非凸问题合法地``变凸''。可靠变换至少要回答：

\begin{itemize}
\tightlist
\item
  新旧变量之间是否在定义域上有一一对应？
\item
  可行点是否双向对应，边界是否丢失？
\item
  目标排序是否由单调变换保存？
\item
  新约束是否真的满足凸性方向？
\end{itemize}

例如取对数要求变量严格为正；允许零的原问题不能直接照搬，因为 \(\log0\) 不存在。有时可以通过闭包或极限处理边界，但必须单独说明。

此外，几何规划中的等式必须是单项式等式。正项式等式一般在对数变换后成为凸函数等于零，仍可能非凸。规则的细节正是隐藏凸性能够成立的条件，不是语法负担。

\subsection{识别隐藏凸性的更一般经验}

对数变换只是众多技巧之一。分式线性变换、透视函数、Schur 补和 epigraph 也常把表面复杂的约束改写成标准凸形式。真正可迁移的步骤是：

\begin{enumerate}
\tightlist
\item
  先辨认乘法、比值、最大值、二次型或谱约束的结构；
\item
  寻找保持可行性与目标顺序的变量或 epigraph 变换；
\item
  在新变量中用已知原子和保凸规则证明凸性；
\item
  求解后把结果映回原变量，并检查边界与数值尺度。
\end{enumerate}
