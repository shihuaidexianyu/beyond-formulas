\section{最小二乘、范数与二次规划}

给定超定线性方程 \(Ax\approx b\)，我们无法让所有方程同时精确成立，只能定义``误差小''意味着什么。选择不同范数等于对误差形态作出不同的建模选择，远超过计算细节。

\subsection{最小二乘选择平方总误差}

最小二乘问题写成

\[
\min_x\lVert Ax-b\rVert_2^2.
\]

它是凸二次函数：展开得

\[
\lVert Ax-b\rVert_2^2=x^TA^TAx-2b^TAx+b^Tb,
\]

其 Hessian 为

\[
2A^TA\succeq0.
\]

若 \(A\) 满列秩，\(A^TA\succ0\)，目标严格凸并有唯一解；若 \(A\) 秩亏，仍是凸问题，但可能有多组同样的预测结果。最小范数解需要额外选择。

一阶条件给出正规方程

\[
A^TAx=A^Tb,
\]

它有清楚的几何意义：残差 \(b-Ax\) 与 \(A\) 的列空间正交。但数值实现不应机械计算 \((A^TA)^{-1}A^Tb\)。QR 或 SVD 能避免条件数平方，并正确处理秩亏。凸建模告诉我们最优点满足什么，数值线性代数决定怎样可靠地算到那里。

\subsection{范数改变``最重要的误差''}

一般范数拟合写成

\[
\min_x\lVert Ax-b\rVert.
\]

由于范数凸、仿射复合保凸，这始终是凸问题。不同选择体现不同偏好：

\begin{itemize}
\tightlist
\item
  \(L_2\) 平方惩罚大残差，光滑且便于统计解释，但对离群点敏感；
\item
  \(L_1\) 最小化绝对误差之和，对少量极端误差更稳健，但目标不可微；
\item
  \(L_\infty\) 最小化最大绝对误差，追求最坏位置的均匀控制。
\end{itemize}

\(L_1\) 和 \(L_\infty\) 拟合都可以用上一节的辅助变量技巧写成线性规划。\(L_2\) 范数约束则自然属于二阶锥规划：

\[
\lVert Ax-b\rVert_2\le t.
\]

同一个数据拟合故事因此连接了 LP、SOCP 与二次优化。

\subsection{二次规划保留交互与曲率}

凸二次规划的一般形式是

\[
\begin{aligned}
\text{minimize}\quad&
\frac12x^TPx+q^Tx+r\\
\text{subject to}\quad&
Gx\le h,\quad Ax=b,
\end{aligned}
\]

其中 \(P\succeq0\)。二次项可以描述能量、方差、平滑度或偏离参考点的代价；非对角元素表达变量之间的相互作用。

若 \(P\) 有负特征值，目标在某些方向向下弯，问题一般非凸。仅仅看到``平方项''不能断言凸，因为交叉项可能使 Hessian 不定。例如

\[
f(x,y)=x^2-y^2
\]

既有正曲率也有负曲率。

Ridge 回归

\[
\min_w
\frac12\lVert Xw-y\rVert_2^2
+\frac\lambda2\lVert w\rVert_2^2
\]

是典型凸二次问题。\(\lambda>0\) 使 Hessian

\[
X^TX+\lambda I
\]

正定，即使 \(X\) 秩亏也获得唯一解。正则化在这里同时具有统计与优化作用：限制模型复杂度，并抬高平坦方向的曲率。

\subsection{用 epigraph 统一目标与约束}

范数目标

\[
\min_x\lVert Ax-b\rVert_2
\]

可写成

\[
\begin{aligned}
\text{minimize}\quad&t\\
\text{subject to}\quad&\lVert Ax-b\rVert_2\le t.
\end{aligned}
\]

这样做看似增加变量，却把复杂目标移入一个标准锥约束。现代求解器正是通过引入辅助变量，把高层表达式转换成线性、二阶锥或半正定锥上的标准问题。

辅助变量本身不是代价；真正要问的是变换是否保持等价、是否暴露可利用结构，以及新问题的数值尺度是否合理。

\subsection{从模型到验证}

求得候选解后，可以按模型类型建立独立检查：

\begin{itemize}
\tightlist
\item
  最小二乘检查残差是否与列空间正交；
\item
  二次规划检查原始可行性、对偶可行性和 KKT 残差（对偶与 KKT 条件在下一单元正式建立）；
\item
  范数拟合比较不同范数下残差分布，而不只比较一个目标数；
\item
  正则化路径观察 \(\lambda\) 改变时解是否连续、复杂度是否符合预期。
\end{itemize}

若一个极小的训练误差来自巨大参数抵消或严重病态设计矩阵，它在数学目标上可能优秀，在预测和数值上却不稳定。优化最优性不是完整的模型有效性。

\subsubsection{留给读者}

考虑带盒约束的 Ridge 问题

\[
\min_w\frac12\lVert Xw-y\rVert_2^2
+\frac\lambda2\lVert w\rVert_2^2,
\qquad -1\le w_i\le1.
\]

把它写成标准凸二次规划，给出 Hessian，并说明 \(\lambda=0\) 与 \(\lambda>0\) 时唯一性分别还需要什么条件。不要显式求逆。
