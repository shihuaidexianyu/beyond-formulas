没有标签时，数据不会自动告诉我们``正确的簇''是什么。K-means 偏好欧氏空间中的近似球形成分，谱聚类偏好相似图上容易切开的连通区域，层次聚类又由 linkage 规则决定哪些局部合并会被保留。不同方法不是在逼近同一个隐藏答案，而是在用不同几何定义结构。

本单元从 K-means 的平方距离目标与交替下降开始，区分目标单调下降、局部最优和聚类具有现实意义。随后把样本变成加权图，用图 Laplacian、Rayleigh 商和连续松弛解释谱聚类为何能识别弯曲或非凸簇。最后讨论聚类数、尺度参数、初始化、稳定性和外部验证，说明一张视觉上漂亮的二维图不能充当高维结构的证明。

PCA、流形学习与矩阵补全仍留在前一单元，因为它们首先回答表示和重构问题；本单元只在需要时调用这些表示，并追问距离、邻接和分区准则怎样共同制造``簇''。
