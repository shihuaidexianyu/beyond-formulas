没有标签时，数据不会自动告诉我们``正确的簇''是什么。K-means 偏好欧氏空间中的近似球形成分，谱聚类偏好相似图上容易切开的连通区域，层次聚类又由 linkage 规则决定哪些局部合并会被保留。不同方法用不同几何定义结构，通常没有一个它们都在逼近的唯一隐藏答案。

本单元从 K-means 的平方距离目标与交替下降开始，区分目标单调下降、局部最优和聚类具有现实意义。随后把样本变成加权图，用图 Laplacian、Rayleigh 商和连续松弛解释谱聚类为何能识别弯曲或非凸簇。最后讨论聚类数、尺度参数、初始化、稳定性和外部验证，说明一张视觉上漂亮的二维图不能充当高维结构的证明。

PCA、流形学习与矩阵补全仍留在前一单元，因为它们首先回答表示和重构问题；本单元只在需要时调用这些表示，并追问距离、邻接和分区准则怎样共同制造``簇''。

\subsection{怎么读这一章}

首次阅读先用 K-means 建立``聚类是目标函数与几何的产物''这一认识，再用谱聚类观察更换相似图后，同一批点如何产生不同分区。细节上可暂时跳过特征值松弛的完整推导，但不能跳过尺度、簇数、初始化和稳定性检查。若无法说明换一种度量或邻接规则为何会换一种``真相''，就还不应给簇赋予现实名称。
