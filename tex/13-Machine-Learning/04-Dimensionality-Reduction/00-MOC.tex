一张包含上千列的数据表未必真的有上千个彼此独立的变化方向。多个传感器可能记录同一物理状态，多项消费指标可能共同反映少数偏好，图像的相邻像素也有强烈结构。降维所利用的第一性事实是：观测坐标很多，不等于生成数据的自由度很多。

这一单元先沿着``完整观测到不完整观测''的顺序展开，再转向弯曲结构：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  PCA 从最大方差到最小重构误差：从相关特征带来的冗余出发，解释中心化、尺度选择、主方向和训练/测试隔离。
\item
  SVD 给出最佳低秩近似：把 PCA 背后的工具提升为一般矩阵近似，并讨论误差、谱隙和计算。
\item
  矩阵补全从缺失条目到低秩正则：进一步拿走大量观测，追问低秩结构何时足以恢复缺失项，何时无论算法多强都不可识别。
\item
  流形假设与非线性降维：当低维结构弯曲时，任何线性投影都会叠压不同层；Isomap 用测地距离替换欧氏距离，LLE 与 t-SNE 只保局部邻居，代价是近邻敏感、噪声短路与没有显式映射。
\item
  因子分析与概率 PCA：把低秩结构写成带观测噪声的生成模型，分清共同因子与各坐标独有噪声，并说明旋转不可辨认性。
\item
  ICA 与 CCA：前者用独立性分离混合源，后者从两组变量寻找最大相关方向；不同结构假设决定``潜在方向''究竟能否被辨认。
\end{enumerate}

前三篇都在使用``低秩''，但任务不同：PCA 寻找样本变化的坐标系，截断 SVD 近似一个已知矩阵，矩阵补全则要从部分条目推断未知矩阵。混淆这三者，往往会把一个代数结论误当成统计保证。第四篇保留``自由度少''的信念，只放弃``全局线性''的假设：流形方法信任小尺度上的线性，不再相信大尺度上的线性。最后两篇回到生成与可辨认性：同样是线性潜变量，采用各向同性噪声、坐标特异噪声、非 Gaussian 独立性或跨视图相关性，会得到完全不同的方向。
