\section{SVD 给出最佳低秩近似}
\label{sec:13-Machine-Learning/04-Dimensionality-Reduction/02}

\begin{quote}
\textbf{先抓住这一点：}截断 SVD 在给定秩下给出 Frobenius 范数和谱范数意义的最佳矩阵近似。它优化的是重构误差，不保证保留分类、因果或稀有事件所需的信息。
\end{quote}

上一节的 PCA 从样本方差出发，而 SVD 面对的是更一般的问题：给定任意矩阵 \(A\)，能否用一个秩更低的矩阵保存它最重要的作用？这既是数据压缩问题，也是理解线性映射的方式。

任意 \(A\in\R^{m\times n}\) 都可以写成

\[
A=U\Sigma V^{\trans}
=\sum_{j=1}^r\sigma_ju_jv_j^{\trans},
\]

其中 \(r\) 是 \(A\) 的秩，\(\sigma_1\ge\cdots\ge\sigma_r>0\)，\(u_j,v_j\) 分别是 \(U,V\) 的列。几何上，\(V^{\trans}\) 先把输入旋转到一组正交方向，\(\Sigma\) 沿每个方向伸缩，\(U\) 再把结果旋转到输出空间。奇异值越大，矩阵沿该方向施加的作用越强。

\subsection{截断为何是最佳近似}

只保留前 \(k\) 项：

\[
A_k=\sum_{j=1}^k\sigma_ju_jv_j^{\trans}.
\]

Eckart--Young--Mirsky 定理说明，在所有秩不超过 \(k\) 的矩阵 \(B\) 中，\(A_k\) 同时最小化谱范数误差和 Frobenius 范数误差：

\[
\min_{\rank(B)\le k}\|A-B\|_2
=\sigma_{k+1},
\]

\[
\min_{\rank(B)\le k}\|A-B\|_F
=\left(\sum_{j>k}\sigma_j^2\right)^{1/2}.
\]

直觉是秩 \(k\) 的近似至多完整保存 \(k\) 个独立方向，因此总有至少一个强度不小于 \(\sigma_{k+1}\) 的方向被遗漏；截断 SVD 恰好把最强的 \(k\) 个方向全部保留，从而达到这个不可突破的下界。

对一张灰度图像而言，\(A_k\) 用 \(k\) 个外积叠加出近似图像。光滑背景和大尺度形状通常先被较大奇异值捕获，细节和噪声逐渐落入尾部。但``尾部就是噪声''并非定理：细小病灶、稀有边缘或局部异常也可能能量很低，却对任务至关重要。

\subsection{低矩阵误差不等于处处准确}

Frobenius 范数累计所有条目的平方误差，所以一个很局部但很重要的错误可能被大量准确条目淹没。谱范数控制最坏输入方向上的线性作用，却仍不直接控制某个业务指标。选择 \(k\) 时，应先明确要保存的是视觉质量、线性算子作用、下游预测，还是存储预算。

谱隙也影响解释，上一节在 scree plot 上寻找的正是它。若 \(\sigma_k\) 与 \(\sigma_{k+1}\) 相差很大，前 \(k\) 维子空间通常对小扰动较稳定；若两者几乎相同，轻微噪声就可使边界两侧的奇异向量明显旋转。重复奇异值对应的单个向量本来就不唯一，只有它们张成的子空间有稳定含义。

\subsection{计算时不要绕道平方条件数}

从 \(A^{\trans} A\) 的特征分解理论上可以得到右奇异向量，但显式形成 \(A^{\trans} A\) 会把条件数平方，也可能丢失小奇异值精度。一般应使用成熟的 SVD 实现。

当矩阵很大而只需前少量成分时，可以用 Lanczos、子空间迭代或 randomized SVD。随机算法通过一个较小的随机探测矩阵近似 \(A\) 的主要列空间，再在该子空间里求小规模 SVD。它通常很快，但结果带有近似与随机误差，因此应：

\begin{itemize}
\tightlist
\item
  设置适当 oversampling 与幂迭代次数；
\item
  检查 \(\|A-A_k\|\) 或随机残差；
\item
  改变随机种子观察子空间稳定性；
\item
  不把算法近似误差与模型截断误差混为一谈。
\end{itemize}
