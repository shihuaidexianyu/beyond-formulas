训练误差接近零的模型，在新数据上可能一塌糊涂。这是由训练误差对真风险的系统性乐观导致的数学现象，不能只当成工程事故。本单元要回答的问题是这份乐观有多大、由什么决定、又怎样被控制住。

主线只有一条：把前面分别学过的三块材料接到同一个问题上。经验风险最小化（\hyperref[sec:09-Convex-Optimization/07-Applications/01]{``光滑经验风险与 \(L_2\) 正则化''}）告诉我们怎样把训练误差压下去；几何间隔（\hyperref[sec:09-Convex-Optimization/07-Applications/03]{``最大间隔与支持向量机''}）给出一个看起来更稳妥的选择准则；Hoeffding 集中界（\hyperref[sec:05-Probability/07-Transforms-and-Bounds/04]{``Chernoff 方法与 Hoeffding 集中界''}）则提供了量化``样本均值偏离期望多少''的工具。本单元用这把尺子逐层推进：先给单个固定假设一个干净的界，再看``从数据中选假设''怎样把它破坏，然后用联合界与一致收敛修复有限假设类，再用 VC 维把无限假设类变回``有效有限''，最后诚实盘点这些界能解释什么、不能解释什么。

\begin{enumerate}
\tightlist
\item
  训练误差为何总是乐观：定义真风险与经验风险，说明同一批数据被``选假设''与``评假设''用了两次会让训练误差向下偏，以及单一固定假设的 Hoeffding 界为什么对选出来的 \(\widehat h\) 失效。
\item
  从联合界到一致收敛：假设类有限时，union bound 以 \(\log\abs{\mathcal{H}}\) 的代价换来对所有假设同时成立的界，ERM 因此获得超出风险保证；同时看到任何实值参数的假设类都让这条路直接断掉。
\item
  VC 维把无限假设类变有限：用``在 \(n\) 个点上能产生多少种标记''替代类的基数，经生长函数、打散与 Sauer 引理得到不依赖 \(\abs{\mathcal{H}}\) 的泛化界，并给出``需要多少样本''的量级答案。
\item
  泛化界的解释力与边界：承认 VC 界数值上很松，但看清它如何解释正则化、结构风险最小化与间隔；同时直面过参数化与双下降这些经典框架解释不了的现代现象。
\end{enumerate}

\subsection{怎么读这一章}

随手阅读时，可以只沿着每篇开头的“核心洞见”前进：第一篇看“选择与评估为何不能共用数据”，第二篇看“同时控制为什么要为候选数量付费”，第三篇看“容量为何是可实现行为而非参数个数”，第四篇看“界是结构性判断而非性能预言”。想掌握证明机制，再读 Hoeffding、union bound、ERM 的三步不等式、Sauer 引理与对称化骨架。真正使用这些结论时，还要逐项核对独立同分布、有界损失、假设类是否预先确定，以及数据是否参与过模型选择。
