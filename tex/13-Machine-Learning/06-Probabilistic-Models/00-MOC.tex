判别模型直接学习输入到输出的关系；生成式模型则描述类别、潜在状态和观测怎样共同产生。后一种路线承担更多假设，却能在标签缺失、归属不确定或状态随时间变化时，把``不知道什么''本身放进模型。

\begin{quote}
\textbf{先抓住这一点：}概率模型、推断算法和现实解释是三个不同层级。模型规定允许哪些分布，算法计算或近似模型中的未知量，数据与外部证据才决定隐变量能否获得现实含义。
\end{quote}

本章沿``逐步拿走可见信息''的贯穿主线前进：先让类别标签可见，再隐藏类别，随后让隐状态沿时间相关；最后抽取指数族和共轭性这一反复出现的代数结构。

\subsection{核心主线}

\begin{enumerate}
\tightlist
\item \hyperref[sec:13-Machine-Learning/06-Probabilistic-Models/01]{``生成式分类：从 Naive Bayes 到 GDA''}：标签可见时，先学习各类怎样生成特征，再反推类别；
\item \hyperref[sec:13-Machine-Learning/06-Probabilistic-Models/03]{``Gaussian 混合模型让类别成为隐变量''}：拿走标签后，用后验责任度保留软归属；
\item \hyperref[sec:13-Machine-Learning/06-Probabilistic-Models/04]{``EM 用下界交替完成隐变量学习''}：把参数未知与归属未知的循环依赖变成两个可执行步骤；
\item \hyperref[sec:13-Machine-Learning/06-Probabilistic-Models/05]{``HMM 把隐变量排列成时间链''}：加入时间依赖，并用动态规划合并共享前缀。
\end{enumerate}

第一次阅读可先完成这四篇。它们共同回答``信息被隐藏后怎样仍然学习''。

\subsection{怎么读这一章}

首次阅读沿``标签可见—标签隐藏—时间相关''推进：生成式分类、GMM、EM、HMM。重点区分模型分布、后验推断与参数学习，并记住隐变量标签通常只在置换意义下可识别。LDA 几何、指数族与共轭先验适合第二遍回读，它们分别解释判别方向、矩匹配和闭式更新为何出现。EM 的单调性、共轭的闭式性和训练似然的提高都只是算法或模型内部性质，不等于全局最优、模型正确或现实解释成立。

\subsection{贯穿判断}

阅读每个模型时都问三次：假设允许哪些分布？算法实际计算了什么？现有数据凭什么支持我们赋予隐变量的名称？计算方便只能回答第二问，不能替代模型检验和现实证据。
