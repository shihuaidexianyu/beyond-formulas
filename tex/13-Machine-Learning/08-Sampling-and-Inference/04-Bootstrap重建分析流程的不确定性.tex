\section{Bootstrap 重建分析流程的不确定性}

前面三节都从给定概率模型出发计算期望或 posterior。Bootstrap 回答另一类问题：如果按同样方式重新收集一份数据，整个分析结果会波动多少？

设完整分析流程为 \(A\)，原数据为 \(D\)，估计量为

\[
\widehat\theta=A(D).
\]

Bootstrap 从经验分布或拟合模型生成复样 \(D^{*b}\)，并重新计算

\[
\widehat\theta^{*b}=A(D^{*b}),
\qquad b=1,\ldots,B.
\]

这些复样结果的经验分布用于估计标准误、偏差、区间或选择稳定性。\(B\) 是我们重复模拟的次数，不是新增了 \(B\) 份真实数据；它能减小 bootstrap 计算误差，不能增加原始样本的信息量。

\subsection{Bootstrap 首先要确定抽样单位}

若数据确实是 iid 个体，可以按行有放回重采样。若同一患者有多次测量，独立单位是患者而不是记录行；逐行抽样会拆散相关结构并高估有效样本量。时间序列可能需要 block bootstrap，分层抽样设计需要在层内复样，回归固定设计下还可能使用 residual 或 wild bootstrap。

因此第一步应画出数据收集层级：谁被随机抽入样本，哪些观测共享同一个个体、地点、批次或时间冲击。重采样方案必须模拟原始抽样机制，而不是服从表格的存储格式。

\subsection{必须重跑整个数据自适应流程}

假设要报告 Lasso 模型某变量被选择的稳定性。如果每个复样只对已经选好的变量和固定 \(\lambda\) 重新拟合，便漏掉了变量筛选与调参本身的波动。正确的 \(A(D^{*b})\) 应在每个复样内部重新：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  拟合缺失值处理、编码和标准化；
\item
  执行特征筛选；
\item
  用训练内验证选择超参数；
\item
  拟合最终模型；
\item
  计算目标系数、预测或性能指标。
\end{enumerate}

这通常比一次拟合昂贵，但它才近似``重新做一遍研究''。如果部署性能是目标，还要为每个复样保留相应 out-of-bag 或外部评估数据，避免在同一复样训练结果上报告训练表现。

\subsection{区间有条件，失败也应被保留}

最简单的 percentile interval 取 bootstrap 分位数，但偏差、偏度和边界参数可能使其覆盖失真。basic、studentized 或 BCa 区间各有不同修正，并非换个公式就获得普遍正确性。Bootstrap 一致性通常要求统计量是经验分布的适当光滑 functional；极值、某些非正则参数、模型选择后的不连续统计量和很少的 cluster 都可能违反常规条件。

复样中的拟合失败不能静默丢弃。若某些样本导致分离、奇异矩阵或缺少类别，这本身就是分析流程不稳定的证据。应记录失败率、失败原因，以及结果对重采样方案、随机种子和 \(B\) 的敏感性。

\subsection{与 posterior 和 Monte Carlo 误差分清}

Bootstrap 分布近似估计量在重复抽样中的分布；Bayesian posterior 是给定已观察数据后参数的不确定性分布；Monte Carlo 标准误则衡量有限模拟对某个计算目标的误差。这三者有时数值相近，含义仍不同。第一节要求把 Monte Carlo 标准误写进报告、第二节用有效样本量折算相关链，为的都是不让数值近似误差混进统计不确定性的账。

以患者级风险模型为例，bootstrap 区间可能描述``若重新抽取一批患者，AUC 会怎样变化''；MCMC posterior 区间描述``在所选 Bayesian 模型下，给定当前数据，参数或预测的不确定性''；增加 bootstrap 次数或链长度只会让各自的数值近似更稳定，不会增加患者数。
