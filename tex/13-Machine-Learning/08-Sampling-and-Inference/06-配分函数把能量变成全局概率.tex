\section{配分函数把能量变成全局概率}

有些模型容易判断一个配置“好不好”，却难以直接给出它的规范化概率。能量模型为状态 \(x\) 指定标量 \(E_\theta(x)\)，再定义

\[
p_\theta(x)
=\frac{\exp[-E_\theta(x)]}{Z(\theta)},
\qquad
Z(\theta)
=\int \exp[-E_\theta(x)]\,dx.
\]

离散状态时积分换成求和。低能量表示相对偏好，只有配分函数 \(Z(\theta)\) 把所有可能状态的总质量归一化后，才得到概率。能量加上与 \(x\) 无关的常数不会改变分布，因为分子和 \(Z\) 同时乘上相同因子；因此能量的绝对零点通常不可辨认。

\subsection{配分函数困难来自全局耦合}

局部能量可以由稀疏图或神经网络快速计算，\(Z\) 却要遍历整个状态空间。\(d\) 个二值变量已有 \(2^d\) 种配置。知道某个样本的能量，并不足以知道它的概率，因为还必须知道其得分相对所有未观察配置处在什么位置。

这也解释了三种常被混用的任务：

\begin{itemize}
\tightlist
\item 比较两个状态的概率比时，\(Z\) 会抵消：
  \[
  \frac{p_\theta(x)}{p_\theta(x')}
  =\exp\{-[E_\theta(x)-E_\theta(x')]\};
  \]
\item 从模型采样时，不一定显式计算 \(Z\)，但仍要构造能探索 \(p_\theta\) 的 Markov 链；
\item 计算精确 likelihood 或学习参数时，\(Z\) 及其梯度通常重新出现。
\end{itemize}

“无需知道归一化常数也能运行 MCMC”不等于“配分函数已经不重要”。困难只是从高维求和转移到了混合速度和 Monte Carlo 误差。

\subsection{似然梯度有正相与负相}

单个观测的 log likelihood 为

\[
\log p_\theta(x)
=-E_\theta(x)-\log Z(\theta).
\]

对参数求导，并利用

\[
\nabla_\theta\log Z(\theta)
=-\E_{p_\theta}
[\nabla_\theta E_\theta(X)],
\]

可得

\[
\nabla_\theta\log p_\theta(x)
=-\nabla_\theta E_\theta(x)
+\E_{p_\theta}
[\nabla_\theta E_\theta(X)].
\]

最大化数据平均 likelihood 时，第一项降低数据附近的能量，常称 positive phase；第二项抬高模型当前过度偏好的区域，常称 negative phase。后者需要对模型分布取期望，正是训练瓶颈。只做正相会让所有训练样本能量下降，却没有约束其他区域，无法得到正确的相对概率。

若能量线性写成 \(E_\theta(x)=-\theta^{\trans}T(x)\)，梯度变为

\[
\nabla_\theta\log p_\theta(x)
=T(x)-\E_{p_\theta}[T(X)],
\]

即数据统计量与模型统计量之差。这与指数族的 moment matching、CRF 的特征计数梯度是同一个结构。差别在于期望能否借助图结构精确计算。

\subsection{RBM 展示了局部条件简单而全局学习仍困难}

二值 restricted Boltzmann machine 由可见变量 \(v\) 和隐变量 \(h\) 构成，能量可写为

\[
E(v,h)
=-a^{\trans}v-b^{\trans}h-h^{\trans}Wv.
\]

因为同层没有边，给定 \(v\) 后各隐单元条件独立：

\[
p(h_i=1\mid v)
=\sigma\bigl(b_i+W_{i:}v\bigr),
\]

给定 \(h\) 后各可见单元也条件独立：

\[
p(v_j=1\mid h)
=\sigma\bigl(a_j+W_{:j}^{\trans}h\bigr).
\]

所以交替 Gibbs 更新很便宜。这种“条件分布容易采样”来自二部图结构，不代表边缘 \(p(v)\) 或配分函数容易计算。正相中的 \(p(h\mid v)\) 可以精确求期望，负相仍需要模型样本。

contrastive divergence 从数据样本初始化 Gibbs 链，只走少量步便用重构样本近似负相。它降低了成本，但短链分布一般不是当前模型的平稳分布，因此更新不是精确 maximum-likelihood gradient。重构看起来相似，只说明链在数据附近短期移动不远，不能证明模型已为全状态空间分配合理概率。persistent CD 保留跨批次链状态，可能更接近平稳采样，也仍受混合缓慢和多峰能量 landscape 影响。

\subsection{能量模型、判别模型与 VAE 的边界}

能量函数是一种参数化方式，不等同于某一种网络。根据归一化范围不同，它可以扮演不同角色：

\begin{itemize}
\tightlist
\item 无条件能量模型在所有 \(x\) 上归一化，目标是 \(p(x)\)；
\item 条件能量模型对固定输入的候选 \(y\) 归一化，得到 \(p(y\mid x)\)，线性链 CRF 就是结构化例子；
\item softmax classifier 对有限类别的 logits 归一化，也可视为一个配分函数很小、能精确计算的条件能量模型。
\end{itemize}

VAE 则显式定义联合生成过程

\[
p_\theta(x,z)=p(z)p_\theta(x\mid z),
\]

并用 \(q_\phi(z\mid x)\) 近似 latent posterior。它的主要困难是对 \(z\) 的边缘化与 posterior inference，ELBO 提供可训练下界。一般能量模型的主要困难是全状态空间归一化和从模型分布取样。两类方法都可能使用潜变量、神经网络和变分近似，却不能因为都“生成数据”就混成同一目标。

若只需排序候选，未归一化能量有时已经够用；若要报告概率、比较不同输入的 likelihood、生成样本或进行校准，归一化与采样质量就不能跳过。选择能量模型的理由应是相对相容性容易表达，而不是把不可计算的 \(Z\) 当成一个可以从公式中省略的常数。

\subsection{怎样判断近似训练是否可信}

训练 objective 下降只是第一层检查。对能量模型还应观察独立链是否混合、模型样本是否覆盖数据的多个模态、不同初值下统计量是否一致，并在小状态空间上枚举 \(Z\) 核对梯度。若使用短链近似，应明确链长和初始化，比较增加链长是否改变结论。

最危险的失败是“局部重构很好、全局分布错误”：模型在每个训练点附近形成很深的能量谷，却在谷之间或远离数据处放置大量未发现的概率质量。配分函数提醒我们，一个样本的概率由它的分数相对于全部可能状态共同决定。
