在\hyperref[sec:09-Convex-Optimization/07-Applications/03]{``最大间隔与支持向量机''}的结尾，对偶问题暴露出一个容易被忽略的观察：训练数据从不单独出现，只以内积 \(\ip{x_i}{x_j}\) 的形式出现。把内积换成某个更一般的函数 \(k(x_i,x_j)\)，线性分离面就悄悄搬到了另一个空间。这个观察能走多远？它能解释多少已经学过的方法，又需要付出什么代价？

这一单元沿着这个问题推进。起点是线性模型的表达力边界：有些数据在任何直线、任何平面下都分不开，但升到高维后却变得线性可分，于是``升到多高、怎样升''成了核心矛盾。出路是不显式构造高维特征，只通过内积访问它，这就是核技巧。随后必须回答一个理论问题：随手写出的相似度函数，什么时候真的是某个空间里的内积？正定核与再生核希尔伯特空间给出严格答案，表示定理则把无穷维的函数优化塌缩成 \(n\) 维的系数问题。最后落到计算：核岭回归给出闭式解，而 Gram 矩阵的存储与求逆代价划定了朴素核方法的规模边界。

\begin{enumerate}
\tightlist
\item
  把数据升到高维为什么有用：XOR 这类数据在低维线性不可分，显式特征映射让它可分，但维度爆炸使显式构造不可行，逼出``只算内积''的出路。
\item
  核技巧：只通过内积看世界：定义核函数，盘点 SVM 对偶、Ridge 回归的 Gram 形式与 PCA 中只依赖内积的环节，给出常见核与``自己设计核''的正定性判据。
\item
  再生核希尔伯特空间：把函数当成点：从 \(k(x,\cdot)\) 张成的空间出发构造内积与完备化，说明求值泛函连续意味着什么，并用表示定理把无穷维优化塌缩成 \(n\) 维问题。
\item
  核岭回归与核方法的代价：用表示定理推出核岭回归闭式解，与 Ridge 的原始形式对照，并讨论 Gram 矩阵的 \(O(n^2)\) 存储、\(O(n^3)\) 求逆以及 Nyström 与随机特征两条出路。
\item
  Gaussian process 把核变成函数先验：同一个核不再只定义相似度与范数，还定义函数值的联合协方差；Gaussian 条件化给出预测均值与模型内不确定性，边缘似然和计算代价则说明这种概率解释并非免费。
\end{enumerate}

\subsection{怎么读这一章}

如果只是想随手拿起这一章，先读每篇开头的“核心洞见”，再看第一篇的 XOR、第二篇的核替换规则、第四篇的两笔规模账和第五篇的 posterior 均值—方差；这条路线回答“为什么有用、怎样计算、何时不可信”。想理解方法为何成立，再读正定性判据、RKHS 的再生性质与表示定理。需要实现或建模时，最后检查核参数、正则强度、数值条件、样本规模与分布外外推；这些边界比记住某个闭式解更重要。

阅读时可以始终追问四个问题：\(\phi\) 到底是谁、是否真的存在？每一步计算是否只用到内积？范数惩罚究竟在惩罚什么？样本量翻倍时，哪一步先撑不住？核方法的整套理论，正是在这四个问题上反复接受检验。

完成这一单元后，建议回到\hyperref[ch:118]{``统计学习理论：训练误差对真风险有多乐观''}，检验由 \(\norm{f}_{\mathcal{H}_k}\) 控制的函数类在泛化界中扮演什么角色。
