高维联合分布往往写得出，却无法直接存储或计算。概率图模型用图编码因子分解和条件独立，把``哪些变量直接相互作用''从一张巨大概率表中分离出来。有向图强调生成顺序，无向图强调局部相容与势函数；图上的缺边只有在相应 Markov 条件下才表示条件独立。

本单元从三层推进。第一层建立 Bayesian network、Markov random field、d-separation、因子图与配分函数的共同语言，再用 Gaussian 图模型说明 covariance 与 precision 的零分别编码哪一种独立；第二层处理序列模型，从 HMM 走向线性 Gaussian 状态空间模型、Kalman filtering 和 smoothing；第三层比较生成式序列模型与条件随机场，解释归一化范围、动态规划和特征设计怎样改变学习与推断。

已有的 HMM 正文留在``概率模型与隐变量''作为离散隐状态入口。本单元省去对前向---后向算法的重复讲解，先从 precision matrix 的静态条件独立进入线性 Gaussian 动态系统，再追问两个更一般的问题：状态是连续且非线性时怎样保留不确定性，标签由整段输入共同决定时怎样直接建模 \(p(y\mid x)\)。最后一节把 HMM、线性状态空间模型、CRF 与 self-attention 放在同一坐标系中，防止因为它们都处理序列就误认为彼此只是不同实现。

\subsection{怎么读这一章}

首次阅读先抓``分解—独立—消元''：第一篇理解图如何声明联合分布的局部因子，第二篇区分 covariance 零与 precision 零，第三、四篇观察链结构怎样让 filtering、smoothing、配分函数与解码可由动态规划完成。最后一篇用于选型回顾。d-separation 的完整规则和消息传递细节可以第二遍练习，但必须从一开始区分统计图、推断算法与因果图；箭头和缺边只有在所声明的 Markov 与因果假设下才有相应含义。
