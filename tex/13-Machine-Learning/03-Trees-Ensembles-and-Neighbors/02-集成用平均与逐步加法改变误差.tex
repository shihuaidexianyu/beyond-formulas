\section{集成用平均与逐步加法改变误差}
\label{sec:13-Machine-Learning/03-Trees-Ensembles-and-Neighbors/02}

\begin{quote}
\textbf{先抓住这一点：}Bagging 平均许多不稳定模型以降低方差，Boosting 逐步加入模型以修正当前残差或梯度。二者都叫集成，却在改变不同的误差来源。
\end{quote}

一棵深树方差很大：训练数据稍变，早期切分就可能改变。集成方法通过组合许多有差异的学习器，放弃寻找唯一一棵完美树。Bagging 主要通过平均降低方差，Boosting 则逐步添加函数以修正当前残差。二者都使用许多树，却不是同一个原理。

\subsection{平均为何需要学习器不完全相关}

设 \(M\) 个回归预测器在某个 \(x\) 处具有相同方差 \(\sigma^2\)，两两相关系数为 \(\rho\)。平均预测

\[
\bar f(x)=\frac1M\sum_{m=1}^{M}f_m(x)
\]

的方差为

\[
\Var(\bar f(x))
=\rho\sigma^2+\frac{1-\rho}{M}\sigma^2.
\]

独立部分随 \(M\) 增大而消失，共同波动 \(\rho\sigma^2\) 不会。仅仅复制同一个模型不能降低方差；成员既要有一定准确性，也要制造差异。

\subsection{Bagging 用 bootstrap 扰动训练数据}

Bagging 从原训练集有放回抽取多个 bootstrap 样本，在每份样本上训练一棵树，再平均回归预测或投票分类结果。每个 bootstrap 样本平均只包含约

\[
1-\left(1-\frac1n\right)^n\longrightarrow 1-e^{-1}
\]

比例的不同训练点，其余样本可作为该棵树的 out-of-bag 评价数据。

Bootstrap 不会创造新的总体信息。它通过模拟训练数据的抽样波动，让不稳定算法产生不同版本；若基学习器本来十分稳定，扰动后的预测几乎相同，平均收益有限。

\subsection{随机森林进一步降低树之间相关性}

若某个强特征在多数 bootstrap 样本上都主导根节点，许多树仍会高度相似。随机森林在每个节点只允许随机选出的部分特征参与候选切分，使强特征不能在所有树的同一位置出现。

这可能略微增加单棵树偏差，却降低树间相关，最终平均方差反而更小。特征子采样数体现的是``单树强度''和``成员差异''之间的权衡，不是越大越充分。

Out-of-bag 误差能复用未进入某棵树的样本，但如果特征处理和超参数选择已看过所有数据，它不再是整个流程完全独立的测试。OOB 重要性也会受相关特征影响：两个特征可互相替代时，打乱其中一个的边际损害可能很小，不能据此断言它没有信息。

\subsection{Boosting 在函数空间中逐步修正}

考虑平方损失加法模型

\[
F_M(x)=\sum_{m=0}^{M}\eta h_m(x).
\]

在第 \(m\) 轮，当前残差为

\[
r_i^{(m)}
=y_i-F_{m-1}(x_i).
\]

拟合一棵小树 \(h_m\) 预测残差，再更新

\[
F_m(x)=F_{m-1}(x)+\eta h_m(x).
\]

对一般可微损失，残差推广为负梯度

\[
r_i^{(m)}
=-\left.
\frac{\partial \ell(y_i,F(x_i))}
{\partial F(x_i)}
\right|_{F=F_{m-1}},
\]

因此 gradient boosting 是在函数值空间沿经验风险负梯度构造新方向。基学习器未必能精确表示该梯度，只是在允许的树类中拟合它。

\subsection{小步和浅树共同限制函数增长}

学习率 \(\eta\)、树深度和轮数共同决定 Boosting 的有效复杂度。较小 \(\eta\) 通常需要更多轮，却让每次修正更温和；深树允许单轮拟合高阶交互，也更容易追随噪声。

训练损失往往随轮数继续下降，验证损失却可能回升。早停是沿迭代路径选择模型，停止轮数必须由验证数据决定。看完测试曲线再选最佳轮数，等价于让测试集参与训练。

Bagging 的成员可近似并行训练，核心是对不稳定估计求平均；Boosting 的下一步依赖当前残差，核心是序贯函数优化。把两者都概括成``多棵树投票''，会丢掉它们对误差来源的不同处理。
