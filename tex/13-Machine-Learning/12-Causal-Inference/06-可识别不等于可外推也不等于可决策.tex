\section{可识别不等于可外推也不等于可决策}

因果效应估计常以一个数字和标准误结束，但可信结论至少经过四道门：目标定义清楚，因果量由数据分布可识别，有限样本估计足够准确，结果能够迁移到真正关心的决策环境。任何一道门关闭，后面的精度都不能倒推前面的正确。

\subsection{隐藏混杂是识别问题，不是普通残差}

若存在未观察变量 $U$ 同时影响 $X$ 与 $Y$，则

\[
X\longleftarrow U\longrightarrow Y
\]

这条后门路径无法由已测协变量阻断。扩大同一观察机制下的样本只会更精确地估计带混杂的关联。

可行回应包括利用随机化或准实验设计，寻找有论证基础的工具变量、负对照或代理变量，以及对未测混杂强度做敏感性分析。它们各有自己的排除限制或结构假设。敏感性分析不能证明无偏，却能回答：一个未测混杂需要多强，才足以推翻当前结论？

\subsection{重叠失败意味着目标人群超出证据支持}

若某些 $C=c$ 下只有处理者，没有对照者，则 $m_0(c)$ 无法由同类样本学习。复杂模型仍会输出数值，那只是结构外推，不是数据提供的反事实比较。

可以把目标改为共同支持区域中的人群、使用 overlap weighting，或明确引入可辩护的响应模型进行外推。改变目标人群可能提升可信度，却也改变 estimand；不能删去困难样本后仍声称估计原总体 ATE。

\subsection{内部有效不保证目标人群有效}

试验可能在参与者中准确识别效应，却未必适用于更老、更病重或资源不同的人群。设 $S=1$ 表示进入研究，目标总体可能有不同的 $C$ 分布，甚至不同的条件效应

\[
\tau(c)=\E[Y(1)-Y(0)\mid C=c].
\]

若效应只随已测 $C$ 改变，并且目标总体在这些 $C$ 上有支持，可以对研究内条件效应重新加权；若选择进入研究还依赖未测 effect modifier，transportability 仍缺少依据。

“在独立测试集上复现”主要验证同一数据机制下的预测或估计稳定性；跨地区、跨时期和跨制度迁移还要解释哪些结构机制保持不变。

\subsection{异质效应不是自动的个体化处方}

条件平均处理效应 $\tau(c)$ 可以揭示人群差异，但数据驱动地搜索许多亚组会产生选择偏差和高方差。灵活模型还可能把基线风险差异误读为处理效应差异。应使用预先指定或诚实分割的异质性分析，并给出不确定性，而不是只展示效应最大的叶节点。

从效应到政策还需要行动成本与约束。若 $\pi(c)\in\{0,1\}$ 是处理规则，其价值可写成

\[
V(\pi)=\E\bigl[Y(\pi(C))\bigr].
\]

即使 $\tau(c)>0$，处理成本、副作用、容量和公平约束也可能使最优行动不是全部处理。因果效应描述改变结果的方向和大小；决策规则还要把所有相关后果放在同一个效用尺度中。

数理统计中的``统计决策与可信预测：模型给出概率之后怎样行动''提供状态、行动、损失、Bayes risk 与容量约束的静态语言；本单元补充的是行动会改变哪一个结果分布。两者缺一不可：决策理论不能用观察风险代替处理效应，因果效应也不能独自决定哪种后果值得付出多少代价。

\subsection{时间变化与相互干扰需要更大的模型}

纵向决策中，先前处理会改变后续协变量，后续协变量又影响下一次处理和结果。把这些变量当普通静态混杂因素回归，可能同时控制中介并留下新的选择偏差。G-formula、marginal structural model 和 sequential ignorability 正是为这种 treatment--confounder feedback 设计的。

网络、传染和市场中，一个单位的处理影响另一个单位的结果，标准 no-interference 假设失效。此时必须定义暴露映射、集群或网络干预；“处理组减对照组”不再对应单一稳定的个体效应。

这些扩展不是把公式写得更复杂而已。它们提醒我们：潜在结果的索引必须包含所有真正改变结果的干预维度。

\subsection{一份可审查的因果结论应分层报告}

至少应分别说明：

\begin{itemize}
\tightlist
\item
  estimand：处理版本、结果时间窗、目标人群和效应尺度；
\item
  design：处理怎样分配，样本怎样进入，时间顺序和缺失机制是什么；
\item
  identification：一致性、无混杂、重叠、无干扰或其他设计假设为何可信；
\item
  estimation：回归、匹配、加权或增广估计如何实现，标准误怎样得到；
\item
  diagnostics：协变量平衡、权重尾部、共同支持、模型敏感性和负面对照；
\item
  transport and decision：结论适用于谁，哪些环境变化会使它失效，行动还受什么代价约束。
\end{itemize}

因果推断并不把观察数据变成实验。它做的是更克制也更有价值的工作：把“若改变 $X$，$Y$ 会怎样”拆成明确目标、可见数据和额外假设，让无法由数据单独保证的部分不再伪装成一个普通回归输出。
