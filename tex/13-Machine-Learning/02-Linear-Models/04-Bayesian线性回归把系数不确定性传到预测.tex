\section{Bayesian 线性回归把系数不确定性传到预测}
\label{sec:13-Machine-Learning/02-Linear-Models/04}

\begin{quote}
\textbf{先抓住这一点：}Bayesian 线性回归不是只给一组最佳系数，而是更新整个系数后验，再把参数不确定性传到预测分布。区间宽度仍依赖先验、噪声与线性模型是否合理。
\end{quote}

普通最小二乘给出一个系数向量 \(\widehat\beta\)，经典区间用重复抽样分布描述估计程序的波动。Bayesian 线性回归改为给系数指定 prior，观察数据后得到参数 posterior，再把这份不确定性积分进新样本预测。三者可以共享同一个线性均值模型，却回答不同概率问题。

先从噪声方差已知的情形开始：

\[
y\mid\beta
\sim
\mathcal N(X\beta,\sigma^2I),
\qquad
\beta
\sim
\mathcal N(m_0,S_0),
\]

其中 \(S_0\) 正定，设计矩阵 \(X\) 视为给定。截距若包含在 \(\beta\) 中，可以给它与其他系数不同的 prior scale，避免无意中过强收缩。

\subsection{Posterior 来自两个二次型相加}

忽略与 \(\beta\) 无关的常数，log posterior 为

\[
\log p(\beta\mid y,X)
=
-\frac1{2\sigma^2}
\norm{y-X\beta}_2^2
-\frac12
\norm{\beta-m_0}_{S_0^{-1}}^2
+C,
\]

其中

\[
\norm{u}_{M}^2=u^{\trans}Mu.
\]

展开并收集关于 \(\beta\) 的二次项和一次项，定义 posterior precision

\[
\Lambda_n
=S_0^{-1}
+\sigma^{-2}X^{\trans}X
\]

以及 information vector

\[
h_n
=S_0^{-1}m_0
+\sigma^{-2}X^{\trans}y.
\]

完成平方可得

\[
S_n=\Lambda_n^{-1},
\qquad
m_n=S_nh_n,
\]

\[
\beta\mid y,X
\sim
\mathcal N(m_n,S_n).
\]

Prior precision 与 data precision 相加，正是 Gaussian 共轭更新的多元版本。数据只在 \(X^{\trans}X\) 有信息的方向增加精度；样本没有辨认的方向仍由 prior 控制。

\subsection{MAP 与 Ridge 相同只在特定参数化下成立}

Posterior mode 等价于最小化

\[
\frac1{2\sigma^2}
\norm{y-X\beta}_2^2
+\frac12
\norm{\beta-m_0}_{S_0^{-1}}^2.
\]

若 \(m_0=0\)、\(S_0=\tau^2I\)，乘以 \(2\sigma^2\) 后得到

\[
\norm{y-X\beta}_2^2
+\lambda\norm{\beta}_2^2,
\qquad
\lambda=\frac{\sigma^2}{\tau^2}.
\]

所以 isotropic zero-mean Gaussian prior 的 MAP 与 Ridge 解相同。在当前 Gaussian posterior 中，mean 与 mode 也恰好相等；对偏斜或多峰 posterior，这两个点一般不同。

``Ridge 等于 Bayesian regression''仍不完整。Ridge 通常只使用 MAP 点，完整 Bayesian 分析保留 \(S_n\) 并在预测时积分参数。相同的最优系数不意味着相同的不确定性输出。

Proper Gaussian prior 还使 \(X^{\trans}X\) 秩亏时 posterior precision 仍可逆。此时可得到唯一 posterior 和预测分布，但某些方向主要由 prior 而非数据辨认；数值可解不能被误读为数据已经识别全部系数。

\subsection{Posterior predictive 分开两种不确定性}

对新特征向量 \(x_*\)，模型规定

\[
Y_*
=x_*^{\trans}\beta+\varepsilon_*,
\qquad
\varepsilon_*\sim\mathcal N(0,\sigma^2).
\]

将 \(\beta\) 对 posterior 积分：

\[
Y_*\mid x_*,X,y
\sim
\mathcal N\left(
x_*^{\trans}m_n,
\sigma^2+x_*^{\trans}S_nx_*
\right).
\]

预测方差中的

\[
\underbrace{\sigma^2}_{\text{观测噪声}}
+
\underbrace{x_*^{\trans}S_nx_*}_{\text{参数不确定性}}
\]

语义不同。增加同分布数据通常会缩小第二项，却不会消除世界本身的观测噪声。Plug-in prediction 只使用 \(x_*^{\trans}m_n\) 和 \(\sigma^2\)，会漏掉第二项。

对一组新设计矩阵 \(X_*\)，联合预测为

\[
y_*\mid X_*,X,y
\sim
\mathcal N\left(
X_*m_n,
\sigma^2I+X_*S_nX_*^{\trans}
\right).
\]

不同新预测之间因共享同一 \(\beta\) 而相关。逐点区间无法表达多个预测共同升高或降低的 posterior 结构。

\subsection{不确定性沿特征方向变化，而不只随距离变化}

若 \(x_*\) 指向数据充分覆盖的特征方向，\(x_*^{\trans}S_nx_*\) 较小；若它含有训练设计几乎没有激发的方向，该项可能很大。所谓``离训练数据远''应在设计矩阵和 prior 诱导的几何中理解，而不是只看原始坐标的 Euclidean distance。

若特征先做标准化，prior scale 的含义更一致；若未标准化，对所有系数使用相同 \(\tau\) 会对不同单位施加完全不同的实际收缩。改变单位不应悄悄改变科学结论，因此 prior 必须和特征变换一起记录。

多项式外推还会出现另一种危险：posterior interval 可能变宽，却仍建立在线性均值形式正确这一前提上。若模型遗漏结构，区间再宽也未必覆盖真实关系。Bayesian uncertainty 是给定模型后的条件不确定性，不自动包含模型类别错误。

\subsection{未知噪声方差会产生 Student-(t) 预测尾部}

实际中 \(\sigma^2\) 通常未知。若使用与 \(\sigma^2\) 相配的 normal--inverse-gamma prior，可以同时更新系数与噪声方差，并在积分二者后得到 multivariate Student-(t) posterior predictive。相对把 \(\sigma^2\) 固定为点估计，Student-(t) 尾部额外反映有限数据下的方差不确定性。

若只给 \(\sigma^2\) 一个极弱 improper prior，需要检查 posterior 是否 proper；样本数不足或设计秩亏时，形式上的积分可能发散。所谓``无信息先验''也不是免费删除假设。

\subsection{数值实现不应形成显式逆矩阵}

公式写 \(S_n=\Lambda_n^{-1}\) 是为了描述 covariance。计算 posterior mean 时应解

\[
\Lambda_n m_n=h_n,
\]

采样或计算 log determinant 时使用 Cholesky 分解。若特征数 \(p\) 远大于样本数 \(n\)，可利用 Woodbury identity 将主要线性系统改写到 \(n\times n\) 空间；选择 primal 或 dual 形式应由维度与条件数决定。

模型检查不能只看 posterior 是否窄。应做 posterior predictive checks，比较生成数据与真实数据的残差形状、异方差、尾部和分组结构；改变合理 prior scale 检查敏感性；在独立数据上核对 predictive interval coverage。若区间系统性漏掉某一类样本，增加 MCMC 样本或提高线性代数精度都不能修复错误 likelihood。

Bayesian 线性回归把前面三条线接在一起：最小二乘提供线性几何，Ridge 提供收缩，Gaussian 共轭提供完整 posterior。下一单元的指数族与共轭先验会说明，这种``precision 加 precision''的闭式更新只是更一般代数结构的一种具体实现。
