线性模型常被当作``最简单的机器学习算法''，但简单并不等于浅薄。它把一个完整的建模过程压缩到了最容易看清的形式：怎样把对象表示成特征，怎样规定误差，怎样从数据估计参数，怎样控制不稳定性，又怎样把数值输出转化为决策。更复杂的模型仍然要回答这些问题，只是答案更难观察。

这一单元不按算法名称逐项介绍，而沿着模型遇到的困难向前推进。

\begin{enumerate}
\tightlist
\item
  线性回归从投影到概率模型：从``找一条直线''进入列空间投影、噪声假设和预测诊断。重点是区分拟合成立、统计推断成立和因果解释成立所需的不同条件。
\item
  正则化从 Ridge 到 Lasso：当特征近似共线或维数过高时，数据不能稳定决定系数。Ridge 与 Lasso 分别对``允许怎样的参数''作出不同选择，可以放在同一建模问题中比较。
\item
  从二分类到 Softmax 分类：连续响应变成类别后，平方误差不再自然。Bernoulli 似然、log-odds 和 softmax 会把线性分数变成概率，并把训练、阈值和校准连接起来。
\item
  Bayesian 线性回归把系数不确定性传到预测：Gaussian prior 不只产生 Ridge 型 MAP，还给出完整参数 posterior，并把观测噪声与参数不确定性分别带入 posterior predictive。
\item
  GLM 用链接函数连接均值与线性预测子：把 Gaussian、Bernoulli 与 Poisson 响应放进指数族框架，区分随机成分、线性预测子和链接函数，并从似然得到加权迭代。
\end{enumerate}

阅读时可以始终追问同一组问题：模型把什么当作固定、把什么当作随机？目标函数从何而来？参数能否被数据唯一识别？优化器求得的数值解是否可靠？验证集上的好表现又能否迁移到真正关心的人群？线性模型的价值，正是让这些问题无法被网络深度或算法名称遮住。

\subsection{怎么读这一章}

若只走主干，先读线性回归、正则化和分类：它们分别展示投影与噪声假设、稳定性与复杂度、以及从分数到概率的连接。Bayesian 线性回归和 GLM 可作为两条扩展：前者追踪参数不确定性，后者更换响应分布与链接函数。第一遍无需记住每个闭式解；只要能从假设重建目标，并说明系数稳定、预测有效和因果解释为何不是同一件事，就达到了本章的第一层目标。

完成这一单元后，可进入\hyperref[ch:112]{``降维与低秩建模''}，研究特征表示本身能否被更少的潜在方向解释，而非急于换一个更复杂的预测模型。Bayesian 线性回归中出现的 Gaussian 共轭更新，会在\hyperref[ch:114]{``概率模型与隐变量''}的指数族与共轭先验两节中推广；GLM 则说明``线性''真正约束的是链接尺度上的系统部分，并不要求观测本身服从正态分布。
