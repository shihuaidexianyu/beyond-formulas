\section{线性代数：寻找让问题变简单的坐标}

线性代数常被误解成一门矩阵计算课。消元、行列式、特征值当然要会算，但这些计算背后的共同问题是：\textbf{一个线性关系保留了什么，丢失了什么；换到什么坐标后，它的结构会显露出来？}

例如，方程组无解或有无穷多解并非三个互不相关的例外，而是列空间与零空间共同决定的结果；最小二乘并非硬套正规方程，而是在列空间里寻找最近点；特征向量、SVD 和 Fourier 基看似属于不同章节，实质上都在寻找能让变换分解为独立标量作用的坐标。

这套笔记按这种认识逐步展开，而不是按术语分类。读者只需要基本代数即可开始；涉及微积分和数据分析的应用会在需要时补充背景。

\subsection{第一段：从解方程到理解空间}

从线性方程组与矩阵开始。我们先把矩阵看成方程组的组织方式，用消元判断是否有解、自由度在哪里，再从重复计算中引出逆矩阵与 LU 分解。重点不是背``有解的三种情况''，而是看主元、自由变量和解集几何怎样互相对应。

随后进入向量空间、基与线性变换。这一单元解释为什么解集、函数、多项式和数据都可以用同一套语言研究，并把消元中隐约出现的列空间、零空间、秩和维数正式连起来。学完后，矩阵不再只是数表，而是某个线性变换在选定基下的坐标表示。

\subsection{第二段：从几何最近点到长期行为}

正交、投影与最小二乘把``长度、垂直和最近''加入代数结构。投影从一个无法精确满足的方程中找出最佳近似，Gram--Schmidt 和 QR 则把任意独立方向整理成数值上更可靠的正交坐标。这部分是统计回归、数值计算和优化的共同入口。

行列式只保留它真正承担的任务：有向体积的缩放因子。行交换为何变号、三角矩阵为何看对角线、乘法公式为何成立，都从这一解释导出。之后，行列式作为``可逆性检测器''和特征多项式的工具自然出现。

特征值、对角化与动力系统研究反复作用同一个变换时什么方向保持不变。对角化使矩阵幂和线性动力系统变成若干独立的一维过程；Jordan 形则解释当特征向量不够时，为什么还会出现多项式增长。

\subsection{第三段：从特殊结构到普适分解}

一般矩阵的特征向量可能不正交，甚至不够组成基；对称矩阵、二次型与正定性展示了一个重要例外。谱定理把对称矩阵变成沿正交方向的独立伸缩，二次型把这些伸缩解释为能量与曲率，Cholesky 分解再把正定性转成可计算的平方结构。

但数据矩阵通常是长方形的。SVD 与数据中的主要方向不再要求输入和输出处于同一个空间：任何矩阵都能分成正交输入方向、非负伸缩和正交输出方向。伪逆、条件数、最佳低秩近似和 PCA 都由这一个分解生长出来。

复数、谱与 Fourier 结构把旋转和相位纳入标量运算。共轭内积恢复长度，Hermitian 与正规矩阵恢复正交谱结构，循环移位的特征基则导出 DFT、卷积定理和 FFT。最后，张量把双索引矩阵推广到多模结构：先区分多线性对象与选基后的数组，再用 CP 与 Tucker 说明 SVD 的哪些思想能够推广、哪些整齐性质只属于二阶。它们共同揭示全课程的核心：好的坐标和结构化表示不是装饰，会把复杂耦合变成可解释的简单作用。

\subsection{学完之后应当能做什么}

完成这条主线后，读者不只应当会计算，还应当能够：

\begin{itemize}
\tightlist
\item
  从矩阵的四个基本子空间判断方程的可解性、自由度与信息损失；
\item
  在精确解不存在时，把投影、最小二乘和伪逆解释成同一个几何选择；
\item
  根据矩阵结构选择 LU、QR、Cholesky、特征分解或 SVD，而不是见到矩阵就求逆；
\item
  区分``问题本身病态''和``算法额外不稳定''；
\item
  把对角化、PCA 与 Fourier 变换看成针对不同结构选择坐标的实例；
\item
  说明每个定理的条件为何必要，并用反例识别越界使用。
\end{itemize}
