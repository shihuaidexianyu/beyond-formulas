线性代数最初要解决的是同时处理许多彼此关联的一次方程。二维数组只是矩阵的外观；矩阵把方程中的系数、列向量的组合和线性变换统一在同一个对象中。消元能够在不改变解集的前提下暴露方程中真正独立的约束；LU 分解则把一次消元保存下来，供许多不同右端重复使用。

本单元沿着``表示问题---化简问题---复用计算''的顺序展开。

\subsection{阅读顺序}

\begin{enumerate}
\item
  从方程到矩阵\\
  从同一个方程组的行图像和列图像出发，理解 \(Ax=b\)、矩阵的列以及矩阵乘法为什么这样定义。
\item
  消元与解的结构\\
  通过 Gaussian 消元寻找主元，判断唯一解、无穷多解与无解，并把全部解写成``特解加齐次解''。
\item
  逆矩阵与 LU 分解\\
  理解可逆性意味着什么，怎样把行操作写成矩阵，以及为何 \(PA=LU\) 是重复求解线性系统的基本工具。
\end{enumerate}

本单元只要求基本代数。读完后，读者应能在行视角、列视角和变换视角之间切换，并能解释消元过程中的主元、自由变量、秩、置换和三角分解各自在解决什么问题。
