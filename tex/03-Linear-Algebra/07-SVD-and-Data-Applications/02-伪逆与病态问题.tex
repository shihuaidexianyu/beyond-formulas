\section{伪逆与病态问题}

\subsection{当``求逆''这个问题本身问错了}

若 \(A\) 是可逆方阵，\(Ax=b\) 的解是 \(x=A^{-1}b\)。但现实中的矩阵经常不方、不满秩，或者方程因测量误差而根本无解。此时继续寻找普通逆矩阵，就像要求一张压扁后的照片恢复出已经丢失的深度：真正的障碍在于映射已经丢失信息，再高明的技巧也无法凭空恢复。

SVD 把这种情况拆得很清楚。若

\[
A=U_r\Sigma_rV_r^\top,
\]

那么输出在 \(u_i\) 方向上的分量来自输入在 \(v_i\) 方向上的分量，并被乘了 \(\sigma_i\)。只要 \(\sigma_i>0\)，就可以在这个方向上除以 \(\sigma_i\)；零奇异值对应的信息则不能凭空恢复。由此定义 Moore--Penrose 伪逆

\[
A^+=V_r\Sigma_r^{-1}U_r^\top
=\sum_{i=1}^{r}\frac{1}{\sigma_i}v_i u_i^\top.
\]

它只反转那些确实可反转的方向，全程没有假设 \(A\) 可逆。

\subsection{伪逆到底选出了哪个解}

对任意 \(b\)，\(AA^+b\) 是 \(b\) 在列空间 \(C(A)\) 上的正交投影：直接乘开得 \(AA^+=U_r\Sigma_rV_r^\top V_r\Sigma_r^{-1}U_r^\top=U_rU_r^\top\)，而 \(U_r\) 的列是 \(C(A)\) 的标准正交基，\(U_rU_r^\top\) 正是到该空间的正交投影。因此

\[
x^+=A^+b
\]

总能最小化残差 \(\|Ax-b\|_2\)。如果有多个最小二乘解，它还会在其中选择范数最小的一个。

先看超定方程

\[
\begin{bmatrix}1\\1\end{bmatrix}x
\approx
\begin{bmatrix}1\\2\end{bmatrix}.
\]

没有一个 \(x\) 能同时等于 1 和 2。伪逆给

\[
x^+=\frac{1}{2}\begin{bmatrix}1&1\end{bmatrix}
\begin{bmatrix}1\\2\end{bmatrix}=\frac32,
\]

也就是把 \(b\) 投影到直线 \(\operatorname{span}(1,1)\) 后得到的最近解。

再看欠定方程 \(x_1+x_2=1\)。它有无穷多个解 \((t,1-t)\)。系数矩阵 \(A=[1\;1]\) 的伪逆是 \(A^+=\tfrac12(1,1)^\top\)，因此

\[
x^+=\begin{bmatrix}1/2\\1/2\end{bmatrix}.
\]

为什么是这一点？所有解都可写成 \(x^++z\)，其中 \(z\in N(A)\)，而 \(x^+\) 位于行空间 \(C(A^\top)=N(A)^\perp\)。两部分正交，所以添加任何非零的零空间分量都会增加长度。

\subsection{从可解性转向敏感性}

即使每个奇异值都非零，极小的奇异值仍会带来麻烦。矩阵

\[
A_\varepsilon=
\begin{bmatrix}1&0\\0&\varepsilon\end{bmatrix},
\qquad 0<\varepsilon\ll1,
\]

把第二方向压得几乎看不见。反解时却要除以 \(\varepsilon\)。若 \(b\) 的第二分量含有大小 \(10^{-6}\) 的测量误差，而 \(\varepsilon=10^{-8}\)，解的第二分量就会产生大小 \(100\) 的误差。方程有唯一解，却几乎无法从有限精度数据中可靠恢复。

为了准确描述``放大''，需要矩阵范数。二范数诱导的算子范数是

\[
\|A\|_2=\max_{x\ne0}\frac{\|Ax\|_2}{\|x\|_2}=\sigma_1,
\]

它测量最强方向的放大倍数。第二个等号来自 SVD：正交变换不改变长度，把 \(x\) 按右奇异向量展开后，\(\|Ax\|_2^2=\sum_i\sigma_i^2c_i^2\le\sigma_1^2\sum_ic_i^2\)，且 \(x=v_1\) 时取到等号。Frobenius 范数

\[
\|A\|_F=\sqrt{\sum_{ij}a_{ij}^2}
=\sqrt{\sum_i\sigma_i^2}
\]

则汇总所有方向的``能量''（后一等号因为 Frobenius 范数在左右乘正交矩阵下不变，可直接把 \(A\) 换成 \(\Sigma\)）。两者回答不同问题，不能因为都叫范数就互换。

对可逆方阵，二范数条件数为

\[
\kappa_2(A)=\|A\|_2\|A^{-1}\|_2
=\frac{\sigma_1}{\sigma_n}.
\]

后一等号是因为 \(A^{-1}=V\Sigma^{-1}U^\top\) 的奇异值恰为 \(1/\sigma_i\)，其中最大者为 \(1/\sigma_n\)。对满列秩的长方形最小二乘问题，也常用 \(\sigma_1/\sigma_r\) 描述有效方向之间的尺度差异。若只扰动 \(b\) 且 \(A\) 固定，可逆方程满足

\[
\frac{\|\delta x\|}{\|x\|}
\le
\kappa_2(A)
\frac{\|\delta b\|}{\|b\|}.
\]

推导只有两步：\(\delta x=A^{-1}\delta b\) 给出 \(\|\delta x\|\le\|A^{-1}\|_2\|\delta b\|\)，而 \(\|b\|=\|Ax\|\le\|A\|_2\|x\|\) 给出 \(1/\|x\|\le\|A\|_2/\|b\|\)，两式相乘即得。这是上界而非每次都达到的预言；它说的是问题对最坏方向的内在敏感性。算法是否还引入额外误差，是另一个问题。一个稳定算法不能把病态问题变得信息充足，它只能避免自己进一步恶化结果。

\subsection{为什么不应轻易形成正规方程}

最小二乘可写成 \(A^\top Ax=A^\top b\)，但

\[
\kappa_2(A^\top A)=\kappa_2(A)^2
\]

（在满列秩情形：\(A^\top A\) 的特征值是 \(\sigma_i^2\)，最大与最小奇异值都被平方）。这意味着原本已经很小的奇异值会被平方，数值敏感性进一步显露。因此实际计算常优先使用 QR 或 SVD，而不是显式计算 \((A^\top A)^{-1}A^\top\)。公式在精确算术中成立，并不意味着它是好的数值算法。

当小奇异值主要承载噪声时，可以截去这些方向，或用正则化减弱除法。代价也很明确：稳定性提高了，但我们主动接受了偏差。下一节的低秩近似会把这种``保留主要方向、舍弃次要方向''的选择变成一个精确的最优化结论。

\subsection{易混淆的边界}

\begin{itemize}
\tightlist
\item
  伪逆不是对矩阵元素逐个取倒数，也不等于在任何情况下写 \((A^\top A)^{-1}A^\top\)；后一式要求满列秩。
\item
  \(A^+A\) 投影到 \(A\) 的行空间，\(AA^+\) 投影到 \(A\) 的列空间，二者通常尺寸不同。
\item
  条件数大说明问题敏感，不自动证明某次计算已经错；条件数小也不免除建模误差。
\item
  当矩阵的秩因微小扰动而改变时，伪逆可能发生巨大变化。这正是零奇异值与``非常小但非零''之间的断崖。
\end{itemize}
