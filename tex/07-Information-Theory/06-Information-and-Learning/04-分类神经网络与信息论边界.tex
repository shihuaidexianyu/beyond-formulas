\section{分类、神经网络与信息论边界}

\subsection{交叉熵训练到底在优化什么}

分类模型输出条件分布 \(q_\theta(y\mid x)\)。总体负对数损失为

\[
\mathcal R(\theta)
=\mathbb E_{(X,Y)\sim P}
[-\log q_\theta(Y\mid X)].
\]

在期望号内加减 \(\log P(Y\mid X)\)，即把 \(-\log q_\theta(Y\mid X)\) 拆成 \(-\log P(Y\mid X)\) 与对数比 \(\log[P(Y\mid X)/q_\theta(Y\mid X)]\)，再分别取期望，可以得到

\[
\boxed{
\mathcal R(\theta)
=H_P(Y\mid X)
+\mathbb E_XD\bigl(P_{Y\mid X}\|q_{\theta}(\cdot\mid X)\bigr).
}
\]

第一项是观测 \(X\) 后标签仍然存在的不可约不确定性；第二项是模型条件分布与真实条件分布的失配。若模型类足够丰富且得到总体最优解，交叉熵在 \(q_\theta=P_{Y\mid X}\) 时最小。用最小描述长度一节的语言说，总体负对数损失就是 \(q_\theta\) 给数据安排的平均理想码长：第一项是真实分布自身的码长，失配项正是用错条件分布多付的码长。

one-hot 标签下，单样本损失简化为

\[
-\log q_\theta(y\mid x).
\]

这解释了为什么错误且高置信的预测会受到巨大惩罚。但单样本损失、总体交叉熵、分类错误率和校准是不同量：两个模型可能准确率相同，却因置信度质量不同而交叉熵不同；训练交叉熵下降也不保证分布外测试表现改善。

\subsection{特征中有多少标签信息}

设类别 \(Y\in\{1,\ldots,K\}\)，特征为 \(X\)。互信息

\[
I(Y;X)=H(Y)-H(Y\mid X)
\]

衡量特征平均消除了多少标签不确定性。若 \(X\) 与 \(Y\) 独立，任何分类器都无法从 \(X\) 中提取标签证据；若 \(Y\) 是 \(X\) 的确定函数，条件熵为零。

Fano 不等式把剩余不确定性与任何分类器的错误率联系起来。令 \(\hat Y=g(X)\)，\(P_e=P(\hat Y\ne Y)\)，则

\[
H(Y\mid X)
\le h_2(P_e)+P_e\log_2(K-1).
\]

粗略利用 \(h_2(P_e)\le1\)，再把 \(\log_2(K-1)\) 放大为 \(\log_2K\)，可反推出

\[
P_e\ge
\frac{H(Y\mid X)-1}{\log_2K}.
\]

它给出的是不可达区域：若特征留下很大的条件熵，任何算法都不可能把错误率压得任意低。它不是某个分类器一定能达到的误差公式，也通常不是紧界；完整 Bayes 错误取决于后验分布的具体形状，而不只取决于一个互信息标量。

\subsubsection{Fano 的关键想法}

令错误指示变量 \(E=\mathbf1\{Y\ne\hat Y\}\)。已知 \(X\) 后，若 \(E=0\)，标签由预测确定；若 \(E=1\)，最多还需在其余 \(K-1\) 类中选择。于是错误是否发生最多贡献 \(h_2(P_e)\) bit，错误发生后类别身份最多贡献 \(P_e\log_2(K-1)\) bit。这就是上式的来源。这是直觉论证：严格证明要先把 \(H(Y\mid X)\) 放大为 \(H(Y\mid\hat Y)\)（\(\hat Y\) 是 \(X\) 的函数，条件熵只会减小），再对 \(H(E,Y\mid\hat Y)\) 用链式法则展开并放缩；上面的计数说法正是这个放缩的直觉版本。

\subsection{深层表示不会创造标签信息}

对前馈表示

\[
X\to T_1\to T_2\to\cdots\to T_L,
\]

且标签只通过输入影响表示时，由数据处理不等式

\[
I(Y;T_{k+1})\le I(Y;T_k)\le I(Y;X).
\]

这似乎与``深层特征越来越好用''矛盾，其实比较对象不同。后续层不能增加总体可用的标签信息，却可以把信息重排成更适合线性分类器、有限样本或特定优化算法使用的形式。一个可逆旋转不改变互信息，却可能显著改变坐标轴对齐；一个有损非线性可能丢掉少量无关细节，同时让决策边界简单得多。

因此网络表现提升不需要互信息逐层增加。信息量、可访问性、几何结构和解码器复杂度是不同问题。

\subsection{单特征排名会漏掉模式}

XOR 再次提供一个最小反例。单独的 \(X_1\) 或 \(X_2\) 与标签互信息都是零，但二者联合完全决定标签。若只保留单变量信息增益最大的特征，可能一个也选不中；非线性网络却能组合二者形成有效表示。

另一方面，两个高度重复的特征各自互信息都很高，联合增加的条件互信息却很小。实际表示评价需要区分独特信息、冗余信息和协同信息，单一标量排序无法完整描述多变量结构。

\subsection{高维互信息估计不是免费测量}

理论中的 \(I(X;T)\) 假设联合分布已知。现实中只有有限样本，而高维连续密度估计本身就很困难。直方图、核方法、近邻估计和神经下界可能产生完全不同的数值；某些方法只提供下界，训练得更好时界变大，不等于真实互信息变大。

尤其在确定性连续网络中，互信息可能无穷或由量化规则决定——上一节已分析过这种病态的两个来源（低维流形与可逆映射）。若经验估计出现

\[
I(Y;T)>I(Y;X),
\]

通常应先检查估计偏差、采样误差与变量定义，而不是宣布数据处理不等式被神经网络推翻。

一个负责任的信息论分析至少要报告噪声或量化机制、估计器目标、上下界方向、有限样本基准和不确定性。互信息曲线可以是诊断线索，不能自动成为机制证明。

\subsection{把几个常混淆的判断分开}

\begin{itemize}
\tightlist
\item
  交叉熵低：模型给真实标签较高的概率，兼顾概率质量与置信度；
\item
  准确率高：最大概率类别经常正确，不评价其余概率如何分配；
\item
  校准好：预测为 \(70\%\) 的事件长期约有 \(70\%\) 发生；
\item
  \(I(Y;T)\) 高：表示在总体分布下含较多标签信息；
\item
  \(I(X;T)\) 低：在明确的随机变量定义下，表示保留较少输入信息。
\end{itemize}

这些性质会相关，却不互相等价。好的实验应分别测量与任务相关的量，而不是用``信息更多''替代所有评价。
