信息论最初关心一个非常实际的问题：消息经过有噪信道后，究竟能以多高的速率可靠传输？Shannon 的回答越过了某一种电路或某一种语言，把``消息的不确定性''\,``观察带来的信息''和``错误模型的代价''提炼成概率分布上的量。由此，熵不只解释压缩极限，互信息不只解释信道容量，也逐渐成为统计推断与机器学习中的共同语言。

但这些概念很容易被口号化。熵不是数据价值的万能评分，KL 不是普通距离，容量不是未编码传输的正确率，交叉熵下降也不自动保证泛化。本套笔记始终把公式放回它解决的问题：随机变量是什么，概率由谁给出，期望对谁取，极限需要哪些条件，数值究竟有何操作含义。

\subsection{怎么读这一章}

阅读前最好熟悉离散随机变量、条件概率、期望、大数定律和基本对数运算。相关内容见``概率论''部分。第一次阅读不需要先掌握测度论；连续熵和一般分布中的技术边界会在出现时说明。

主线由六段组成：

\begin{enumerate}
\tightlist
\item
  信息与熵从``为什么罕见结果更有信息''出发，建立自信息、Shannon 熵、微分熵和最大熵原理。
\item
  联合信息研究观察一个变量后另一个变量还剩多少不确定性，由链式法则走向互信息、Markov 结构和数据处理不等式。
\item
  分布之间的差异从错误概率模型造成的额外码长推导交叉熵与 KL，再解释 Gibbs 不等式、方向性和 Jensen--Shannon 散度。
\item
  无损信源编码把熵落到压缩：前缀码受什么约束，Huffman 为什么最优，算术编码如何累积分数 bit，典型集又怎样给出压缩极限。
\item
  有噪信道编码把互信息落到通信：信道与输入分布如何共同决定信息量，容量为何需要优化，以及低于容量为什么能可靠通信。
\item
  信息与学习回到推断与表示：信息增益、MDL、Bayesian 更新、信息瓶颈、分类界和神经网络分析怎样共享同一批概念，又各有哪些限制。
\end{enumerate}

这条顺序不是目录上的任意排列。前两单元建立基本量，第三单元解释模型失配，第四和第五单元分别回答``能压多短''和``能传多快''，最后一单元才把这些量迁移到学习问题。若只为某个应用查阅，也建议至少先读相关概念第一次被推导的章节，而不是只记最终公式。

\subsection{两条贯穿全科的第一性原则}

第一条是把信息看作排除可能性的能力。概率越小的结果越令人惊讶；观察若能显著改变后验，就提供更多信息；信道若让不同输入产生可区分的输出，就能传递消息。自信息、互信息、实验信息增益和信道容量都是这条思想在不同随机结构中的表达。

第二条是把概率翻译为码长。理想码长为 \(-\log p\)，平均码长产生熵；用错误分布编码多出的代价产生 KL；典型序列数量约为 \(2^{nH}\)，因此熵成为压缩率；一个信道能容纳多少彼此可分的典型输出区域，则由互信息和容量控制。编码解释构成许多不等式与极限定理的共同来源，对理解这些结果有实质作用。

\subsection{阅读时反复检查的四件事}

\begin{itemize}
\tightlist
\item
  对数底决定单位：底 2 得 bit，自然对数得 nat。
\item
  熵、互信息和散度都是相对于明确概率模型的量；变量与分布不清，数值就没有完整含义。
\item
  渐近定理需要长块、信源或信道假设以及明确的错误标准，不能直接当作有限文件或短码性能承诺。
\item
  连续变量中的微分熵和确定性表示互信息有额外技术边界，不能照搬离散直觉。
\end{itemize}
