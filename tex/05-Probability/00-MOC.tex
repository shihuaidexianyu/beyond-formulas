\section{概率论：在不确定性中进行可检验的推理}

概率论不负责消除不确定性，而是规定：当结果不能预先确定时，我们怎样清楚地描述可能性，怎样在新信息到来后更新判断，怎样从重复观察中提取稳定规律。

真正困难的通常不是积分或组合数，而是三个更早的问题：

\begin{itemize}
\tightlist
\item
  随机实验和观察机制究竟是什么；
\item
  哪些独立、同分布或均匀假设来自现实，哪些只是为了方便；
\item
  当前结论是精确概率、上界、渐近近似，还是长期保证。
\end{itemize}

这套笔记沿着``模型 → 推断 → 数值摘要 → 多变量结构 → 极限''的认知顺序展开。

\subsection{从一次随机实验开始}

概率空间先把现实过程翻译成样本空间、事件和概率测度。它特别强调``等可能''不是样本点写得对称就自动成立，样本空间也必须保留问题需要的信息。

这是概率论最容易被跳过、却最关键的一步。错误模型上的精确计算不会自动变成正确答案。

\subsection{新信息怎样改变概率}

条件概率、Bayes 与独立性把观察到的信息看成对样本空间的限制和重新规范化。乘法法则沿时间顺序展开事件，全概率公式汇总不同原因，Bayes 公式则从结果反推原因。

同一单元还区分互斥、独立、两两独立、相互独立和条件独立。每当把联合概率写成乘积，都必须能说明这种分解来自什么机制。

\subsection{从复杂结果提取数值}

随机变量与分布把样本点映射为实数。PMF、PDF 和 CDF 不是三个并列术语：CDF 是统一对象，PMF 是离散点上的概率质量，PDF 是连续累计概率的局部变化率。

期望与矩进一步压缩分布。均值给平方损失下最佳常数预测，方差描述剩余波动，协方差描述线性共同变化，条件期望则是在已有信息下的最佳均方预测。

这些摘要有用，但会丢失信息。均值和方差相同不代表分布相同，相关系数为零也不代表独立。

\subsection{多个变量为什么不能只看各自分布}

联合分布保留变量如何共同出现。边缘化是忘掉变量，条件化是固定观察，卷积汇总所有产生同一个和的组合，Jacobian 则补偿多维坐标变换的体积伸缩。

协方差矩阵把波动提升为几何：不同方向的方差由二次型 $a^\mathsf T\Sigma a$ 给出，主轴和特征值连接到 Gaussian 椭圆、PCA 和最小二乘。

\subsection{常见分布要从机制识别}

常见分布没有按名字列公式表，而按生成方式组织：

\begin{itemize}
\tightlist
\item
  固定次数成败试验与等待成功；
\item
  放回和不放回抽样；
\item
  Poisson 计数及其 Exponential/Gamma 等待时间；
\item
  区间和概率单纯形上的 Uniform/Beta/Dirichlet；
\item
  Gaussian 及由它产生的卡方和 Student $t$。
\end{itemize}

读者应能从停止规则、支撑集和参数含义重新推导分布，而不只是认出密度形状。

\subsection{求不出分布时仍可控制风险}

变换与概率界给出两种策略。PGF、MGF 和特征函数把独立和的卷积变成乘法；Union Bound、Markov、Chebyshev、Chernoff 与 Hoeffding 则根据现有信息给出逐渐更强的坏事件上界。

界的强弱反映信息预算：只知道均值，不可能普遍得到与独立有界样本同样锐利的指数尾界。

\subsection{大量重复为何产生稳定}

极限定理先区分几乎处处、依概率、均方和依分布收敛，再分别解释：

\begin{itemize}
\tightlist
\item
  大数定律为什么让样本平均趋向总体均值；
\item
  中心极限定理为什么把剩余误差放在 $1/\sqrt n$ 尺度并近似为 Gaussian；
\item
  Monte Carlo 怎样把这些定理变成带标准误的数值方法。
\end{itemize}

大样本只减少随机误差，不会消除选择偏差、模型错误、分布漂移或共同系统误差。

\subsection{学习前置与后续}

离散样本空间的计数依赖 组合计数；连续密度与变换需要 微积分；随机向量与协方差几何需要 线性代数。

学完后可以进入 随机过程，把单个或有限个随机变量推广到随时间演化的随机系统；也可以进入 数理统计，研究如何由有限数据推断未知分布或参数。
