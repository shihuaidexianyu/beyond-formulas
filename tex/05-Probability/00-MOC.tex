概率论研究如何处理不确定性：当结果不能预先确定时，我们怎样清楚地描述可能性，怎样在新信息到来后更新判断，怎样从重复观察中提取稳定规律。它提供的这些规则不会让不确定性凭空消失。

真正的困难通常出现在积分或组合数之前，集中于三个问题：

\begin{itemize}
\tightlist
\item
  随机实验和观察机制究竟是什么；
\item
  哪些独立、同分布或均匀假设来自现实，哪些只是为了方便；
\item
  当前结论是精确概率、上界、渐近近似，还是长期保证。
\end{itemize}

这套笔记沿着``模型 → 推断 → 数值摘要 → 多变量结构 → 极限''的认知顺序展开。

\subsection{怎么读这一章}

八个单元不必只按目录顺序阅读，可以沿四条任务线进入：

\begin{itemize}
\tightlist
\item
  \textbf{先把随机问题建对：}从\hyperref[ch:037]{``概率空间：先说清楚什么可能发生''}到\hyperref[ch:038]{``条件概率、Bayes 与独立性：新信息怎样改变判断''}。先定义实验、观察与事件，再决定新信息如何限制样本空间；不要从公式反推模型。
\item
  \textbf{把结果压成可计算对象：}读\hyperref[ch:039]{``随机变量与分布：把复杂结果压缩成可研究的数值''}和\hyperref[ch:040]{``期望与矩：怎样用少量数字概括一个分布''}。前者保留完整分布，后者有意只保留任务需要的摘要。
\item
  \textbf{研究多个量怎样共同变化：}读\hyperref[ch:041]{``联合分布：单个分布之外，变量怎样一起变化''}；边缘分布相同并不足以确定依赖结构，协方差也只看见线性关系。
\item
  \textbf{从有限信息得到可用保证：}按需要查阅\hyperref[ch:042]{``常见分布：不要背表，要先辨认随机机制''}，再进入\hyperref[ch:043]{``变换与概率界：不知道完整分布时还能说什么''}和\hyperref[ch:044]{``极限定理：重复随机性为何会产生稳定规律''}。这里要持续区分精确分布、有限样本界与渐近近似。
\end{itemize}

“常见分布”更适合作为机制索引，而不是必须一次背完的公式表。第一次阅读可以在能回答三件事时停下：随机性来自什么机制？当前使用了哪些独立或同分布假设？答案是精确值、上界、近似还是极限陈述？

\subsection{从一次随机实验开始}

\hyperref[ch:037]{``概率空间：先说清楚什么可能发生''}先把现实过程翻译成样本空间、事件和概率测度。它特别强调``等可能''不是样本点写得对称就自动成立，样本空间也必须保留问题需要的信息。

这是概率论最容易被跳过、却最关键的一步。错误模型上的精确计算不会自动变成正确答案。

\subsection{新信息怎样改变概率}

\hyperref[ch:038]{``条件概率、Bayes 与独立性：新信息怎样改变判断''}把观察到的信息看成对样本空间的限制和重新规范化。乘法法则沿时间顺序展开事件，全概率公式汇总不同原因，Bayes 公式则从结果反推原因。

同一单元还区分互斥、独立、两两独立、相互独立和条件独立。每当把联合概率写成乘积，都必须能说明这种分解来自什么机制。

\subsection{从复杂结果提取数值}

\hyperref[ch:039]{``随机变量与分布：把复杂结果压缩成可研究的数值''}把样本点映射为实数。PMF、PDF 和 CDF 不是三个并列术语：CDF 对任意实随机变量都存在，PMF 描述离散点上的概率质量，而 PDF 只在分布具有密度时描述累计概率的局部变化率。

\hyperref[ch:040]{``期望与矩：怎样用少量数字概括一个分布''}进一步压缩分布。均值给平方损失下最佳常数预测，方差描述剩余波动，协方差描述线性共同变化，条件期望则是在已有信息下的最佳均方预测。

这些摘要有用，但会丢失信息。均值和方差相同不代表分布相同，相关系数为零也不代表独立。

\subsection{多个变量为什么不能只看各自分布}

\hyperref[ch:041]{``联合分布：单个分布之外，变量怎样一起变化''}保留变量如何共同出现。边缘化是忘掉变量，条件化是固定观察，卷积汇总所有产生同一个和的组合，Jacobian 则补偿多维坐标变换的体积伸缩。

协方差矩阵把波动提升为几何：不同方向的方差由二次型 \(a^{\trans}\Sigma a\) 给出，主轴和特征值连接到 Gaussian 椭圆、PCA 和最小二乘。

\subsection{常见分布要从机制识别}

\hyperref[ch:042]{``常见分布：不要背表，要先辨认随机机制''}没有按名字列公式表，而按生成方式组织：

\begin{itemize}
\tightlist
\item
  固定次数成败试验与等待成功；
\item
  放回和不放回抽样；
\item
  Poisson 计数及其 Exponential/Gamma 等待时间；
\item
  区间和概率单纯形上的 Uniform/Beta/Dirichlet；
\item
  Gaussian 及由它产生的卡方和 Student \(t\)。
\end{itemize}

读者应能从停止规则、支撑集和参数含义重新推导分布，而不只是认出密度形状。

\subsection{求不出分布时仍可控制风险}

\hyperref[ch:043]{``变换与概率界：不知道完整分布时还能说什么''}给出两种策略。PGF、MGF 和特征函数把独立和的卷积变成乘法；Union Bound、Markov、Chebyshev、Chernoff 与 Hoeffding 则根据现有信息给出逐渐更强的坏事件上界。

界的强弱反映信息预算：只知道均值，不可能普遍得到与独立有界样本同样锐利的指数尾界。

\subsection{大量重复为何产生稳定}

\hyperref[ch:044]{``极限定理：重复随机性为何会产生稳定规律''}先区分几乎处处、依概率、均方和依分布收敛，再分别解释：

\begin{itemize}
\tightlist
\item
  大数定律为什么让样本平均趋向总体均值；
\item
  中心极限定理为什么把剩余误差放在 \(1/\sqrt n\) 尺度并近似为 Gaussian；
\item
  Monte Carlo 怎样把这些定理变成带标准误的数值方法。
\end{itemize}

大样本只减少随机误差，不会消除选择偏差、模型错误、分布漂移或共同系统误差。

\subsection{学习前置与后续}

离散样本空间的计数依赖\hyperref[ch:027]{``组合计数：先描述对象怎样生成，再决定怎样数''}；连续密度与变换需要\hyperref[ch:009]{``微积分：局部变化怎样累积成整体''}；随机向量与协方差几何需要\hyperref[ch:016]{``线性代数：寻找让问题变简单的坐标''}。

学完后可以进入\hyperref[ch:045]{``随机过程：给不确定性加上时间''}，把单个或有限个随机变量推广到随时间演化的随机系统；也可以进入\hyperref[ch:085]{``数理统计：从有限样本推断未知机制''}，研究如何由有限数据推断未知分布或参数。
