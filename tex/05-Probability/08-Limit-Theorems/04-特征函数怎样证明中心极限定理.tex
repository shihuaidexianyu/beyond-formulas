\section{特征函数怎样证明中心极限定理}
\label{sec:05-Probability/08-Limit-Theorems/04}

\begin{quote}
\textbf{先抓住这一点：}特征函数把独立和的卷积变成乘法。中心化后的二阶展开决定 Gaussian 指数，Lévy 连续性定理再把函数极限翻译回分布极限。
\end{quote}

上一节把中心极限定理作为结论陈述：独立同分布、有限非零方差变量的中心化标准和趋向 Gaussian。为什么在这组条件下，不同母分布反复卷积并标准化后会奔向同一个极限？前面\hyperref[sec:05-Probability/05-Joint-Distributions/03]{``随机变量之和与卷积''}给出的直觉是``卷积让形状越来越圆''，但那不是证明。本节补上关键工具：特征函数把独立和的卷积变成乘法，而乘法的极限可以逐点算出来。

\subsection{Lévy 连续性定理：分布收敛看特征函数}

\hyperref[sec:05-Probability/07-Transforms-and-Bounds/01]{``生成函数、矩母函数与特征函数''}已经给出特征函数的两条基本性质：它总存在，且唯一确定分布。证明极限定理还需要第三条：

\begin{quote}
\textbf{Lévy 连续性定理.} 随机变量列 \(X_n\) 依分布收敛到 \(X\)，当且仅当它们的特征函数逐点收敛：\(\varphi_{X_n}(t)\to\varphi_X(t)\) 对每个实数 \(t\) 成立。
\end{quote}

这里只使用不证明。它把``分布形状接近''（依分布收敛，见\hyperref[sec:05-Probability/08-Limit-Theorems/01]{``随机变量的多种收敛''}）翻译成函数值的逐点收敛——后者是普通分析对象，可以用 Taylor 展开处理。证明矩型极限定理的标准程序因此是：写出特征函数，算它的极限，认出极限是某个已知分布的特征函数，再引用连续性定理。

\subsection{把 CLT 的代数全部算出来}

设 \(X_1,\ldots,X_n\) 独立同分布，\(\E X_i=\mu\)，\(\Var(X_i)=\sigma^2<\infty\)。令 \(Y_i=(X_i-\mu)/\sigma\)，则 \(\E Y_i=0\)，\(\E[Y_i^2]=1\)，标准化和为

\[
Z_n=\frac1{\sqrt n}\sum_{i=1}^n Y_i.
\]

第一步，独立和的特征函数是乘积，缩放只改变自变量：

\[
\varphi_{Z_n}(t)
=\prod_{i=1}^n\varphi_{Y_i}\!\left(\frac{t}{\sqrt n}\right)
=\left[\varphi_Y\!\left(\frac{t}{\sqrt n}\right)\right]^n,
\]

其中 \(\varphi_Y\) 是 \(Y_i\) 共同的特征函数。

第二步，对 \(\varphi_Y\) 在原点做 Taylor 展开。因为 \(\E Y=0\)、\(\E[Y^2]=1\)，由 \(\varphi_Y^{(k)}(0)=i^k\E[Y^k]\)（\hyperref[sec:05-Probability/07-Transforms-and-Bounds/01]{``生成函数、矩母函数与特征函数''}中已建立），

\[
\varphi_Y(s)=1-\frac{s^2}{2}+o(s^2),
\qquad s\to0.
\]

代入 \(s=t/\sqrt n\)：

\[
\varphi_Y\!\left(\frac{t}{\sqrt n}\right)
=1-\frac{t^2}{2n}+o\!\left(\frac1n\right).
\]

第三步，取 \(n\) 次幂的极限。由标准极限 \((1+x_n/n)^n\to e^x\)（当 \(nx_n\to x\)），

\[
\varphi_{Z_n}(t)
=\left[1-\frac{t^2/2+o(1)}{n}\right]^n
\longrightarrow e^{-t^2/2}.
\]

第四步，认出极限。标准正态 \(Z\sim\mathcal N(0,1)\) 的特征函数是

\[
\varphi_Z(t)
=\frac1{\sqrt{2\pi}}\int e^{itx}e^{-x^2/2}\,dx
=e^{-t^2/2},
\]

可由配方并围道移动（或利用 MGF 在虚轴上的解析延拓）得到。于是 \(\varphi_{Z_n}(t)\to\varphi_Z(t)\) 逐点成立，Lévy 连续性定理给出

\[
Z_n\xrightarrow{d}\mathcal N(0,1).
\]

四步分工清楚：独立让乘积出现，二阶矩有限让二次展开合法，幂的极限给出指数，连续性定理把函数收敛翻译回分布收敛。Gaussian 之所以是极限，是因为它是唯一一个特征函数形如 \(e^{-t^2/2}\) 的分布——而那个指数函数正是``\((1-t^2/2n)^n\)''的极限。

\subsection{每一步条件各防住什么失败}

\begin{itemize}
\tightlist
\item
  \textbf{独立性}让特征函数相乘。只有不相关而联合分布任意时，第一步就断了——和的分布不由边缘决定。
\item
  \textbf{有限方差}让 Taylor 展开到二阶。Cauchy 分布的均值与方差不存在，其特征函数是 \(e^{-|t|}\)，在 \(t=0\) 不可导；Cauchy 样本平均的特征函数经同样的乘法运算后仍是 \(e^{-|t|}\)，平均还是 Cauchy，正如上一节警告的那样，不会变窄。
\item
  \textbf{同分布}让所有因子相同，乘积是幂。不同分布时仍能成立（Lindeberg 条件），但代数不再是``同一个因子取 \(n\) 次方''，需要逐项控制。
\end{itemize}

证明还解释了 Berry--Esseen 的入口：把展开做到更高阶并估计余项，就能得到有限 \(n\) 的误差界，而不仅是极限陈述；余项的大小由三阶矩控制，这正是上一节引用的结论的来源。

\subsection{同样的机器还证明什么}

特征函数加连续性定理是分布极限的通用机器，不只服务 CLT：

\begin{itemize}
\tightlist
\item
  \textbf{Poisson 极限定理}：Binomial\((n,\lambda/n)\) 的特征函数 \((1-\lambda/n+\lambda e^{it}/n)^n\to\exp(\lambda(e^{it}-1))\)，正是 Poisson\((\lambda)\) 的特征函数——``稀有事件大量试验趋向 Poisson''由此严格化；
\item
  \textbf{大数定律（弱形式）}：若 \(\E\abs{X}<\infty\) 且 \(\mu=\E[X]\)，则 \(\varphi_X(u)=1+i\mu u+o(u)\)。因此 \(\varphi_{\bar X_n}(t)=[\varphi_X(t/n)]^n\to e^{it\mu}\)，即 \(\bar X_n\) 依分布收敛到常数 \(\mu\)，也就依概率收敛到 \(\mu\)。等价地，中心化平均 \(\bar X_n-\mu\) 的特征函数才趋于 \(1=e^{it\cdot0}\)；
\item
  \textbf{稳定律}：无限方差时，幂的极限不再是 \(e^{-t^2/2}\)。在对称稳定律中，典型形式为 \(e^{-c|t|^\alpha}\)（\(\alpha<2\)）；非对称稳定律还带有编码偏斜的相位项。极限分布族也随之更换——对称 Cauchy 正是 \(\alpha=1\) 的成员。
\end{itemize}

同一条``乘法—展开—认形''的程序覆盖这些定理，差异只在展开到几阶、极限指数长什么样。

\subsection{边界清单}

\begin{itemize}
\tightlist
\item
  连续性定理要求对每个固定 \(t\) 逐点收敛到极限特征函数；它不要求导数一致收敛或在整个实轴上一致收敛，但绝不是只检查某一个 \(t\)。
\item
  逐点收敛的极限必须是某个合法分布的特征函数；若极限函数在 \(t=0\) 不连续，质量可能已经``逃到无穷远''（例如 \(X_n\) 均匀分布在 \([-n,n]\) 上时，特征函数趋于 \(t=0\) 处为 1、其余为 0 的函数，它不是特征函数，此时分布不收敛）。
\item
  特征函数证明给的是极限，不给有限 \(n\) 的误差界；误差界需要 Berry--Esseen 一类的余项估计。
\item
  二阶展开要求二阶矩；更高阶展开要求更高阶矩。矩不存在的区域正是稳定律的地盘，不是``证明失效''的意外。
\end{itemize}

至此，概率论的两条主线汇合：变换工具（上一单元）不只是计算技巧，还是极限定理的证明引擎。以后再看到``某分布收敛到某分布''，可以先问一句：它的特征函数长什么样，极限又认得出吗。
