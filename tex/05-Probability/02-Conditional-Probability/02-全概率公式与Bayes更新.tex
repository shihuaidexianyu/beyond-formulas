\section{全概率公式与 Bayes 更新}
\label{sec:05-Probability/02-Conditional-Probability/02}

\begin{quote}
\textbf{先抓住这一点：}Bayes 公式不是把竖线两边交换，而是用``先验乘似然''重新分配解释权重。分母由所有候选原因产生证据的总概率给出。它反转信息方向，不自动反转或证明因果方向。
\end{quote}

很多问题存在一个看不见的原因 \(H\)，我们只能观察它产生的结果 \(E\)。工厂机器影响次品率，疾病影响检测结果，天气影响道路状态。通常从原因到结果的概率比较容易获得：

\[
P(E\mid H).
\]
但推断真正关心的是反方向：

\[
P(H\mid E).
\]
Bayes 公式先计算各种原因产生该结果的总概率，再重新分配解释权重；竖线两边不能直接交换。

\subsection{全概率公式：把所有来路加起来}

若事件 \(H_1,\ldots,H_m\) 两两互斥并覆盖样本空间，即

\[
\bigcup_{i=1}^{m}H_i=\Omega,
\]
则任意事件 \(E\) 可拆成互斥并

\[
E=\bigcup_{i=1}^{m}(E\cap H_i).
\]
因此

\[
P(E)=\sum_{i=1}^{m}P(H_i)P(E\mid H_i).
\]
这就是全概率公式。它表达的是``结果 \(E\) 可以沿哪些互斥路径发生''。

某工厂中机器 A 生产 \(70\%\) 的零件，次品率 \(1\%\)；机器 B 生产 \(30\%\)，次品率 \(4\%\)。随机抽到次品的概率为

\[
P(D)
=0.7\cdot0.01+0.3\cdot0.04
=0.019.
\]
不能把两个次品率简单平均为 \(2.5\%\)，因为两台机器产量权重不同。

\subsection{Bayes 公式：结果怎样重排原因}

由条件概率定义，

\[
P(H_i\mid E)
=\frac{P(H_i\cap E)}{P(E)}
=\frac{P(H_i)P(E\mid H_i)}
{\sum_jP(H_j)P(E\mid H_j)}.
\]
在工厂例子中，已知零件是次品，它来自机器 B 的概率为

\[
P(B\mid D)
=\frac{0.3\cdot0.04}{0.019}
\approx0.632.
\]
机器 B 只贡献三成产量，却贡献了约六成三的次品。原先的产量比例仍然有效，但新证据对 B 更有利：次品在 B 下出现的似然 \(P(D\mid B)=0.04\) 远高于在 A 下的 \(0.01\)，解释权重因此被推向 B。

Bayes 更新可以读成：

\[
\text{后验}\propto\text{先验}\times\text{似然}.
\]
\begin{itemize}
\tightlist
\item
  先验 \(P(H_i)\)：观察证据前对原因的权重；
\item
  似然 \(P(E\mid H_i)\)：若原因成立，证据出现得有多自然；
\item
  后验 \(P(H_i\mid E)\)：看到证据后重新规范化的权重。
\end{itemize}

\subsection{基准率为什么不能忽略}

一种疾病患病率为 \(1\%\)。检测对患者有 \(99\%\) 的阳性率，对健康者有 \(5\%\) 的假阳性率。某人检测阳性，患病概率是多少？

设 \(D\) 为患病，(+) 为阳性，则

\[
P(D\mid+)
=\frac{0.01\cdot0.99}
{0.01\cdot0.99+0.99\cdot0.05}
\approx0.167.
\]
尽管检测对患者非常敏感，阳性后的患病概率仍只有约 \(16.7\%\)。原因是健康人基数巨大，\(5\%\) 的假阳性会产生许多阳性结果。

若公式显得反直觉，可以想象一万人：

\begin{itemize}
\tightlist
\item
  约一百人患病，其中九十九人阳性；
\item
  约九千九百人健康，其中四百九十五人假阳性；
\item
  阳性共五百九十四人，真正患病者只有九十九人。
\end{itemize}

自然频数把抽象比例变成看得见的人数，往往是理解 Bayes 的最好脚手架。

\subsection{连续更新与赔率形式}

对两个假设 \(H_1,H_0\)，把各自的 Bayes 后验相除，公共的分母 \(P(E)\) 约去，Bayes 公式可写成赔率形式：

\[
\frac{P(H_1\mid E)}{P(H_0\mid E)} = \frac{P(H_1)}{P(H_0)}
\cdot
\frac{P(E\mid H_1)}{P(E\mid H_0)}.
\]
第二个比值称为似然比。若连续得到的证据在给定假设下条件独立（见本单元\hyperref[sec:05-Probability/02-Conditional-Probability/04]{``条件独立与信息结构''}），各自的似然比会相乘，因此对数赔率中证据贡献相加。这正是许多分类器、序贯检验与 Bayesian 推断的基本结构。

但``重复检测''只有在给定真实状态后近似条件独立时才能直接相乘。若两次检测共享同一设备偏差，它们并没有提供两份独立证据。

\subsection{Bayes 公式不会修复坏模型}

计算出的后验只在先验、似然和抽样机制可信时有意义。样本选择偏差、检测准确率随人群变化、漏掉其他假设，都会让精确小数显得虚假。

所以 Bayes 更新应同时报告：

\begin{itemize}
\tightlist
\item
  候选原因是否穷尽；
\item
  先验从哪里来；
\item
  似然是否适用于当前人群与条件；
\item
  证据之间是否可以按假定方式组合。
\end{itemize}

下一节会正式讨论独立性。它用联合概率能否分解给出精确条件，比``看起来没关系''更严格。
