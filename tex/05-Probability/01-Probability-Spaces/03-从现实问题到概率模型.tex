\section{从现实问题到概率模型}

掌握公理并不意味着会做概率题。公理只保证一个已经给定的概率模型内部一致；它不会替你决定硬币是否公平、抽样是否独立、时间是否均匀，也不会保证自然语言题目只对应一种模型。

概率计算之前最重要的工作，是把现实假设写清楚。

\subsection{``等可能''必须来自模型}

掷两枚公平硬币时，四个有序结果

\[
HH,HT,TH,TT
\]
等可能，因此事件概率可以用

\[
\frac{\text{有利样本点数}}{\text{全部样本点数}}
\]
计算。

但若把样本空间压缩为

\[
\{0\text{ 个正面},1\text{ 个正面},2\text{ 个正面}\},
\]
三个结果不等可能：概率分别为 $1/4,1/2,1/4$。这正是第一节 $\Omega'=\{0,1,2\}$ 那个压缩样本空间：样本点写得对称，并不自动意味着概率相同。

古典概率的计数公式只在有限且等可能的样本点上成立。若硬币正面概率为 $p$，且两次投掷相互独立（独立性的精确定义在下一章给出），两次结果的概率是

\[
p^2,\quad p(1-p),\quad (1-p)p,\quad(1-p)^2,
\]
此时仍是同一个样本空间，却不是均匀概率。

\subsection{先问随机机制，而不是套名称}

``从十个人中选两人''可能表示：

\begin{itemize}
\tightlist
\item
  同时无序抽取，两人组合等可能；
\item
  依次有序抽取；
\item
  每个人独立以某概率入选，因此人数本身随机；
\item
  按权重抽样，不同人被选中的机会不同。
\end{itemize}

这四种机制会产生不同样本空间和概率。题目里出现``随机''两个字，不足以说明究竟采用哪一种。

建立模型时可以依次写出：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  一次完整结果是什么；
\item
  哪些结果被视为事件；
\item
  概率质量或密度怎样分配；
\item
  分配依赖哪些现实假设；
\item
  所问结论对这些假设有多敏感。
\end{enumerate}

\subsection{计数只负责分子与分母}

从一副牌中不放回抽五张，恰好两张 A 的概率可写成

\[
\frac{\binom42\binom{48}3}{\binom{52}5}.
\]
这个公式背后有三层推理：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  所有五张牌的无序集合等可能；
\item
  有利手牌要从四张 A 中选两张，再从其余四十八张中选三张；
\item
  每一手牌被恰好计数一次。
\end{enumerate}

若抽牌有放回，或每张牌权重不同，组合数公式便不再对应真实概率。组合计数提供计数工具，概率模型则决定这些计数能否变成概率。

\subsection{连续均匀意味着按长度、面积或体积}

若 $X$ 在 $[0,10]$ 上均匀，事件 $2\le X\le5$ 的概率是区间长度之比：

\[
P(2\le X\le5)=\frac{5-2}{10}=\frac3{10}.
\]
平面区域上的均匀随机点则按面积计算。这里的``均匀''必须相对于某个几何量说明。

Bertrand 弦悖论是一个著名警告：``在圆中随机取一条弦''并没有唯一含义。随机选弦的中点、随机选两个端点、随机选方向和距圆心距离，会得到不同分布；对``弦长超过内接等边三角形边长''这一事件，三种机制的经典答案分别是 $1/4$、$1/3$ 和 $1/2$。矛盾不在概率论，而在``随机取弦''的机制没有说完整。

\subsection{模型可以有用而不必完美真实}

公平独立硬币、均匀抽样和 Poisson 到达都只是模型。评价模型不应只问``现实是否严格满足''，还要问：

\begin{itemize}
\tightlist
\item
  它保留了哪些与问题有关的结构？
\item
  忽略的因素是否会显著改变结论？
\item
  能否由数据检验或校准参数？
\item
  在什么范围内使用才可靠？
\end{itemize}

例如把大量顾客到达近似为独立、稳定速率的 Poisson 过程，在高峰期或群体事件中可能失效，但在合适时间尺度上仍能提供有价值的排队估计。

概率论的严谨性有两层：给定模型后的推导必须正确；从现实到模型的假设也必须透明。只做到第一层，会得到``计算无误但回答了另一个问题''的结果。

\subsection{下一步：信息改变概率}

到这里，概率 $P(A)$ 描述的是当前模型下事件 $A$ 的可能性。但现实中的推断总在接收新信息：检测结果为阳性、第一张牌是 A、某个数据包没有到达。新信息会把样本空间限制到与观察相容的部分，然后重新规范化；对原概率做简单加减无法完成这个更新。第一节两个孩子的例子已经演示过一次这种重新计数。
