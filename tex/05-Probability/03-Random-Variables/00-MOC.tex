一次随机实验的完整结果可能非常复杂。抛十次硬币有 $2^{10}$ 条序列，但如果问题只关心正面数，就没有必要逐条追踪顺序。随机变量把每个样本点映射成一个数，让我们可以研究``数值落在哪里''，而不必始终停留在原始样本空间。

本单元按统一概念到具体表示的顺序展开：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  随机变量是样本空间上的函数：理解随机性来自输入结果，而变量本身是一个确定函数。
\item
  离散分布与概率质量函数：用 PMF 与 CDF 描述可数取值。
\item
  连续分布、密度与分位数：理解密度不是概率，CDF 才是离散与连续的统一对象。
\item
  随机变量变换与逆变换抽样：当 $Y=g(X)$ 时，怎样从 $X$ 的分布得到 $Y$ 的分布。
\end{enumerate}

随后``期望与矩''会进一步把整个分布压缩成位置、波动与关联的数值摘要。
