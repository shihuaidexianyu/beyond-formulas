\section{随机游走：从局部步进到整体尺度}
\label{sec:06-Random-Processes/04-Applications/01}

\begin{quote}
\textbf{先抓住这一点：}随机游走展示最典型的尺度涌现：单步偏差累积成线性漂移，独立波动累积成 \(\sqrt n\) 扩散，而边界、维度和微小偏置会从根本上改变击中与返回行为。
\end{quote}

最简单的一维随机游走从 \(S_0=0\) 开始，每步增加独立增量

\[
Z_n=
\begin{cases}
+1,&\text{概率 }p,\\
-1,&\text{概率 }q=1-p.
\end{cases}
\]
于是

\[
S_n=\sum_{k=1}^{n}Z_k.
\]
这个模型看似只是反复抛硬币，却连接扩散、排队、金融价格、算法和偏微分方程。

\subsection{局部偏差如何累积}

单步均值与方差为

\[
\E[Z]=p-q=2p-1,
\]
\[
\Var(Z)=\E[Z^2]-(\E[Z])^2=1-(p-q)^2=4pq,
\]
其中 \(\E[Z^2]=p+q=1\)，因为 \(Z^2\) 恒等于 \(1\)。由期望的可加性与增量的独立性，

\[
\E[S_n]=n(p-q),
\]
\[
\Var(S_n)=4npq.
\]
当 \(p\ne1/2\)，平均位置线性漂移；当 \(p=1/2\)，均值保持零，但标准差为

\[
\sqrt n.
\]
因此无偏不表示位置靠近原点。典型距离随 \(\sqrt n\) 增长，这是扩散尺度。

由中心极限定理，

\[
\frac{S_n-n(p-q)}{2\sqrt{npq}}
\xrightarrow{d}\mathcal N(0,1).
\]
这里要求 \(0<p<1\)。当 \(p=0\) 或 \(p=1\) 时单步方差为零，路径是确定性的，不能再用上述标准化形式。
离散步进在大尺度下呈现 Gaussian 轮廓。

\subsection{位置分布来自 Binomial}

若前 \(n\) 步中有 \(K\) 次向右，则

\[
S_n=K-(n-K)=2K-n.
\]
所以 \(S_n=s\) 只可能在 \(s\) 与 \(n\) 同奇偶时发生，且

\[
P(S_n=s)
=\binom n{(n+s)/2}
p^{(n+s)/2}
q^{(n-s)/2}.
\]
奇偶限制正是上一单元所说的周期 \(2\) 在路径上的体现：从零出发，偶数步只能在偶数位置，奇数步只能在奇数位置。

\subsection{边界把游走变成击中问题}

若资金在 \(0\) 与 \(N\) 之间游走，到边界停止，就得到赌徒破产。公平时从 \(i\) 出发先到 \(N\) 的概率为

\[
\frac iN,
\]
平均吸收时间为

\[
i(N-i).
\]
这两个结果在上一单元已用首步差分方程求出，下一节还会用 Martingale 停时重新得到击中概率。这说明即使游戏公平，有限资金玩家最终仍会以概率一触碰某个边界；``每局期望收益零''不等于``永远安全''。

有漂移时击中概率不再线性。同一首步方程 \(h_i=ph_{i+1}+qh_{i-1}\) 在 \(p\ne q\) 时的通解为 \(A+B(q/p)^i\)，代入边界条件 \(h_0=0\)、\(h_N=1\) 得

\[
h_i=\frac{1-(q/p)^i}{1-(q/p)^N}.
\]
很小的单步偏差经过长时间会显著改变最终命运。

\subsection{维度改变返回概率}

在整数格 \(\Z^d\) 上，每步均匀走向一个相邻点：

\begin{itemize}
\tightlist
\item
  一维和二维简单对称随机游走常返；
\item
  三维及以上暂留。
\end{itemize}

高维提供越来越多远离原点的方向，返回概率总和从发散变为收敛——这正是上一单元用期望返回次数判别常返性的准则。二维虽会返回，但平均返回时间仍无穷。

\subsection{从随机游走到 Brownian motion}

把步长缩为 \(\sqrt{\Delta t}\)，时间步缩为 \(\Delta t\)，并在单位时间内走 \(1/\Delta t\) 步。无偏游走每步方差为 \(\Delta t\)，单位时间的累计方差恒为 \((\sqrt{\Delta t})^2\cdot(1/\Delta t)=1\)，不随缩放消失；适当插值后在极限中得到 Brownian motion。这一段是尺度直觉的描述，严格的极限结论由 Donsker 不变原理给出。

这个缩放解释为什么扩散方程、Gaussian 热核和 Brownian 路径彼此相连。极限路径连续，却几乎处处不可微：许多微小独立扰动不会变成光滑轨迹。

\subsection{模型边界}

真实运动可能有：

\begin{itemize}
\tightlist
\item
  步长相关；
\item
  方向持续性；
\item
  状态相关漂移；
\item
  重尾跳跃；
\item
  随机环境；
\item
  边界反射或吸收。
\end{itemize}

简单随机游走是基准，不是所有轨迹的默认模型。它的价值在于清楚展示局部规则如何决定整体尺度和击中性质。
