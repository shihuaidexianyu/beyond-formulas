\section{首步分析、击中时间与吸收链}

Markov 性最实用的推理方式之一，是先看第一步去了哪里，再把余下问题交给同一个未知函数。这叫首步分析。

它适合计算：

\begin{itemize}
\tightlist
\item
  最终能否到达目标；
\item
  到达前平均需要多少步；
\item
  吸收前访问某状态多少次；
\item
  最终被哪个吸收状态捕获。
\end{itemize}

\subsection{击中概率的调和方程}

对目标集合 $A$，定义击中时间

\[
T_A=\inf\{n\ge0:X_n\in A\},
\]
以及

\[
h_i=P_i(T_A<\infty).
\]
若 $i\in A$，则 $h_i=1$。若 $i\notin A$，按第一步条件化：

\[
h_i=\sum_jP_{ij}h_j.
\]
未知击中概率在非目标状态上等于邻居值的概率加权平均，因此是离散调和函数。边界条件决定具体解。

若还有失败吸收集合 $B$，求``先到 $A$ 再到 $B$''时设置

\[
h_i=1\quad(i\in A),
\qquad
h_i=0\quad(i\in B),
\]
内部仍满足 $h=Ph$。

\subsection{击中时间多出一步成本}

令

\[
m_i=\E_i[T_A].
\]
在目标内 $m_i=0$。目标外先走一步，再从新状态继续，所以

\[
m_i=1+\sum_jP_{ij}m_j.
\]
常数 $1$ 是本次转移消耗的时间。若每一步在状态 $i$ 有成本 $c_i$，则推广为

\[
v_i=c_i+\sum_jP_{ij}v_j.
\]
这些方程只有在相应击中与期望有限、边界条件充分时才给出所需唯一有限解。无限链中可能存在其他非负解或期望无穷。

\subsection{赌徒破产}

资金状态为 $0,1,\ldots,N$。每局以概率 $p$ 赢一单位，以 $q=1-p$ 输一单位；到 $0$ 或 $N$ 停止。

令 $h_i$ 为从 $i$ 出发先到 $N$ 的概率，则

\[
h_0=0,\qquad h_N=1,
\]
\[
h_i=ph_{i+1}+qh_{i-1}.
\]
公平情形 $p=q=1/2$ 的解是线性函数：

\[
h_i=\frac iN.
\]
线性函数确实满足递推：$h_i$ 恰是 $h_{i-1}$ 与 $h_{i+1}$ 的平均；边界条件 $h_0=0$、$h_N=1$ 再定出斜率 $1/N$。
起始资金占目标资金的比例，就是先达到目标的概率。

期望吸收时间满足

\[
m_0=m_N=0,
\]
\[
m_i=1+\frac12m_{i+1}+\frac12m_{i-1},
\]
解为

\[
m_i=i(N-i).
\]
代入验证：$1+\frac12(i+1)(N-i-1)+\frac12(i-1)(N-i+1)=i(N-i)$，且 $i=0$ 与 $i=N$ 处确实为零。先达到目标的概率随初始资金线性变化，符合直观；等待时间却随区间尺度二次增长。

\subsection{吸收链的矩阵结构}

把有限吸收链状态重新排序，使暂留状态在前、吸收状态在后：

\[
P=
\begin{bmatrix}
Q&R\\
0&I
\end{bmatrix}.
\]
$Q$ 描述暂留状态间转移，$R$ 描述从暂留到吸收。

若最终以概率一吸收，基本矩阵为

\[
N=(I-Q)^{-1}
=I+Q+Q^2+\cdots.
\]
元素 $N_{ij}$ 是从暂留状态 $i$ 出发、吸收前访问暂留状态 $j$ 的期望次数；级数中的 $I$ 项表示出发时刻若已在 $j$ 也计作一次访问。

吸收前期望步数向量为

\[
t=N\mathbf1,
\]
其中 $\mathbf1$ 是全一向量，右乘它即对 $N$ 的各行求和。
进入各吸收状态的概率矩阵为

\[
B=NR.
\]
这些公式把首步递推的线性方程一次性写成矩阵解。

\subsection{首步分析比背矩阵公式更一般}

$(I-Q)^{-1}$ 只适合有限吸收链。首步分析还能处理：

\begin{itemize}
\tightlist
\item
  无限状态随机游走；
\item
  状态相关成本；
\item
  到集合前的累积报酬；
\item
  连续时间链的生成矩阵方程；
\item
  动态规划与 Bellman 方程。
\end{itemize}

共同结构都是``当前价值 = 立即贡献 + 下一状态价值的条件期望''。

\subsubsection{自己补完}

一只棋子在状态 $(0,1,2,3)$ 上移动，$(0,3)$ 吸收。在 $(1,2)$ 时，每步以相同概率向左或向右。

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  用首步分析求从 $1$ 出发先到 $3$ 的概率；
\item
  求从 $1$ 出发到吸收的期望步数；
\item
  写出暂留子矩阵 $Q$；
\item
  计算基本矩阵 $N=(I-Q)^{-1}$，并核对期望步数。
\end{enumerate}
