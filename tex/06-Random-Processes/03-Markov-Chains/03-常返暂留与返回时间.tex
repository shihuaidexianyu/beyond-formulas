\section{常返、暂留与返回时间}
\label{sec:06-Random-Processes/03-Markov-Chains/03}

\begin{quote}
\textbf{先抓住这一点：}可达只意味着存在一条正概率路径，常返却要求最终回来这件事以概率一发生。正常返还进一步要求平均返回时间有限，因此它才对应可归一化的长期占用概率。
\end{quote}

从状态 \(i\) 出发，定义首次返回时间

\[
T_i^+=\inf\{n\ge1:X_n=i\}.
\]
若

\[
P_i(T_i^+<\infty)=1,
\]
称 \(i\) 为常返状态；若这个概率小于一，称为暂留状态。其中下标 \(i\) 是以初始状态为条件的简写：\(P_i(\cdot)=P(\cdot\mid X_0=i)\)；后文的 \(\E_i\) 同理。

可达性只要求某一步存在返回机会，常返性则要求最终返回几乎必然发生。

\subsection{一次返回与无穷多次返回}

对 Markov 链，常返状态一旦返回，过程从该状态重新开始，``再次返回''的概率仍是 \(1\)；归纳可得返回至少 \(k\) 次的概率对任意 \(k\) 都是 \(1\)，取极限即以概率一返回无穷多次。这里用到强 Markov 性：在首次返回时刻 \(T_i^+\) 处，过程如同从 \(i\) 重新出发。

暂留状态则有正概率一去不返，其总访问次数期望有限。

一个判据是

\[
\sum_{n=0}^{\infty}(P^n)_{ii}.
\]
若该和发散，状态 \(i\) 常返；若收敛，状态暂留。每项是第 \(n\) 步处于 \(i\) 的概率，求和等于从 \(i\) 出发访问 \(i\) 的期望总次数。为何期望次数有限等价于暂留？设 \(\rho=P_i(T_i^+<\infty)\)；每次回到 \(i\) 后过程重新开始，``再多回来一次''的概率恒为 \(\rho\)，故期望访问次数为 \(1+\rho+\rho^2+\cdots\)，有限当且仅当 \(\rho<1\)。

\subsection{通信类共享常返类型}

若 \(i\leftrightarrow j\)，则 \((i,j)\) 要么都常返，要么都暂留。因此常返性是通信类性质。证明梗概：取 \(m,n\) 使 \((P^m)_{ij}>0\)、\((P^n)_{ji}>0\)，则 \((P^{m+k+n})_{jj}\ge(P^n)_{ji}(P^k)_{ii}(P^m)_{ij}\)，于是 \(\sum_k(P^k)_{jj}\) 被 \(\sum_k(P^k)_{ii}\) 控制；两个方向各用一次即得两级数同敛散。

有限 Markov 链中：

\begin{itemize}
\tightlist
\item
  每个闭通信类都是常返的；
\item
  不属于任何闭类的状态是暂留的；
\item
  有限不可约链中所有状态都正常返（定义见下一小节）。
\end{itemize}

``有限''非常关键。无限状态不可约链可以全部暂留。

\subsection{正常返与零常返}

常返还分为：

\begin{itemize}
\item
  正常返：

  \[
  \E_i[T_i^+]<\infty;
  \]
\item
  零常返：

  \[
  \E_i[T_i^+]=\infty.
  \]
\end{itemize}

两者都会以概率一返回，但零常返的平均返回时间无穷。

有限不可约链中状态都正常返。无限链中，简单对称随机游走在一维整数线上常返，却是零常返；它会回来，但返回时间分布的尾部很重，平均返回时间无穷。

\subsection{随机游走揭示维度效应}

在 \(\Z^d\) 上每步均匀走向一个相邻格点：

\begin{itemize}
\tightlist
\item
  \(d=1,2\)：原点常返；
\item
  \(d\ge3\)：原点暂留。
\end{itemize}

高维空间提供了更多逃离方向，使总返回概率低于一。这个结论提醒我们，``每步无偏''不等于``总会回来''。

证明需要分析返回概率或 Fourier 方法，这里保留结论与直觉；后面\hyperref[sec:06-Random-Processes/04-Applications/01]{``随机游走：从局部步进到整体尺度''}一章会进一步讨论击中概率与尺度。

\subsection{平稳概率与平均返回时间}

在不可约正常返链中存在唯一平稳分布 \(\pi\)，并有 Kac 公式：

\[
\E_i[T_i^+]=\frac1{\pi_i}.
\]
长期占用概率越高，平均返回越快。例如 \(\pi_i=0.2\) 意味着平均每五步左右回到 \(i\) 一次。

这个数字表示长期平均间隔，不保证每五步固定返回。

反过来，无限不可约链若只有零常返状态，就不存在可规范化为总和一的平稳概率分布；可能存在总质量无限的不变测度。

\subsection{返回、击中与停留要区分}

\begin{itemize}
\tightlist
\item
  返回时间要求从 \(i\) 出发、至少走一步后再到 \(i\)；
\item
  击中时间可从别的状态到目标集合；
\item
  访问次数计算到过多少次；
\item
  连续停留时间在连续时间 Markov 链（CTMC，本部分后续单元的主题）中还包含每次状态驻留时长。
\end{itemize}

定义若不区分 \(n\ge0\) 与 \(n\ge1\)，从目标状态出发的击中时间会平凡等于零，无法研究首次返回。
