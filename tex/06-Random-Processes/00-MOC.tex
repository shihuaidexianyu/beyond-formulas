\section{随机过程：给不确定性加上时间}

概率论研究随机结果，随机过程研究随机结果怎样随时间共同演化。真正新增的困难不是多了一个下标，而是时间依赖：

\begin{itemize}
\tightlist
\item
  每个时刻边缘分布相同，路径仍可能完全不同；
\item
  当前状态若没有保留足够信息，未来就不具 Markov 性；
\item
  平稳分布存在，不代表从任意初值收敛；
\item
  平均到达率低于平均服务率，只是排队稳定的最低门槛。
\end{itemize}

这套笔记按``描述轨迹 → 具体到达模型 → 状态转移 → 连续时间 → 典型应用''的顺序展开。

\subsection{先学会描述一整条随机轨迹}

随机过程基础区分固定时间的随机变量与固定实验的样本路径，说明有限维分布如何决定过程律，并用均值、协方差和平稳性压缩时间结构。

其中一个关键区分是：平稳描述 ensemble 分布不随时间改变，遍历则允许用一条长路径的时间平均估计 ensemble 平均。现实中通常只有一条时间序列，因此两者不能混为一谈。

\subsection{从事件到达建立第一类过程}

计数过程从离散 Bernoulli 时隙进入连续 Poisson 到达。固定窗口的计数、相邻 Exponential 间隔和第 $r$ 个 Gamma 到达时刻是同一个过程的不同观察方式。

进一步放弃无记忆性便得到更新过程。它解释了长期更新率、更新报酬和检查悖论，也提醒读者：平均间隔的一半通常不是随机到达者的平均等待时间。

\subsection{当状态压缩了全部相关历史}

离散时间 Markov 链从状态设计开始。转移矩阵通过 Chapman--Kolmogorov 生成多步行为，但长期结论还取决于通信类、周期性、常返与正常返。

这里必须同时区分三件事：

\begin{itemize}
\tightlist
\item
  $\pi P=\pi$ 的平稳分布；
\item
  $\mu P^n\to\pi$ 的时刻分布收敛；
\item
  一条路径访问频率趋向 $\pi$ 的遍历定理。
\end{itemize}

首步分析则把击中概率和等待时间变成线性方程，是 Markov 性最通用的计算方法之一。

\subsection{连续时间中，速率代替一步概率}

连续时间 Markov 链与排队用生成矩阵 $Q$ 描述瞬时跳转速率。局部展开

\[
P(h)=I+hQ+o(h)
\]
通过 Kolmogorov 方程生成

\[
P(t)=e^{tQ}.
\]
生灭链把局部流量平衡转成可递推的平稳概率；M/M/1 队列则展示负载

\[
\rho=\lambda/\mu
\]
为何必须小于一，以及 $\rho$ 接近一时平均时延为何非线性爆炸。

\subsection{四个模型看见四种用途}

随机过程的四种典型用法把理论带入不同场景：

\begin{itemize}
\tightlist
\item
  随机游走展示独立小步怎样形成漂移与扩散；
\item
  Martingale 把``当前信息下没有可预测漂移''变成停时工具；
\item
  PageRank 通过 teleportation 修复网页图的闭类与周期，用平稳概率定义重要性；
\item
  HMM 在状态不可见时，用动态规划从观测序列反推隐藏轨迹。
\end{itemize}

目录编号把应用放在连续时间之前，但首次阅读更自然的路线是：基础 → 计数 → 离散 Markov 链 → 连续时间链与排队 → 应用。若只学习 HMM 或 PageRank，可以在离散 Markov 链后直接进入相应应用章。
