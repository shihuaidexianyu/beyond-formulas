单变量中，输入只能从左右接近，导数是一个斜率。多元空间里，同一点可以沿无穷多条路径接近，输入变化是向量，最好的局部线性模型也不再只是一个数。偏导数只观察坐标轴方向，真正的可微性则要求一个线性映射同时解释所有小方向。

\item
  多元函数、极限与连续
\item
  偏导数为何不等于可微
\item
  梯度与方向导数
\item
  Jacobian 与多元链式法则
\item
  Hessian 与无约束极值
\item
  约束几何与 Lagrange 乘子
\item
  多重积分与变量替换
\item
  向量场、曲线积分与势函数
\item

\end{enumerate}

前六节把``局部变化''升级为线性代数；后三节把``微小贡献求和''升级到区域、曲线和曲面。最终 Green、Gauss、Stokes 定理会显出同一结构：区域内部的局部导数累积，等于边界上可观测的总效应。

\subsection{怎么读这一章}

第一次读，可以先顺着前六篇回答一个问题：多元函数在一点附近究竟怎样变化？读到梯度、Jacobian 和 Hessian 的作用就够了，同时记住偏导存在不等于可微、梯度为零也不等于已经找到极值。后三篇可以另走一条线：先看换坐标时体积为什么要修正，再看沿路径和沿边界累积时哪些量会抵消；复杂的曲面计算留到需要时再回来查。
