单变量中，输入只能从左右接近，导数是一个斜率。多元空间里，同一点可以沿无穷多条路径接近，输入变化是向量，最好的局部线性模型也不再只是一个数。偏导数只观察坐标轴方向，真正的可微性则要求一个线性映射同时解释所有小方向。

本单元沿这条扩展主线组织：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  多元函数、极限与连续
\item
  偏导数为何不等于可微
\item
  梯度与方向导数
\item
  Jacobian 与多元链式法则
\item
  Hessian 与无约束极值
\item
  约束几何与 Lagrange 乘子
\item
  多重积分与变量替换
\item
  向量场、曲线积分与势函数
\item
  通量、散度、旋度与边界定理
\end{enumerate}

前六节把``局部变化''升级为线性代数；后三节把``微小贡献求和''升级到区域、曲线和曲面。最终 Green、Gauss、Stokes 定理会显出同一结构：区域内部的局部导数累积，等于边界上可观测的总效应。
