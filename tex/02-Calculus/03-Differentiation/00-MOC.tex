平均变化率比较两个不同点；瞬时变化率却要把两点合并，而差商在间隔为零时没有定义。导数用极限保留差商的趋势，并揭示一个更深的事实：可微函数在足够小的尺度上，最好的近似是一条直线。

本单元按概念而非公式数量展开：

\begin{enumerate}
\tightlist
\item
  从平均变化率到导数
\item
  求导规则从哪里来
\item
  复合、隐式与反函数求导
\item
  中值定理连接局部与整体
\item
  线性近似与极值问题
\end{enumerate}

求导技巧只是压缩计算。真正要获得的能力是：看懂导数的单位与符号，判断函数是否真的可微，用中值定理把局部斜率转成整体结论，并在优化问题中区分候选点与最终答案。

\subsection{怎么读这一章}

先从割线斜率的极限得到导数，再从定义推导规则。复合、隐式和反函数求导都只是链式法则的不同组织方式。最后两篇用中值定理把局部导数连接到整体变化，并把导数用于近似和极值。
