\section{线性近似与极值问题}

\subsection{可微真正意味着误差比步长更小}

在 \(a\) 附近，函数可写成

\[
f(a+h)=f(a)+f'(a)h+r(h),
\]

其中

\[
\frac{r(h)}h\to0.
\]

所以线性化

\[
L(x)=f(a)+f'(a)(x-a)
\]

不只是切线方程；它是局部一阶主导模型，剩余误差相对于 \(|x-a|\) 更快变小。

例如在 \(a=4\) 附近，\(f(x)=\sqrt x\) 有 \(f(4)=2\)、\(f'(4)=1/4\)，所以

\[
\sqrt{4.1}\approx2+\frac14(0.1)=2.025.
\]

实际值约为 2.0249，与上一节用中值定理得到的严格上界 \(|\sqrt{4.1}-2|\le0.025\) 相互印证：近似误差远小于这个保证界。线性近似适用于小改变量；离展开点越远，二阶曲率造成的误差越明显。

微分记号 \(dy=f'(x)\,dx\) 可以理解为输入小变化 \(dx\) 经局部线性映射产生的预测输出变化。它不是把两个无限小实数随意相除，而是导数线性化的紧凑表达。

\subsection{极值候选来自哪里}

若 \(f\) 在内部点 \(c\) 取得局部极值且可导，则 \(f'(c)=0\)。因此内部临界点包括 \(f'(x)=0\) 或导数不存在的点。但这是必要候选，不是充分结论。

在闭区间 \([a,b]\) 求全局极值的可靠流程是：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  找内部临界点；
\item
  加上两个端点；
\item
  比较所有候选函数值。
\end{enumerate}

只解 \(f'=0\) 会漏掉端点，也会误把 \(x^3\) 的水平拐点当极值。

一阶导数符号变化能分类：由正变负是局部极大，由负变正是局部极小；不变号则通常不是极值。

\subsection{二阶导数描述斜率怎样变化}

\(f''>0\) 表示斜率递增，图像向上凸（常称凸或凹向上）；\(f''<0\) 表示斜率递减。若 \(f'(c)=0\) 且 \(f''(c)>0\)，\(c\) 为严格局部极小；若 \(f''(c)<0\)，为严格局部极大。

\(f''(c)=0\) 时判别没有结论。\(x^4\) 在 0 有局部最小，\(x^3\) 在 0 不是极值，两者二阶导数都为零。此时应回到导数变号、更高阶项或定义。

拐点要求凹凸性发生改变，不是简单满足 \(f''=0\)。\(f(x)=x^4\) 在 0 二阶导为零，却两侧都向上凸。

\subsection{一个建模完整的优化例子}

用 100 米围栏围一个靠墙矩形，靠墙一侧不需围。设垂直墙的边长为 \(x\)，平行墙的另一边为 \(y\)，约束

\[
2x+y=100,\qquad x,y\ge0.
\]

面积

\[
A=xy=x(100-2x),\qquad 0\le x\le50.
\]

求导

\[
A'(x)=100-4x,
\]

内部临界点 \(x=25\)，对应 \(y=50\)。端点面积都为零，或由 \(A''=-4<0\) 知内部点为最大，因此最大面积为 \(1250\) 平方米。

真正困难常在求导之前：选择变量、写约束、消去变量并确定可行区间。若忽略 \(x\in[0,50]\)，代数模型会允许负边长。

\subsection{从线性近似到 Taylor 展开}

若二阶、三阶导数也存在，可以继续匹配更多局部变化信息：

\[
f(a+h)
\approx f(a)+f'(a)h+\frac{f''(a)}{2}h^2+\cdots.
\]

线性化是 Taylor 展开的第一阶。后续级数单元会讨论何时这不只是形式近似、余项怎样控制，以及函数是否真的等于其 Taylor 级数。
