\section{微积分：局部变化怎样累积成整体}

微积分围绕一对互补问题展开：

\begin{itemize}
\tightlist
\item
  一个量在某一瞬间、某一个点附近怎样变化？
\item
  无数微小变化在一段时间或一个区域内累积成什么？
\end{itemize}

导数回答第一问，积分回答第二问，微积分基本定理说明它们互为局部与整体的逆过程。极限让``无限靠近''和``无限细分''变得严格；级数研究无限过程何时真的产生有限结果；多元微积分再把方向、空间几何和线性代数加入这幅图景。

\subsection{从依赖关系开始，而不是从求导公式开始}

函数与预备知识先建立实数距离、定义域、图像、复合和反函数。指数与对数被解释为恒定相对变化的模型，三角函数被解释为旋转与周期的坐标。后面的每条导数公式都依赖这些函数究竟在哪些输入上有意义。

\subsection{极限解决``不能直接到达''的问题}

极限与连续从穿孔函数、左右趋势和无穷行为出发。严格定义的核心不是符号本身，而是输出误差能否由一个统一的输入窗口控制。连续性随后把点值与附近趋势接上，介值定理和极值定理展示局部无跳跃怎样产生整体保证。

第一遍接触 \(\varepsilon\)--\(\delta\) 时，可以先读懂线性与平方函数的完整例子，再回来处理量词。允许理解逐步完整，比在符号前停住更有价值。

\subsection{导数是局部最佳线性模型}

导数从平均速度的差商极限得到瞬时速度和切线斜率。乘积、商和链式法则都从局部变化怎样组合推导，而不是孤立口诀。中值定理把一个点的导数信息连接到整个区间，线性化和二阶导数则把近似、曲线形状和优化统一起来。

学完后不应只会``求 \(f'\)''，还应能解释导数单位、判断尖点或竖直切线为何不可导，并知道临界点只是极值候选。

\subsection{积分把局部贡献累加}

积分从 Riemann 小块求和开始。速度乘小时间、密度乘小长度、函数高度乘小宽度都属于``局部量 × 小尺度''。微积分基本定理说明累积函数的导数回到原来的率，也让定积分可通过原函数端点差计算。

换元与分部积分分别是链式法则、乘积法则的逆向识别。面积、体积、概率和质量不是额外公式列表，而是选择合适的小块贡献。反常积分则提醒我们：出现无穷区间或奇点时，必须回到极限，不能把无穷当普通端点。

\subsection{无限项需要研究尾部，而不是相信趋势图}

数列与级数区分``项趋于零''和``部分和收敛''。比较、积分、比值和交错判别都在测量尾部衰减尺度。幂级数把每个输入对应到一个级数，Taylor 展开用导数匹配局部信息；余项决定近似是否可信，也划清``无穷可微''与``等于自身 Taylor 级数''的边界。

\subsection{多元空间中，导数升级为线性映射}

多元微积分先用路径反例说明多元极限必须统一控制所有方向，再区分偏导数与真正可微性。梯度是标量函数的局部变化向量，Jacobian 是向量映射的全导数矩阵，Hessian 是二阶曲率的二次型；Lagrange 乘子来自约束切空间的法向几何。

多重积分的 Jacobian 行列式记录坐标变换对面积体积的伸缩。最后，线积分、通量、散度与旋度通过 Green、Gauss、Stokes 定理汇合为同一个原则：内部的局部导数累积，等于边界上的总效应。

\subsection{完成这条主线后}

读者应能够：

\begin{itemize}
\tightlist
\item
  用极限而不是含混的``无限接近''判断趋势；
\item
  把导数解释为局部线性化，把积分解释为累积极限；
\item
  从问题结构推导公式的适用条件和单位；
\item
  区分必要条件、候选解与充分结论；
\item
  为级数与反常积分选择与尾部尺度匹配的判别；
\item
  在多元问题中用向量和矩阵组织方向变化；
\item
  看到微积分基本定理与 Green/Gauss/Stokes 之间的共同边界结构。
\end{itemize}
