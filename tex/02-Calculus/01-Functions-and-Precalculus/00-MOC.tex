微积分研究变化与累积。在谈``变化率''之前，必须先知道哪个量依赖哪个量、允许哪些输入、输出怎样随输入移动。函数是把输入唯一送到输出的完整规则，公式只负责表达这条规则；图像、复合与反函数则提供不同的观察方式。

本单元按下面顺序建立共同语言：

\begin{enumerate}
\tightlist
\item
  从实数轴到函数
\item
  图像变换与函数复合
\item
  反函数与可逆性
\item
  指数与对数
\item
  三角函数与周期
\end{enumerate}

后两节介绍两类基本变化模型：指数函数描述相同相对比例的重复变化，三角函数描述旋转与周期。理解这些模型比背公式表更重要，因为它们会在导数、积分、级数和微分方程中不断出现。

\subsection{怎么读这一章}

先把函数看成带定义域、陪域的对应关系，再学习图像变换与复合怎样构造新函数。反函数要求可逆性，指数、对数和三角函数则提供后续微积分最常用的变化模式。每次先看定义域和尺度，再看公式。
