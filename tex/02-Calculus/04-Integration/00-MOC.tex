导数问``此刻变化多快''，积分问``这些变化在一段时间内累积成多少''。面积只是最直观的入口；路程、质量、概率、功和平均值都共享同一种结构：把区间切细，用局部量乘小宽度，再让分割极限消除近似依赖。

本单元依次阅读：

\begin{enumerate}
\tightlist
\item
  从小块求和到定积分
\item
  微积分基本定理
\item
  换元与分部积分
\item
  部分分式与反常积分
\item
  面积、体积、概率与累积量
\end{enumerate}

重点是看见``局部率 × 小尺度''的建模方式，以及微积分基本定理为什么让累积与变化互相撤销。积分技巧随后只是反向识别链式法则、乘积法则和代数结构。

\subsection{怎么读这一章}

先把定积分理解成小块贡献的极限总和，再看微积分基本定理怎样把累积量与变化率连接起来。换元和分部积分分别对应链式法则与乘积法则的逆用法，反常积分则必须先写成极限再判断收敛。
