\section{微积分基本定理：变化与累积互相撤销}
\label{sec:02-Calculus/04-Integration/02}

\begin{quote}
\textbf{先抓住这一点：}把函数从固定起点累积到 \(x\)，累积量的瞬时变化率会还原被积函数。反过来，导数在区间上的总累积只留下原函数端点之差。
\end{quote}

\subsection{累积到当前位置会产生一个新函数}

固定起点 \(a\)，定义

\[
F(x)=\int_a^x f(t)\,dt.
\]

\(F(x)\) 表示从 \(a\) 到 \(x\) 已累积的总量。把终点从 \(x\) 增加到 \(x+h\)，新增量为

\[
F(x+h)-F(x)=\int_x^{x+h}f(t)\,dt.
\]

差商

\[
\frac{F(x+h)-F(x)}h
=\frac1h\int_x^{x+h}f(t)\,dt
\]

是小区间上 \(f\) 的平均值。连续性保证这个平均值回到 \(f(x)\)：任给 \(\varepsilon>0\)，当 \(h\) 足够小时，小区间上每点都满足 \(|f(t)-f(x)|<\varepsilon\)，于是（以 \(h>0\) 为例）

\[
\left|\frac1h\int_x^{x+h}f(t)\,dt-f(x)\right|
=\left|\frac1h\int_x^{x+h}[f(t)-f(x)]\,dt\right|
\le\frac1h\int_x^{x+h}|f(t)-f(x)|\,dt
<\varepsilon.
\]

故 \(h\to0\) 时平均值的极限就是 \(f(x)\)，即

\[
F'(x)=f(x).
\]

这就是微积分基本定理第一部分：连续率的累积函数，其瞬时变化率回到原来的率。

\subsection{原函数让定积分不必重新做极限和}

若 \(F'=f\)，则

\[
\int_a^b f(x)\,dx=F(b)-F(a).
\]

这是基本定理第二部分。它由第一部分推出：令 \(G(x)=\int_a^x f(t)\,dt\)，第一部分给出 \(G'=f=F'\)，于是 \(G-F\) 在区间上导数恒为零，必为常数（这一步正是中值定理的推论：区间上导数恒为零的函数必为常数）。代入端点确定常数：\(x=a\) 时 \(G(a)-F(a)=-F(a)\)，故 \(G(b)=F(b)-F(a)\)。它把``无限细求和''转化为找原函数并比较端点。

例如 \(F(x)=x^3/3\) 的导数为 \(x^2\)，所以

\[
\int_1^2x^2\,dx
=\left[\frac{x^3}3\right]_1^2
=\frac73.
\]

方括号表示端点差，不是把上下限分别随意代进被积函数。

\subsection{不定积分与定积分不是同一种对象}

不定积分

\[
\int f(x)\,dx=F(x)+C
\]

表示所有导数为 \(f\) 的函数族。常数 \(C\) 必须保留，因为常数导数为零，且在区间上任意两个原函数只差常数。

定积分 \(\int_a^b f(x)\,dx\) 通常是一个数；变上限积分 \(\int_a^x f(t)\,dt\) 是一个函数。符号相似，但对象类型不同。定积分计算中常数在端点相减时抵消，所以不写 \(+C\)。

积分变量是哑变量：

\[
\int_a^x f(t)\,dt=\int_a^x f(u)\,du.
\]

用 \(t\) 而不是 \(x\) 作内部变量，能避免把积分上限与被积变量混淆。

\subsection{净变化定理}

若 \(Q'(t)=r(t)\)，则

\[
Q(b)-Q(a)=\int_a^b r(t)\,dt.
\]

这句话比``积分算面积''更普适：存量变化等于流率的累积。水箱水量变化等于净流量积分，位置变化等于速度积分，速度变化等于加速度积分，概率分布函数变化等于密度积分（``密度''可暂理解为``每单位取值的概率''，严格讨论见后文概率一节）。

若已知初值 \(Q(a)=Q_0\)，便有

\[
Q(t)=Q_0+\int_a^t r(s)\,ds.
\]

这是许多微分方程积分形式的原型。

\subsection{条件与边界}

基本定理的简洁版本假设 \(f\) 连续。更一般条件下仍有更强版本，但不能从任何随意函数的积分直接断言处处可导。若 \(f\) 在某点跳跃，累积函数通常仍连续，却可能在该点出现折角——例如在 0 处从 \(-1\) 跳到 \(1\) 的符号函数，累积出 \(|x|\)，在 0 连续却有折角——其导数不能等于一个不连续的点值。
