\section{三次样条：用局部性换稳定}

\subsection{不用一条高次曲线统治整个区间}

全局高次多项式有一个结构性问题：改变一个节点会影响整个区间，端点振荡还可能随次数恶化。样条采取另一条路线：每个小区间只用低次多项式，再在节点处把各段平滑拼接。

三次插值样条 \(s\) 满足：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  在每个 \([x_i,x_{i+1}]\) 上是次数不超过 3 的多项式；
\item
  \(s(x_i)=y_i\)；
\item
  \(s,s',s''\) 在整个区间连续。
\end{enumerate}

三次不是任意选择。每段有足够自由度满足值与平滑拼接，同时二阶导连续使曲率不突然跳变；系统又保持局部带状结构。

\subsection{为什么还需要边界条件}

内部插值和 \(C^2\) 拼接仍留下两个自由度，需要两条端点条件。可以直接数出来：\(n\) 段三次式共 \(4n\) 个系数，插值条件占去 \(2n\) 个，\(n-1\) 个内节点处的一阶、二阶导连续再占去 \(2(n-1)\) 个，恰好还剩 \(2\) 个。常见选择包括：

\begin{itemize}
\tightlist
\item
  natural：\(s''(x_0)=s''(x_n)=0\)；
\item
  clamped：给定 \(s'(x_0)\) 与 \(s'(x_n)\)；
\item
  not-a-knot：让首尾两个内节点不形成真正的三次分段结点。
\end{itemize}

它们代表不同的端点信息，不是可随意互换的语法选项。若物理上知道端点斜率，clamped 往往比 natural 更合适；natural 条件并不表示真实函数端点曲率必为零，只是所选插值族的边界约束。

\subsection{三对角系统从哪里来}

记 \(h_i=x_{i+1}-x_i\)，以节点二阶导数 \(M_i=s''(x_i)\) 为未知量。每段三次多项式可由端点函数值和二阶导数表示：在 \([x_i,x_{i+1}]\) 上 \(s''\) 是端点值为 \(M_i,M_{i+1}\) 的线性函数，积分两次并用两端函数值定出积分常数，可得该段两端的一阶导数

\[
s'(x_i^{+})=\frac{y_{i+1}-y_i}{h_i}-\frac{h_i(2M_i+M_{i+1})}{6},
\qquad
s'(x_{i+1}^{-})=\frac{y_{i+1}-y_i}{h_i}+\frac{h_i(M_i+2M_{i+1})}{6}.
\]

要求一阶导数在内节点连续，即左段的 \(s'(x_i^{-})\) 等于右段的 \(s'(x_i^{+})\)，整理后得到

\[
h_{i-1}M_{i-1}
+2(h_{i-1}+h_i)M_i
+h_iM_{i+1}
=6\left(
\frac{y_{i+1}-y_i}{h_i}
-\frac{y_i-y_{i-1}}{h_{i-1}}
\right).
\]

每个方程只涉及相邻三个 \(M\)，所以形成三对角线性系统。严格递增节点与常见合法边界条件下，系统可唯一求解，计算成本为 \(O(n)\)，远低于一般稠密系统。

这正是局部结构的回报：新增或改变一点虽然会通过系统传播，但影响不像全局高次基那样无约束振荡，求值也只需找到所在区间并计算一段三次式。

\subsection{natural spline 的最小弯曲解释}

在所有通过数据点、二阶弱可导的函数中，natural cubic spline 最小化

\[
\int_{x_0}^{x_n}[g''(x)]^2\,dx.
\]

它选择弯曲能量最小的插值曲线。若写 \(g=s+h\)，其中 \(h\) 在所有节点为零，交叉项 \(\int s''h''\) 会消失：逐段分部积分后，段内积分项因 \(s'''\) 分段取常数、\(h\) 在节点处为零而为零，剩下的端点项因 natural 边界条件 \(s''=0\) 而为零。从而

\[
\int(g'')^2
=\int(s'')^2+\int(h'')^2
\ge\int(s'')^2.
\]

这为``样条看起来平滑''提供了一个明确变分意义，而不只是视觉印象。

\subsection{平滑不等于保持形状}

普通三次样条可能在单调数据之间过冲，产生负值或新的局部极值。\(C^2\) 平滑控制导数连续性，却不自动保证单调、凸性或正性。

若这些形状约束重要，可使用 PCHIP 等分段三次 Hermite 方法——正是第一节提到的 Hermite 插值的分段版本——牺牲部分二阶光滑性来保持单调。选择方法时应先问需要什么性质，而不是默认曲线越光滑越好。

对足够光滑函数和规则网格，三次样条函数值误差典型为 \(O(h^4)\)；但端点边界、非均匀网格和函数不光滑都会改变常数或阶。网格加密实验只能在进入渐近区后验证这一规律。

\subsection{构造前的实际检查}

\begin{itemize}
\tightlist
\item
  节点必须严格递增，重复节点会使 \(h_i=0\)、上述三对角方程的右端出现除零，需另行处理；
\item
  边界条件要记录，不能只说``用了 cubic spline''；
\item
  若数据含噪，精确插值样条可能仍会追踪噪声，应考虑平滑样条；
\item
  若要求单调或非负，应显式选形状保持方法；
\item
  外推行为主要由端点段与边界条件决定，通常不可靠。
\end{itemize}

\subsubsection{自己判断方法}

一组传感器校准数据理论上严格单调，但含轻微噪声。比较 natural cubic interpolation、PCHIP 和 smoothing spline 各自会保护或牺牲什么。没有唯一答案，但每个选择都应对应一个明确的业务或物理约束。
