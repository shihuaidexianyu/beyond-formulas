\section{从精确插值到整体逼近}

\subsection{为什么可以主动不通过数据点}

若观测含噪，要求曲线逐点命中意味着把噪声也当成真函数结构。即使数据无噪，整体目标也可能是控制最大误差、均方误差或计算成本，而不是每个采样点零残差。

逼近问题选择 \(p_n\in\mathbb P_n\) 使

\[
\|f-p_n\|
\]

尽量小。范数决定``最好''的含义：

\begin{itemize}
\tightlist
\item
  \(L^2\) 关注平方误差的平均能量；
\item
  离散最小二乘关注样本残差平方和；
\item
  \(L^\infty\) 关注全区间最大误差；
\item
  加权范数可以强调某些位置或概率更高的区域。
\end{itemize}

不说明范数就说``最佳多项式''，问题并未定义完整。

\subsection{\texorpdfstring{\(L^2\) 逼近是一种正交投影}{L\^{}2 逼近是一种正交投影}}

给定内积

\[
\langle f,g\rangle
=\int_a^bf(x)g(x)w(x)\,dx,
\]

若基函数 \(\phi_0,\ldots,\phi_n\) 正交，最佳 \(L^2\) 逼近

\[
p_n=\sum_{k=0}^nc_k\phi_k
\]

满足残差与整个子空间正交：

\[
\langle f-p_n,\phi_j\rangle=0.
\]

把 \(p_n\) 的展开式代入，正交性使交叉项全部消失，只剩下第 \(j\) 项，于是

\[
c_j=
\frac{\langle f,\phi_j\rangle}
{\langle\phi_j,\phi_j\rangle}.
\]

例如把 \(f(x)=x^2\) 在 \([-1,1]\) 上投影到 \(\operatorname{span}\{1,x\}\)。对称性使 \(x\) 系数为零，常数系数为

\[
\frac{\int_{-1}^1x^2dx}{\int_{-1}^11dx}
=\frac13.
\]

该子空间中的最佳近似是常数 \(1/3\)，它不通过所有函数值，却最小化整个区间的平方误差。

\subsection{为什么要使用正交多项式}

高次幂基 \(1,x,x^2,\ldots\) 在有限区间上可能高度相关，Vandermonde 或正规方程变得病态。Legendre、Chebyshev 等正交多项式让系数具有投影意义，并改善数值条件。

对离散数据做最小二乘时，应利用 QR 或 SVD 求解设计矩阵问题，而不是默认形成正规方程；正规方程会把条件数近似平方。这里的线性代数细节会在数值线性代数单元展开。

\subsection{一致逼近关心最坏点}

最小最大逼近寻找

\[
\min_{p\in\mathbb P_n}\|f-p\|_\infty.
\]

对闭区间上的连续函数，最佳多项式的误差会在至少 \(n+2\) 个点交替达到等幅正负极值，这称为 equioscillation（交错定理）。若最大误差只集中在一边，通常还能调整多项式降低峰值。

Chebyshev 多项式与这种最坏误差控制密切相关。工程中的快速特殊函数实现常更关心最大相对误差，而不是样本均方误差。

\subsection{存在性不等于算法与速率}

Weierstrass 逼近定理保证：闭区间上的任意连续函数，对任意 \(\varepsilon>0\)，都存在多项式 \(p\) 使

\[
\|f-p\|_\infty<\varepsilon.
\]

它说明多项式空间足够丰富，却没有告诉我们需要多高次数、怎样稳定找到多项式，或有限精度下如何求值。函数越光滑，通常系数衰减越快；解析函数常有快速谱收敛，不光滑点则限制收敛率。

不连续函数不能在包含跳点的闭区间上被连续多项式一致逼近到任意小误差，因为一致极限仍连续。改变范数或排除跳点邻域后，结论可能不同。

\subsection{插值还是逼近}

选择前可以问：

\begin{itemize}
\tightlist
\item
  数据是否被认为精确？
\item
  是否必须重现节点值，还是允许平滑噪声？
\item
  最关心平均误差还是最坏误差？
\item
  节点可否选择？
\item
  需要局部更新，还是一次建立全局模型？
\item
  区间内使用，还是危险的外推？
\end{itemize}

插值不是逼近的低级版本，逼近也不是``故意算不准''。它们满足不同约束，适合不同数据生成机制。
