\section{采样、混叠与可恢复性}

\subsection{不同连续信号可以留下完全相同的样本}

以采样周期 \(T\) 观察复指数：

\[
e^{2\pi i(\nu+k/T)nT}
=e^{2\pi i\nu nT}e^{2\pi ikn}
=e^{2\pi i\nu nT}.
\]

所以频率相差整数倍采样频率 \(f_s=1/T\) 的连续信号，在所有样本点完全相同。采样把无限连续世界折叠到一个有限频率区间；一旦两个频率混叠，仅凭样本无法判断原来是哪一个。

例如用 100 Hz 采样 60 Hz 正弦，其离散样本与 \(-40\) Hz、对实信号而言与 40 Hz 的某个相位表现不可区分。采样后的算法不能恢复已经丢失的身份信息。

\subsection{Shannon 采样定理给出可恢复条件}

若连续信号严格带限到 \(|\nu|<B\)，且

\[
f_s>2B,
\]

采样导致的频谱副本互不重叠，可以从样本唯一恢复原信号：

\[
x(t)=\sum_{n\in\Z}
x(nT)
\operatorname{sinc}\left(\frac tT-n\right),
\]

其中

\[
\operatorname{sinc}(u)=\frac{\sin(\pi u)}{\pi u}.
\]

\(2B\) 称为给定带宽的 Nyquist rate，\(f_s/2\) 称为给定采样率的 Nyquist frequency。二者名称接近，角色相反。

重建公式来自频域中保留中心副本的理想低通滤波；理想矩形滤波器的时域响应正是无限支撑 sinc。

\subsection{\texorpdfstring{为什么工程上不把采样率恰设为 \(2B\)}{为什么工程上不把采样率恰设为 2B}}

基本定理假设严格带限、双无限样本、精确时钟与无限支撑重建。现实信号通常不严格带限，模拟 anti-alias filter 也需要过渡带；恰在 Nyquist 边界的正弦还可能因相位采成全零。

因此实际系统会：

\begin{itemize}
\tightlist
\item
  在采样前做模拟低通；
\item
  为滤波器过渡带留出余量；
\item
  让采样率高于理论下界；
\item
  量化允许的带外 alias 能量；
\item
  单独考虑时钟抖动、量化噪声和传感器带宽。
\end{itemize}

抗混叠必须发生在采样前。数字滤波可以删除已混入带内的能量，却无法知道它原本来自带内还是带外。

\subsection{有限观察窗产生频谱泄漏}

实际只观察有限时长，相当于把无限信号乘以一个窗口。时域乘法对应频域卷积，因此理想单频谱线会被窗口频谱扩散。

若正弦频率恰落在 DFT bin，矩形窗内整周期闭合，能量集中；否则端点拼接不连续，能量泄漏到许多 bin。（DFT 下一节才正式定义；这里只需把 bin 理解为有限 \(N\) 个样本所能分辨的均匀频率格点。）Hann、Blackman 等平滑窗降低远旁瓣，却加宽主瓣，牺牲分辨率。选择窗口是在旁瓣泄漏与主瓣宽度之间权衡，无法用一个开关单向地``让频谱更准确''。

零填充会在同一有限窗频谱上增加采样点，使曲线看起来更平滑、峰值定位更细，但不会增加新的观测时间，也不会改善由窗长决定的真实频率分辨能力。

\subsection{aliasing 与 leakage 不要混为一谈}

\begin{itemize}
\tightlist
\item
  aliasing：采样率不足或信号带外能量导致不同连续频率生成同一离散频率，不可逆；
\item
  leakage：有限窗使频率能量扩散到邻近 bin，可通过窗口权衡但不能完全消除有限观测效应；
\item
  scalloping：真实频率位于 bin 之间时峰值幅度被低估；
\item
  quantization：幅值离散化造成另一类噪声。
\end{itemize}

看到频谱``脏''时，必须先判断是哪一种机制，不能统称 FFT 误差。

\subsection{采样定理的边界}

真实有限时长信号不可能同时严格时限和严格带限。非均匀采样、稀疏先验和压缩感知可以在不同假设下重建，但不是对 Shannon 定理的简单绕过；它们加入了新的结构信息。
