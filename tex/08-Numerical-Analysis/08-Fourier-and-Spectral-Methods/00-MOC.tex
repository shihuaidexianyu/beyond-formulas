正弦和复指数不是只用于``找周期''的技巧，而是一组正交坐标。进入频率空间后，微分变成乘以频率，卷积变成逐点相乘，函数光滑性表现为高频系数衰减。连续函数被采样后又会出现混叠、频谱泄漏和离散周期性；FFT 只加速 DFT，并不会自动修复这些信息损失。而当频率成分随时间变化时，Fourier 坐标还会抹掉``何时''；加窗与小波给出两种把时间与频率（尺度）联合起来的折中。

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  从 Fourier 级数到连续变换
\item
  采样、混叠与可恢复性
\item
  DFT 是变换，FFT 是算法
\item
  谱方法：用光滑性换高精度
\item
  时频局部化：Fourier 丢掉了``何时''
\item
  小波与多尺度分解
\end{enumerate}
