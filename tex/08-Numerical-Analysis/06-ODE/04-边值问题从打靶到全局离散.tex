\section{边值问题：从打靶到全局离散}

\subsection{两端条件打破了单向推进}

初值问题在左端给足状态，可以沿时间向前积分。边值问题却把条件分散在区间两端，例如

\[
-y''(x)=f(x),\qquad y(0)=0,\quad y(1)=0.
\]

左端只知道 \(y(0)\)，不知道初始斜率；随便向前走一般不会命中右端条件。问题的未知量是整条曲线，而不只是下一步状态。

边界条件可能是：

\begin{itemize}
\tightlist
\item
  Dirichlet：给函数值；
\item
  Neumann：给导数或通量；
\item
  Robin：值与导数的线性组合；
\item
  非线性或多点条件。
\end{itemize}

边界类型直接影响解的存在性、唯一性和离散系统结构。

\subsection{shooting：把未知初值变成求根}

对二阶方程，把未知初始斜率记为 \(s\)，从

\[
y(a)=\alpha,\qquad y'(a)=s
\]

解一个 IVP。终点残差

\[
F(s)=y(b;s)-\beta
\]

应为零，于是外层用二分、割线或 Newton 求 \(F(s)=0\)。

shooting 可以复用成熟 IVP 求解器，适合低维、稳定且初值敏感性不过强的问题。但若轨迹包含增长模态，初始斜率的微小误差会在长区间指数放大，\(F(s)\) 极病态。multiple shooting 把区间切段，为各段引入初值，并用匹配条件联立，缓解单段传播敏感性。

\subsection{有限差分：把整条曲线一起求}

取网格 \(x_i=ih\)，中心差分

\[
y''(x_i)
\approx\frac{y_{i-1}-2y_i+y_{i+1}}{h^2}.
\]

它由 \(y(x_i+h)\) 与 \(y(x_i-h)\) 的 Taylor 展开相加得到：奇数阶导数项相消，截断误差为 \(\frac{h^2}{12}y''''(\xi_i)\)，所以是二阶近似。

对 \(-y''=f\) 得

\[
-y_{i-1}+2y_i-y_{i+1}=h^2f(x_i).
\]

连同边界值，形成三对角线性系统。它把全区间耦合一次表达出来，不会像单 shooting 那样沿不稳定方向逐步放大。

若连续问题适定、差分一致为 \(O(h^2)\)，且离散逆算子一致有界，精确解采样代入产生的截断残差 \(\tau=O(h^2)\)；误差满足

\[
A_he=-\tau,
\]

从而（取任一向量范数及与其相容的矩阵范数）

\[
\|e\|\le\|A_h^{-1}\|\|\tau\|=O(h^2).
\]

这再次体现``一致性加稳定性推出收敛''，而不只是 Taylor 公式本身。

\subsection{collocation 与更一般离散}

配置法先选有限维近似 \(y_h\)，要求微分方程在若干 collocation nodes 精确成立，并同时满足边界条件。基函数可取分段多项式、样条或全局谱多项式；有限元则通过弱形式控制残差。

光滑问题上谱配置可以非常快速收敛，不规则几何和局部变化则更适合有限元或自适应网格。方法选择取决于解的光滑性、维度、边界和结构，而不是``高阶总是更好''。

\subsection{边界条件可能让问题无解或不唯一}

纯 Neumann 问题

\[
-y''=f,\qquad y'(0)=y'(1)=0
\]

对方程两端在 \([0,1]\) 上积分，得 \(-\bigl(y'(1)-y'(0)\bigr)=\int_0^1 f(x)\,dx\)，所以两个 Neumann 条件若成立，必须满足兼容条件

\[
\int_0^1f(x)\,dx=0;
\]

且解只确定到一个加法常数：若 \(y\) 是解，\(y+C\) 也满足方程与两个边界条件。离散矩阵相应奇异；随意加小对角线会改变问题，而不是修复实现。应加入均值归一化或固定一个自由度，并检查兼容性。

非线性 BVP 还可能有多个解，Newton 初值会决定收敛到哪个分支。小代数残差只说明离散系统求得好，不能替代网格加密验证连续误差。

\subsection{验证必须同时覆盖三层}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  代数层：离散线性或非线性残差；
\item
  边界层：所有边界条件是否按正确尺度满足；
\item
  连续层：网格加密是否呈预期收敛阶，守恒或物理结构是否保持。
\end{enumerate}

只检查节点方程而不加密网格，可能精确解了一个粗糙但错误的离散问题。
