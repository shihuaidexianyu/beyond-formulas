\section{Runge--Kutta 与自适应步长}

\subsection{Euler 只看起点斜率，能否在一步内多看几次}

若先用起点斜率预测中点，再在中点重新评估斜率，就得到显式中点法：

\[
k_1=f(t_n,y_n),
\]

\[
k_2=f\left(t_n+\frac h2,
y_n+\frac h2k_1\right),
\]

\[
y_{n+1}=y_n+hk_2.
\]

中点斜率比起点斜率更能代表整个区间，使全局误差提升到 \(O(h^2)\)。二阶从何而来？在中点作 Taylor 展开，偶数阶误差项相消（假设解足够光滑）：

\[
y(t_n+h)=y(t_n)+h\,y'\!\left(t_n+\tfrac h2\right)+O(h^3),
\]

即``中点斜率乘步长''的局部误差已是 \(O(h^3)\)；而预测中点 \(y_n+\frac h2k_1\) 与真实中点只差 \(O(h^2)\)，经 Lipschitz 连续的 \(f\) 和外层因子 \(h\) 传入后，局部误差仍是 \(O(h^3)\)。局部三阶累积成全局二阶，与上一节``局部高一阶、全局低一阶''是同一机制。

Runge--Kutta（RK）方法系统化了这种``一步内多次采样斜率，再加权组合''的思想。

一般 \(s\) 阶段 RK 写成

\[
k_i=f\left(t_n+c_ih,
y_n+h\sum_{j=1}^sa_{ij}k_j\right),
\]

\[
y_{n+1}=y_n+h\sum_{i=1}^sb_ik_i.
\]

系数用 Butcher tableau 表示。显式 RK 的 \(a_{ij}=0\) 对 \(j\ge i\)，阶段可按顺序直接计算；隐式 RK 的阶段相互耦合，需要解方程。

\subsection{经典 RK4 为什么是四阶}

经典 RK4 在起点、两个预测中点和终点取四个斜率，权重为

\[
\frac16(1,2,2,1).
\]

其局部截断误差为 \(O(h^5)\)，固定区间全局误差为 \(O(h^4)\)。四个阶段并不自动意味着四阶；阶数来自系数满足一组 Taylor 或 rooted-tree 匹配条件。随便挑四次斜率通常不会得到 RK4 的精度。

高阶方法减少达到某精度所需步数，但每步函数评估更多，稳定区也不一定更适合当前问题。比较方法要计总函数评估、Jacobian/线性求解、内存与事件开销，而不只看阶数。

\subsection{固定步长不知道哪里困难}

轨迹可能大部分平缓，只在短时间内快速变化。固定小步长浪费平缓区，固定大步长又会跨过困难区。嵌入式 RK 对同一组阶段使用两套权重，得到 \(p\) 阶与较低阶两个结果；差值估计当前步的局部误差。

对多分量状态，常用缩放误差

\[
E=\left[
\frac1d\sum_{j=1}^d
\left(
\frac{e_j}{a_{\rm tol}+r_{\rm tol}\max(|y_{n,j}|,|y_{n+1,j}|)}
\right)^2
\right]^{1/2}.
\]

若 \(E\le1\) 接受该步，否则拒绝并缩小步长。新步长大致按

\[
h_{\rm new}
=\text{safety}\times h E^{-1/(p+1)}
\]

调整，并限制最大放大/缩小比例以避免剧烈震荡。

绝对容差处理接近零的分量，相对容差处理不同数量级；只设一个统一绝对容差会让大、小分量的意义失衡。

\subsection{局部控制不等于全局误差保证}

嵌入差值只估计当前步、当前解附近的截断误差。非线性动力学会继续放大或衰减这些误差（与上一节局部误差累积成全局误差是同一机制），模型参数与初值误差也未包含其中。局部容差通常与全局精度相关，但不能简单解释成``最终答案误差必小于 tol''。

误差估计还假设两套方法处于渐近区。函数不光滑、事件附近或刚性导致的 order reduction 都会让估计失真。

\subsection{事件与稠密输出}

若要检测 \(g(t,y)=0\) 的碰撞或开关事件，不能只看离散时间点。求解器在接受步内构造 dense output 插值，再对事件函数求根，定位准确时间，并按需要终止或重启动力系统。

跨过不连续事件后继续沿旧的高阶多项式，会破坏阶数与物理状态。事件处理是 ODE 算法的一部分，不是最后绘图时补一个点。

\subsection{何时不用经典显式 RK}

\begin{itemize}
\tightlist
\item
  强刚性：稳定性会迫使极小步（下一节详细分析），应使用 BDF、Radau 或其他隐式方法；
\item
  Hamilton 系统长时积分：辛方法可能比一般高阶 RK 更保结构；
\item
  强不连续或随机驱动：需匹配专用方法；
\item
  极大系统：Jacobian-free、指数积分或分裂方法可能更合适。
\end{itemize}

``RK4 很准''只在光滑、非刚性、适当步长的 IVP 中成立。
