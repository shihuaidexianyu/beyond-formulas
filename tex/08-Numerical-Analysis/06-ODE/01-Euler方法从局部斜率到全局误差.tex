\section{Euler 方法：从局部斜率到全局误差}
\label{sec:08-Numerical-Analysis/06-ODE/01}

\begin{quote}
\textbf{先抓住这一点：}Euler 方法拿当前位置的切线当作下一小段轨迹。一步造成的误差虽是 \(O(h^2)\)，走完固定时间却要约 \(1/h\) 步，误差还会一路传播，所以最终通常只剩一阶精度。
\end{quote}

\subsection{一条切线能把我们带多远}

初值问题写成

\[
y'(t)=f(t,y(t)),\qquad y(t_0)=y_0.
\]

已知 \((t_n,y_n)\)，最直接的预测是沿当前斜率走一步：

\[
\boxed{
y_{n+1}=y_n+h f(t_n,y_n).
}
\]

这就是显式 Euler。它把微分方程的局部信息转换为一段直线运动；曲率越大或步长越长，直线偏离真实轨迹越多。

\subsection{局部二阶为何只得到全局一阶}

把精确解从 \(t_n\) 展开：

\[
y(t_n+h)
=y(t_n)+h f(t_n,y(t_n))
+\frac{h^2}{2}y''(\xi_n).
\]

若一步从精确起点出发，Euler 的局部截断误差为 \(O(h^2)\)。但在固定时间区间 \([t_0,T]\) 上要走

\[
N\approx\frac{T-t_0}{h}
\]

步，旧误差还会通过动力系统继续传播。

记 \(e_n=y(t_n)-y_n\)。把上面的 Taylor 等式与 Euler 步相减，得

\[
e_{n+1}
=e_n+h\bigl[f(t_n,y(t_n))-f(t_n,y_n)\bigr]
+\frac{h^2}{2}y''(\xi_n).
\]

若 \(f\) 对 \(y\) 满足 Lipschitz 条件（常数为 \(L\)），且解足够光滑使
\(|y''|\le 2C\)，则误差递推形如

\[
|e_{n+1}|
\le(1+hL)|e_n|+Ch^2.
\]

逐步展开这个递推，并用 \(1+hL\le e^{hL}\)（这正是离散 Grönwall 不等式的内容），得

\[
|e_n|
\le Ch^2\sum_{k=0}^{n-1}(1+hL)^k
=\frac{Ch}{L}\bigl[(1+hL)^n-1\bigr]
\le\frac{Ch}{L}\bigl(e^{L(t_n-t_0)}-1\bigr),
\]

于是在固定区间上

\[
\max_{t_n\le T}|e_n|=O(h).
\]

所以局部二阶缺陷积累成全局一阶误差。把局部截断阶与全局阶混为一谈，是 ODE 方法中最常见的概念混淆。

\subsubsection{例：指数衰减}

对

\[
y'=-y,\qquad y(0)=1,
\]

显式 Euler 给

\[
y_n=(1-h)^n,\qquad t_n=nh,
\]

而真解为 \(e^{-t_n}\)。固定 \(T\)、把 \(h\) 减半时，全局误差渐近约减半，验证一阶收敛。

但若 \(h>2\)，\(|1-h|>1\)，数值解反而增长，尽管真实解衰减。这个现象受稳定性限制，无法只用截断阶解释；后面\hyperref[sec:08-Numerical-Analysis/06-ODE/03]{``稳定区域与刚性方程''}一节会把它变成可计算的判据。

一杯热咖啡放在桌上时，它与室温的温差 \(e\) 会平滑衰减，不会忽冷忽热。简单热平衡模型写成

\[
e'=-\frac{k}{C}e.
\]

其中 \(k\) 表示有效换热强度，\(C\) 表示热容量。显式 Euler 离散化为

\[
e_{n+1}
=\left(1-\frac{hk}{C}\right)e_n.
\]

连续系统始终衰减，离散近似却只有在

\[
\left|1-\frac{hk}{C}\right|<1,
\qquad\text{即}\qquad
0<h<\frac{2C}{k}
\]

时才保持这种定性行为。若 \(hk/C>1\)，数值温差会正负交替，仿佛咖啡一会儿比房间热、一会儿又比房间冷；若 \(hk/C>2\)，这种虚构的摆动还会越来越大。错误来自时间步过长，不来自咖啡真的违反了热传导。

冷库温控沿用同一个模型时，\(h\) 就变成控制器读取温度和更新动作的时间间隔。软件若取值过大，可能在仿真里制造真实冷库并不存在的振荡；若据此继续调大控制增益，又会把数值伪影变成真实设备上的激烈动作。步长在这里成为控制器与物理系统之间的一项设计参数，已经超出绘图精细度的范畴。

\subsection{后向 Euler：使用未来斜率}

将斜率取在新点，得到

\[
y_{n+1}=y_n+h f(t_{n+1},y_{n+1}).
\]

\(y_{n+1}\) 同时出现在等式两边，每步需要解线性或非线性方程，因此称为隐式方法。对 \(y'=-y\)，方程 \(y_{n+1}=y_n-h y_{n+1}\) 可以直接解出

\[
y_{n+1}=\frac{y_n}{1+h},
\]

任意 \(h>0\) 都保持衰减。

后向 Euler 与显式 Euler 都是一阶，却有完全不同的稳定区域。每步更贵，换来对快速衰减模态的强阻尼。隐式不等于自动精确：非线性 Newton 迭代、Jacobian 近似和线性求解容差都会进入总误差。

\subsection{一致性、稳定性与收敛性}

一个离散方法要逼近连续问题，至少要区分：

\begin{itemize}
\tightlist
\item
  一致性：精确解代入一步公式时，局部缺陷随 \(h\to0\) 消失；
\item
  稳定性：局部扰动不会在离散推进中被不合理放大；
\item
  收敛性：固定时间区间上，数值解随 \(h\to0\) 逼近精确解。
\end{itemize}

高阶一致公式若不稳定，仍可能完全错误；稳定方法若步长太大，也可能只得到稳定但不准确的轨迹。

\subsection{ODE 计算的误差不只一种}

真实求解器还要处理：

\begin{itemize}
\tightlist
\item
  初值和参数误差；
\item
  时间离散误差；
\item
  每步浮点误差；
\item
  隐式内层求解误差；
\item
  事件定位误差；
\item
  模型本身与现实的偏差。
\end{itemize}

把步长减到极小只能压低时间离散误差，对其余各类无能为力；步长过小反而会让舍入累积和计算成本上升。

\subsection{阶数实验应该怎样做}

选有解析解且足够光滑的问题，在同一终止时间分别使用 \(h,h/2,h/4\)。若 Euler 已进入渐近区，误差比应接近 2。测试还应确保各步长都在稳定范围内；否则测到的是不稳定爆炸，而不是截断阶。
