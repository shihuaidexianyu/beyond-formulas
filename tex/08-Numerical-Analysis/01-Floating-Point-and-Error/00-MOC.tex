数学公式通常在实数上成立，程序却只能在有限集合中运算。数值分析的第一课是学会拆分三种来源：输入或问题本身是否敏感，离散近似省略了什么，以及算法怎样传播每一步舍入。机器精度要放在这三种来源中理解。只有分清它们，才知道该换算法、换精度、换模型，还是承认数据已经不足以决定答案。

建议依次阅读：

\begin{enumerate}
\tightlist
\item
  从实数到浮点数
\item
  误差如何产生与放大
\item
  条件数衡量问题本身的敏感性
\item
  前向误差、后向误差与稳定算法
\end{enumerate}

\subsection{怎么读这一章}

第一次读，不必先背 binary64 的每个字段。更重要的是学会把一次错误分成三问：输入稍微变一点，正确答案会不会大变；所用公式是否省略了某些项；浮点算法又额外丢了多少信息。前三问分别指向条件性、截断误差和舍入传播，只有分开以后，才知道提高精度是否真的有用。
