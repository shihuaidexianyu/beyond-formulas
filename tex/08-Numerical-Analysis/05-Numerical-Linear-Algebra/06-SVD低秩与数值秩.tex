\section{SVD、低秩与数值秩}

\subsection{任意矩阵都有一组最自然的输入输出方向}

奇异值分解写成

\[
A=U\Sigma V^*,
\]

其中 \(U,V\) 正交或酉，

\[
\Sigma=\operatorname{diag}(\sigma_1,\ldots,\sigma_r),
\qquad
\sigma_1\ge\cdots\ge\sigma_r\ge0.
\]

\(V\) 的第 \(i\) 个方向 \(v_i\) 先被 \(A\) 放大 \(\sigma_i\) 倍，再转到 \(U\) 的第 \(i\) 个方向 \(u_i\)，即 \(Av_i=\sigma_iu_i\)。它适用于方阵、长方阵、奇异矩阵和非正规矩阵，直接揭示秩、条件数、列空间、零空间与最小二乘结构。

\subsection{\texorpdfstring{为什么不应默认从 \(A^*A\) 求 SVD}{为什么不应默认从 A\^{}*A 求 SVD}}

理论上，\(V\) 是 \(A^*A\) 的特征向量，特征值为 \(\sigma_i^2\)。但显式形成 \(A^*A\) 会让条件数平方（与前面最小二乘正规方程的讨论是同一陷阱）：

\[
\kappa_2(A^*A)=\kappa_2(A)^2.
\]

很小的奇异值平方后更接近舍入尺度，可能完全丢失。标准稠密 SVD 算法先用稳定正交变换把 \(A\) 双对角化，再在双对角矩阵上迭代；还包括 divide-and-conquer、Jacobi 等变体。

计算 SVD 的正确方式是调用可靠库，而不是自己拼 \(A^TA\) 的 eig。

\subsection{截断 SVD 为什么是最佳低秩近似}

保留前 \(k\) 项：

\[
A_k=U_k\Sigma_kV_k^*.
\]

Eckart--Young 定理给

\[
\|A-A_k\|_2=\sigma_{k+1},
\]

并且 \(A_k\) 在所有秩不超过 \(k\) 的矩阵中最小化 2-范数误差；在 Frobenius 范数下也最佳，误差为

\[
\|A-A_k\|_F
=\left(\sum_{i>k}\sigma_i^2\right)^{1/2}.
\]

直觉上，任何秩 \(k\) 近似最多保留 \(k\) 个独立输入方向，至少会漏掉前 \(k+1\) 个右奇异方向中的一个；该方向被 \(A\) 放大至少 \(\sigma_{k+1}\)。截断 SVD 恰好把被漏掉的最大放大控制在这个下界。

这解释了 PCA、图像压缩和潜在因子模型为什么从最大奇异值方向开始保留。

\subsection{数值秩与阈值}

精确算术中，非零奇异值个数就是秩；含噪数据几乎总会把理论零奇异值抬成小正数。数值秩需要阈值，例如把

\[
\sigma_i\le\tau\sigma_1
\]

视为不可辨识，其中 \(\tau\) 应结合矩阵尺寸、机器精度和数据噪声。

没有尺度说明的``秩为 7''不是完整数值结论。换单位可能整体缩放奇异值，噪声模型也决定哪些方向应被当作信号。

伪逆

\[
A^+=V\Sigma^+U^*
\]

会把非零奇异值取倒数。很小的 \(\sigma_i\) 取倒数后会把噪声巨大放大；截断 SVD 或 Tikhonov 正则化通过衰减这些方向来稳定逆问题，但也引入偏差。

\subsection{奇异值稳定，奇异向量未必稳定}

奇异值满足 Weyl 扰动界

\[
|\sigma_i(A+E)-\sigma_i(A)|
\le\|E\|_2.
\]

但若两个奇异值相等或间隙很小，对应单个奇异向量可以在其子空间内大幅旋转。向量符号或复相位本来也不唯一。

因此验证时应：

\begin{itemize}
\tightlist
\item
  检查重构残差 \(\|A-U\Sigma V^*\|\)；
\item
  检查 \(U,V\) 正交性；
\item
  检查奇异值非负有序；
\item
  对重复或聚集奇异值比较子空间，而不是逐元素比较向量；
\item
  不把符号相反的向量误判为错误。
\end{itemize}

\subsection{大规模截断 SVD}

若矩阵巨大且只需前 \(k\) 个方向，完整 SVD 浪费资源。Lanczos 双对角化利用 matvec；randomized SVD 先用随机投影捕获近似列空间，再在小矩阵上做精确 SVD。后者的精度取决于 oversampling、幂迭代次数与谱衰减。

随机方法应报告近似残差和随机参数，而不是因输出形状与 SVD 相同就当作精确分解。谱间隙小或尾部衰减慢时，需要更多子空间维度和幂迭代。
