\section{线性方程为何分解而不求逆}

\subsection{\texorpdfstring{公式 \(x=A^{-1}b\) 隐藏了真正的计算任务}{公式 x=A\^{}\{-1\}b 隐藏了真正的计算任务}}

若目标只是解

\[
Ax=b,
\]

显式计算整个 \(A^{-1}\) 会求出许多并不需要的列，增加运算、存储与舍入。数值库中的 solve 通常不会先形成逆矩阵，而是把 \(A\) 分解成容易反复求解的结构。

对一般稠密方阵，Gaussian 消元等价于带主元 LU 分解

\[
PA=LU,
\]

其中 \(P\) 是行置换，\(L\) 是单位下三角，\(U\) 是上三角。分解之后，每次求解只需先置换右端，再做前代与回代两次三角求解：

\[
Ly=Pb,\qquad Ux=y.
\]

分解成本约 \(O(n^3)\)，每个右端的前代与回代只需 \(O(n^2)\)。若同一个 \(A\) 对应很多 \(b\)，复用分解尤其重要。

\subsection{消元为什么需要换行}

矩阵

\[
A=\begin{pmatrix}0&1\\1&1\end{pmatrix}
\]

明明可逆，却不能从左上角零主元直接开始消元。交换两行后即可继续。

即使主元非零，若它非常小，消元乘子也会巨大，把舍入误差与中间元素放大。部分选主元在当前列未处理行中选择绝对值最大的元素作为主元，使该列消元乘子绝对值不超过 1。

这产生 \(PA=LU\) 而非简单 \(A=LU\)。置换不是实现杂项，而是算法稳定性的组成部分。

\subsection{后向稳定还要看元素增长}

浮点 LU 计算出的 \(\hat L,\hat U\) 通常可解释为

\[
P(A+\Delta A)=\hat L\hat U,
\]

其中 \(\Delta A\) 的大小由机器精度、矩阵规模和消元过程中元素增长控制。增长因子可写成

\[
\rho=
\frac{\max_{i,j,k}|a_{ij}^{(k)}|}
{\max_{i,j}|a_{ij}|}.
\]

部分选主元在实际问题中通常表现后向稳定，但存在罕见的指数增长最坏例。因此更准确的说法是：误差界含增长因子，而不是``部分选主元无条件稳定''。

稀疏矩阵中还要在数值主元与 fill-in 之间取舍。最稳定的主元可能破坏稀疏结构，最省填充的顺序又可能遇到小主元，实际求解器会结合重排序、阈值选主元和缩放。

\subsection{小残差与真解精度之间隔着条件数}

计算得到 \(\hat x\) 后，残差

\[
r=b-A\hat x
\]

是可直接检查的。若 \(r\) 小，\(\hat x\) 精确解了右端略微改变的系统

\[
A\hat x=b-r.
\]

但前向误差满足

\[
x-\hat x=A^{-1}r.
\]

所以病态矩阵可能把很小残差放大成很大解误差。一个成熟 LU 求解器能做到接近机器精度的后向误差，却不能恢复输入数据中因大条件数丢失的有效位。

实际可使用缩放残差，例如

\[
\frac{\|b-A\hat x\|}
{\|A\|\|\hat x\|+\|b\|},
\]

再结合条件数估计判断前向精度。只报告 residual norm 而不说明尺度，会让不同问题无法比较。

\subsection{结构比通用公式更有价值}

若矩阵具有额外结构，通用 pivoted LU 未必是最佳选择：

\begin{itemize}
\tightlist
\item
  对称正定：Cholesky 更快、更省存储；
\item
  长方形最小二乘：QR 比正规方程更稳定；
\item
  对称不定：使用保持对称结构的带主元 \(LDL^T\)；
\item
  带状、稀疏或块结构：选择对应专用分解；
\item
  极大稀疏系统：直接分解可能因存储不可行，应转向迭代法。
\end{itemize}

算法接口都可能叫 solve，但背后的数学假设不同。把一般矩阵误报为正定以换速度，得到的不是``近似更快''，而是调用了不满足前提的算法。

\subsection{迭代改进}

若已有近似解 \(x_k\)，可以计算高精度或可靠残差

\[
r_k=b-Ax_k,
\]

再用现有 LU 因子求修正

\[
A\delta x_k=r_k,\qquad x_{k+1}=x_k+\delta x_k.
\]

这称为 iterative refinement。混合精度中可用低精度快速分解，再用较高精度残差恢复精度，但是否成功仍取决于条件数、因子误差与残差精度。

\subsubsection{判断题}

``形成 \(A^{-1}\) 后，多个右端只需矩阵乘法，所以一定比保存 LU 更合适。''指出这句话忽略的至少三项成本或稳定性问题。
