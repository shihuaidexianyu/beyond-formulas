\section{大型稀疏系统与 Krylov 迭代}
\label{sec:08-Numerical-Analysis/05-Numerical-Linear-Algebra/04}

\begin{quote}
\textbf{先抓住这一点：}矩阵大到不能分解时，Krylov 方法只反复计算``矩阵乘一个向量''，并从这些结果张成的空间里找答案。预条件器不会改变原方程的解，它只是换一种更容易走向这个解的坐标和尺度。
\end{quote}

\subsection{当分解本身装不进内存}

PDE、图与优化问题产生的矩阵可能有百万维，却每行只有少量非零元。稠密 \(O(n^3)\) 分解既浪费稀疏性，fill-in 也可能让因子无法存储。迭代法只要求矩阵---向量乘，有时甚至不显式存矩阵，只提供算子 \(v\mapsto Av\)。

目标转向用有限次 matvec、内积和预条件应用得到满足业务精度的近似解，无须完成一次精确分解。

\subsection{平稳迭代先建立基本图景}

把 \(A=M-N\)，可写

\[
Mx_{k+1}=Nx_k+b,
\]

即

\[
x_{k+1}=Gx_k+c,\qquad G=M^{-1}N.
\]

误差满足

\[
e_{k+1}=Ge_k,\qquad e_k=G^ke_0.
\]

记 \(\rho(G)\) 为 \(G\) 的谱半径，即其特征值模的最大值。要求 \(e_k=G^ke_0\) 对任意 \(e_0\) 都趋于零，等价于 \(G^k\to0\)，而这正等价于谱半径小于 1：

\[
\rho(G)<1.
\]

Jacobi、Gauss--Seidel 等方法属于这一类。它们简单、局部，却可能收敛很慢；在多重网格中则常作为平滑器而非独立求解器。

\subsection{Krylov 空间从残差中积累可用方向}

给定初值 \(x_0\) 和残差 \(r_0=b-Ax_0\)，第 \(k\) 个 Krylov 空间是

\[
\mathcal K_k(A,r_0)
=\operatorname{span}
\{r_0,Ar_0,\ldots,A^{k-1}r_0\}.
\]

方法在仿射空间

\[
x_0+\mathcal K_k(A,r_0)
\]

中选择最合适的近似。每增加一步，就允许一个更高次的矩阵多项式作用于初始残差；收敛快慢取决于是否存在低次多项式能在相关谱上近似 \(1/z\)。

\subsection{CG、MINRES 与 GMRES 不是可互换名字}

共轭梯度法（CG）要求 \(A\) 对称正定（SPD）。它在 \(A\)-范数 \(\|e\|_A=\sqrt{e^{\trans}Ae}\) 下选择最优误差，并在精确算术下至多 \(n\) 步得到精确解。经典界为

\[
\frac{\|e_k\|_A}{\|e_0\|_A}
\le2\left(
\frac{\sqrt\kappa-1}{\sqrt\kappa+1}
\right)^k,
\]

其中 \(\kappa=\kappa_2(A)\)。实际收敛还受特征值聚集影响，可能比最坏条件数界好得多。

MINRES 适用于对称但可能不定的矩阵；GMRES 适用于一般非对称矩阵，并在 Krylov 空间中最小化 2-范数残差。完整 GMRES 需要保存和正交化越来越多的基向量，常用重启降低内存，但重启可能显著拖慢或停滞。

把非 SPD 矩阵交给 CG，会破坏推导所依赖的内积与正定性，后果远比``可能慢一点''严重。

\subsection{预条件决定实际成败}

若 \(M\approx A\) 且易于求解，用 \(M^{-1}\) 把系统变成谱更有利的等价问题。好的预条件器应：

\begin{itemize}
\tightlist
\item
  每次应用便宜；
\item
  显著聚集特征值或降低有效条件数；
\item
  保持所选迭代法需要的对称/正定结构；
\item
  并行与存储成本可接受。
\end{itemize}

Jacobi、块对角、不完全 LU/Cholesky、多重网格和问题专用近似都可作为预条件。最少迭代次数不一定最快：昂贵预条件器可能让每步成本超过节省。

\subsection{残差停止与前向误差}

求解器常监控相对残差

\[
\frac{\|r_k\|}{\|b\|}
\]

或预条件残差。但病态系统仍可能小残差、大前向误差——第一节已在 LU 场景看到，前向误差是残差经 \(A^{-1}\) 放大后的结果；不同预条件残差也未必等于原系统残差。应根据应用目标设置绝对与相对容差，并定期计算真实残差。

有限精度下 Krylov 基会失去正交性，CG 的``至多 \(n\) 步''不再是实际承诺。停滞可能来自舍入、预条件失效、尺度不当或右端已接近不可辨识方向。

\subsection{方法选择的最小决策树}

\begin{itemize}
\tightlist
\item
  小到中等稠密、需要高可靠性：直接分解；
\item
  稀疏 SPD：CG 加 SPD 预条件；
\item
  稀疏对称不定：MINRES 或专用 \(LDL^{\trans}\)；
\item
  一般非对称：GMRES、BiCGSTAB 等并结合预条件；
\item
  多个右端：衡量复用直接因子、块 Krylov 或回收子空间；
\item
  只有算子接口：matrix-free Krylov。
\end{itemize}
