\section{Cholesky：利用正定结构}

\subsection{正定性让消元不再需要一般主元策略}

若实对称矩阵 \(A\) 满足

\[
x^TAx>0\qquad\text{对所有 }x\ne0,
\]

则存在唯一的下三角矩阵 \(L\)，其对角元为正，使

\[
\boxed{A=LL^T.}
\]

这就是 Cholesky 分解。正定性保证消元过程中需要开方的量为正，不必像一般 LU 那样做行主元；只存一个三角因子，运算量约为一般 LU 的一半。

对复 Hermitian 正定矩阵，相应形式是 \(A=LL^*\)。

\subsection{公式怎样从逐项匹配产生}

由

\[
a_{ij}=\sum_{k=1}^{\min(i,j)}l_{ik}l_{jk},
\]

按列递推：

\[
l_{jj}=\sqrt{
a_{jj}-\sum_{k<j}l_{jk}^2
},
\]

\[
l_{ij}=
\frac{
a_{ij}-\sum_{k<j}l_{ik}l_{jk}
}{l_{jj}},\qquad i>j.
\]

根号内量是逐步形成的 Schur complement 对角元。它必为正可用一步论证看出：消去前若干列后，剩余块是 \(S=A_{22}-A_{21}A_{11}^{-1}A_{12}\)；对任意 \(x\ne0\)，取 \(y=-A_{11}^{-1}A_{12}x\)，直接展开可得

\[
x^TSx=
\begin{pmatrix}y\\x\end{pmatrix}^{T}
A
\begin{pmatrix}y\\x\end{pmatrix}>0,
\]

所以 Schur complement 仍正定，其正对角元正是根号内的量，开方恒有意义。

解 \(Ax=b\) 时依次做

\[
Ly=b,\qquad L^Tx=y.
\]

若有多个右端，复用 \(L\)。

Gaussian 过程给出了一个常见的完整计算链。给定协方差矩阵 \(K\) 和观测向量 \(y\)，预测与似然计算通常需要 \(K\alpha=y\) 以及 \(\log\det K\)。若 \(K=LL^T\)，前者只需两次三角求解，后者则由

\[
\log\det K
=2\sum_i\log l_{ii}
\]

直接得到；既不需要显式形成 \(K^{-1}\)，也不需要把行列式先算成一个可能上溢或下溢的巨大数。若重复或极接近的采样位置使 \(K\) 接近奇异，后文的 jitter 又会同时改变数值条件与观测噪声模型。于是一次分解的选择，连接了计算成本、稳定性和统计解释。

\subsection{正定矩阵从哪里来}

\begin{itemize}
\tightlist
\item
  二次能量与凸二次优化的 Hessian；
\item
  独立观测下最小二乘正规方程的系数矩阵 \(A^TA\)（当 \(A\) 满列秩）；
\item
  合法协方差矩阵在严格正定情形；
\item
  椭圆型 PDE 离散；
\item
  Gaussian 模型的精度矩阵。
\end{itemize}

Cholesky 不只节省计算，也作为结构检查。若分解在某步根号内出现非正数，可能说明矩阵不对称、不正定、数据构造错误，或矩阵极接近半正定而受到舍入影响。

\subsection{\texorpdfstring{为什么不应为用 Cholesky 而强行形成 \(A^TA\)}{为什么不应为用 Cholesky 而强行形成 A\^{}TA}}

最小二乘正规方程 \(A^TAx=A^Tb\) 确实可用 Cholesky，但它把条件数平方（上一节已分析正规方程的这一问题）。Cholesky 对正规方程的分解本身可以后向稳定，却是对一个已经更病态的问题稳定。

这再次说明：稳定算法无法消除前一步问题变换制造的条件恶化。最小二乘默认用 QR 或 SVD，不是因为 Cholesky 算法差，而是因为形成 \(A^TA\) 改变了数值敏感性。

\subsection{半正定与``加 jitter''}

协方差矩阵可能只半正定，含精确零特征值，此时标准 Cholesky 不存在正对角因子。实践中常加

\[
A_\lambda=A+\lambda I
\]

使其严格正定。这个操作称为 regularization 或 jitter，但它改变了问题：特征值整体抬高，条件数改善，同时模型也加入了额外噪声或惩罚。

不能把加 jitter 当作不需记录的数值补丁。应说明 \(\lambda\) 的尺度、选择原因，以及结果对它是否敏感。

\subsection{稀疏 Cholesky 的真正成本是填充}

稀疏正定矩阵分解时，\(L\) 可能比 \(A\) 稠密。通过对未知量重排序可显著减少 fill-in；图结构、嵌套剖分与近似最小度排序因此成为算法核心。算术复杂度不只由矩阵维度和非零数决定，也由消元图决定。

大型问题若填充仍不可接受，可把不完全 Cholesky 用作共轭梯度法预条件器，在直接分解与纯迭代之间取折中。

\subsection{结果检查}

检查

\[
\frac{\|A-LL^T\|}{\|A\|}
\]

以及三角结构、正对角与对称性。分解成功并不证明输入是数学上严格正定到任意精度；分解失败也应先检查尺度与对称误差，而不是立即加一个任意大对角量。
