求 \(f(x)=0\) 时，我们通常只能查询函数值，有时还能得到导数。二分法用一个含根区间换取确定保证；Newton 法用局部导数换取二次速度；割线法则用历史函数值近似导数。理解时要知道每种方法维护了什么证据、何时进入快收敛区，以及停止条件能证明什么；迭代公式要放在这些判断中记忆。

\begin{enumerate}
\tightlist
\item
  括区与二分法
\item
  不动点与 Newton 法
\item
  割线法、收敛阶与混合策略
\end{enumerate}

\subsection{怎么读这一章}

第一次读，可以把三种方法看成三种不同的承诺。二分法始终保存``根还在这个区间里''的证据；Newton 法相信当前切线在附近足够像原函数；割线法则用两个旧函数值猜斜率。方法快不快，要和它可能失去什么保证一起看。
