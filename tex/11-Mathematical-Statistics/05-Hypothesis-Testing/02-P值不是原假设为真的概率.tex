\section{\texorpdfstring{\(p\) 值不是原假设为真的概率}{p 值不是原假设为真的概率}}

选定检验统计量 \(T(X)\)，且越大的 \(T\) 越不利于 \(H_0\)。观察到 \(t_{\mathrm{obs}}\) 后，简单原假设下的 \(p\) 值定义为

\[
p = \Pr_{H_0}
(T(X)\ge t_{\mathrm{obs}}).
\]
它问的是：

\begin{quote}
假设原假设成立，并按同样设计重新采样，得到至少同样极端统计量的概率是多少？
\end{quote}

数据的极端程度必须相对于预先定义的统计量和尾部方向。双侧检验不能总是机械把单侧 \(p\) 值乘二；非对称离散分布中需要明确``同样或更极端''的排序。

\subsection{复合原假设要控制最不利参数}

若

\[
H_0:\theta\in\Theta_0
\]

包含多个分布，一个合法构造常取

\[
p = \sup_{\theta\in\Theta_0}
\Pr_\theta
(T(X)\ge t_{\mathrm{obs}}).
\]
这样保证对所有原假设参数都控制第一类错误。用某个方便参数代替最不利值，可能使 \(p\) 值反保守。

连续简单原假设且统计量分布连续时，\(p\) 值在 \(H_0\) 下服从 Uniform\((0,1)\)（概率积分变换：连续变量代入自身分布函数即得均匀分布）；离散检验的 \(p\) 值通常更保守，不会覆盖全部连续数值。

\subsection{\texorpdfstring{\(p\) 值不回答哪些问题}{p 值不回答哪些问题}}

\(p=0.03\) 不意味着：

\begin{itemize}
\tightlist
\item
  \(H_0\) 为真的概率是 3%；
\item
  结果由随机机会造成的概率是 3%；
\item
  \(H_1\) 为真的概率是 97%；
\item
  复现实验得到显著结果的概率是 97%；
\item
  效应很大或具有实际意义；
\item
  数据、模型和研究设计没有偏差。
\end{itemize}

要给假设本身分配概率，需要先验与 Bayesian 模型；要讨论复现概率，需要指定真实效应、未来设计和分析流程；要判断重要性，需要效应尺度与决策损失。先验与似然比如何合成后验概率，下一节``似然比不是后验赔率''会具体展开。

\subsection{\texorpdfstring{一个 \(z\) 检验例子}{一个 z 检验例子}}

双侧标准正态检验观察到

\[
|z_{\mathrm{obs}}|=2.
\]

由于标准正态分布对称，此处双侧 \(p\) 值恰为单侧尾概率的两倍：

\[
p = 2\Pr(Z\ge2)
\approx0.0455.
\]
在预先固定的 level 0.05 检验中会拒绝，在 level 0.01 中不会。\(p\) 值可以理解为``使当前结果刚好进入拒绝域的最小常用显著性水平''，但它不是一个研究结论的连续可信度刻度。

\(p=0.049\) 与 \(p=0.051\) 的证据几乎相同，却会被二元标签切开。报告精确 \(p\) 值、效应和区间比只写``显著/不显著''更诚实。

\subsection{可选停止会破坏尾概率校准}

若研究者每收集一些样本就计算 \(p\) 值，一旦小于 0.05 就停止，最终``至少一次显著''的概率会超过 0.05。经典固定样本 \(p\) 值假设分析时点预先规定。

合法的 sequential test、alpha spending、always-valid \(p\) value 或 e-value 可以允许中途查看，但必须从设计阶段采用相应理论。不能事后把固定样本检验当成序贯检验。

同样，先尝试多种排除规则、协变量、终点和模型，再只报告最小 \(p\) 值，也相当于隐形多重比较，其计量与校正见``多重比较改变错误的计量单位''一节。

\subsection{\texorpdfstring{\(p\) 值与置信区间的联系}{p 值与置信区间的联系}}

双侧 level-\(\alpha\) 检验与 \(1-\alpha\) 置信区间通常互为反演：

\[
H_0:\theta=\theta_0
\text{ 被拒绝}
\quad\Longleftrightarrow\quad
\theta_0\notin C_{1-\alpha}(X).
\]

区间还能显示效应大小与精度，因此比一个阈值化 \(p\) 值提供更多信息。不过若区间方法与检验方法不同，例如一个用 Wald、另一个用 likelihood ratio，数值上可能不完全对应。

\subsubsection{留给读者}

解释以下说法哪里错：``\(p=0.2\)，所以原假设有 80% 概率正确。''然后给出两个可能产生 \(p=0.2\) 的完全不同情形：一个效应很小且估计精确，一个效应较大但样本很少。为什么只看 \(p\) 值无法区分？
