假设检验要在看数据前规定一条决策规则，并控制它在原假设成立时误报的概率；事后比较``哪个故事更像真的''没有这种错误率保证。

本单元按决策逻辑展开：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  两类错误与功效共同定义检验区分原假设、备择、显著性水平、第二类错误和功效，并解释样本量与效应量。
\item
  \(p\) 值不是原假设为真的概率从尾概率定义出发，逐一拆解常见误读和可选停止造成的失准。
\item
  似然比比较两个模型对数据的解释力从 Neyman--Pearson 引理走向广义似然比、Wilks 近似及其边界。
\item
  多重比较改变错误的计量单位区分 family-wise error 与 false discovery rate，并把筛选过程纳入推断。
\end{enumerate}

检验输出的是在特定模型、设计和错误控制下的决策证据。它不等于效应大小、实际重要性或因果结论。
