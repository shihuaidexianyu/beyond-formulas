\section{两类错误与功效共同定义检验}

假设检验把参数空间分成原假设与备择假设：

\[
H_0:\theta\in\Theta_0,
\qquad
H_1:\theta\in\Theta_1.
\]

数据进入后，检验规则决定``拒绝 \(H_0\)''或``不拒绝 \(H_0\)''。后一种说法刻意避免``接受 \(H_0\)''：没有足够证据拒绝，不等于证明原假设正确。

检验必须在看数据前指定统计量与拒绝域。否则选择最显著的方向、终点或子群会改变实际错误概率。

\subsection{第一类与第二类错误}

当 \(H_0\) 真实却拒绝它，发生第一类错误。检验的 size 定义为

\[
\sup_{\theta\in\Theta_0}
\Pr_\theta(\text{reject }H_0).
\]

若 size 不超过 \(\alpha\)，称为 level-\(\alpha\) 检验。

当某个备择 \(\theta\in\Theta_1\) 真实却不拒绝，发生第二类错误：

\[
\beta(\theta) = \Pr_\theta(\text{do not reject }H_0).
\]
功效函数为

\[
\pi(\theta)=1-\beta(\theta) = \Pr_\theta(\text{reject }H_0).
\]
第一类错误通常由一个统一上界控制；第二类错误和功效随具体备择参数变化。只说``检验功效是 80%''没有完整意义，必须说明针对多大的效应、何种方差和样本量。

\subsection{已知方差正态均值检验}

设

\[
X_i\sim N(\mu,\sigma^2),
\]

\(\sigma\) 已知。检验

\[
H_0:\mu=\mu_0
\quad\text{vs}\quad
H_1:\mu>\mu_0.
\]

统计量

\[
Z=
\frac{\bar X-\mu_0}{\sigma/\sqrt n}
\]

在 \(H_0\) 下服从 \(N(0,1)\)。level-\(\alpha\) 拒绝域为

\[
Z>z_{1-\alpha}.
\]

其中 \(z_{1-\alpha}\) 是标准正态的 \(1-\alpha\) 分位数，\(\Phi\) 为标准正态分布函数，即 \(\Phi(z_{1-\alpha})=1-\alpha\)。

若真实均值为 \(\mu_1>\mu_0\)，功效为

\[
\begin{aligned}
\pi(\mu_1)
&=
\Pr_{\mu_1}
\left(
\bar X>
\mu_0+z_{1-\alpha}\frac{\sigma}{\sqrt n}
\right)\\
&=
\Pr_{\mu_1}
\left(
\frac{\bar X-\mu_1}{\sigma/\sqrt n}>
z_{1-\alpha}-\frac{\sqrt n(\mu_1-\mu_0)}{\sigma}
\right)\\
&=
1-\Phi
\left(
z_{1-\alpha}
-\frac{\sqrt n(\mu_1-\mu_0)}{\sigma}
\right).
\end{aligned}
\]

功效随效应量 \(\mu_1-\mu_0\)、样本量 \(n\) 增大而增大，随噪声 \(\sigma\) 增大而下降。

\subsection{两类错误不能在固定信息下同时任意缩小}

把拒绝阈值提高，会减少第一类错误，却增加第二类错误；降低阈值则相反。更多样本能够让两种分布更容易区分，从而在保持 \(\alpha\) 时提高功效。

因此样本量设计需要预先指定：

\begin{itemize}
\tightlist
\item
  可接受的 \(\alpha\)；
\item
  目标功效 \(1-\beta\)；
\item
  最小有实际意义的效应；
\item
  噪声或基线率；
\item
  单侧还是双侧检验。
\end{itemize}

若在效应几乎为零处要求高功效，所需样本可能极大；这不是软件故障，而是两个生成机制本来就难以区分。

\subsection{单侧与双侧必须由问题决定}

若只有``提高''才有科学意义，可以预先选择单侧备择；同一 \(\alpha\) 集中在一侧，针对该方向功效更高。若下降也重要，应使用双侧检验。

不能先看估计方向，再决定使用哪一侧。这相当于把两个潜在检验中更有利的一个挑出来，实际第一类错误不再是宣称的 \(\alpha\)。

``反方向结果就不报告''也不是单侧检验的合理使用。方向应由研究问题和行动规则预先确定。

\subsection{统计显著不等于实际重要}

样本很大时，极小效应也可能得到很小 \(p\) 值；样本很小时，重要效应也可能因功效不足而未显著。应同时报告：

\begin{itemize}
\tightlist
\item
  效应估计；
\item
  置信区间；
\item
  预先定义的最小实际效应；
\item
  检验功效或可检测效应；
\item
  数据质量与模型假设。
\end{itemize}

``不显著''不能证明效应为零。若目标是证明差异小到可忽略，应使用 equivalence test 或 non-inferiority design，明确一个实际容忍界，而不是依赖普通检验未拒绝。
