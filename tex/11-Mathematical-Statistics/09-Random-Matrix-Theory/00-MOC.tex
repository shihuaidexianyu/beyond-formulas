样本协方差矩阵是统计推断的核心对象。低维时（样本量 \(n\) 远大于维数 \(p\)），大数定律保证它收敛到真实协方差；但当 \(p\) 与 \(n\) 同阶增长时，样本协方差的特征值会系统性偏离真实值——即使数据完全干净。随机矩阵理论给出这种偏差的精确形状。

本单元回答三个问题：

\begin{enumerate}
\tightlist
\item
  样本协方差在高维为何会失真：\(p/n\to c>0\) 时，样本特征值的分布不再集中于真实特征值附近，而是弥散成一个有宽度的谱。
\item
  Marchenko--Pastur 律给出谱的精确形状：独立同分布数据的样本协方差谱收敛到一个确定的分布，其支撑与形状由 \(c\) 决定。
\item
  谱偏差怎样影响统计推断：主成分分析、协方差收缩、特征值校正各自如何处理这种失真，以及``大样本渐近''与``高维渐近''的差别。
\end{enumerate}

这三个问题共用一条主线：高维统计的困难不只是``参数多''，而是随机矩阵的谱在高维极限下呈现出与低维直觉完全不同的形状。理解这种形状，是设计高维协方差估计与随机投影方法的前提。

\subsection{怎么读这一章}

这一章讨论的是维数与样本量同阶时，经典“大样本、固定维数”直觉为何失效。先观察样本协方差的谱怎样被噪声拉宽，再用 Marchenko--Pastur 律建立基准，最后看这些谱偏差怎样传到 PCA 与随机投影。
