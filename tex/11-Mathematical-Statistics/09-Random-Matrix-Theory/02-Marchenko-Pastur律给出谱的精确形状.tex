\section{Marchenko--Pastur 律给出谱的精确形状}

样本协方差谱在高维弥散成什么样？答案不是``大致集中在真实值附近''，而是一个完全确定的分布，形状只由比值 $c=p/n$ 决定。这就是 Marchenko--Pastur（MP）律，随机矩阵理论中最基本的结果之一。

\subsection{MP 律的陈述}

设 $X_1,\ldots,X_n\in\R^p$ 独立同分布，均值为零、协方差为 $I$，且四阶矩有限。样本协方差为

\[
S_n=\frac1n\sum_{i=1}^n X_iX_i^\trans.
\]

当 $p,n\to\infty$ 且 $p/n\to c\in(0,\infty)$ 时，$S_n$ 的经验谱分布（特征值的直方图）几乎必然收敛到 Marchenko--Pastur 分布 $\mu_c$：

\begin{itemize}
\tightlist
\item
  若 $c\le1$：$\mu_c$ 是 $[(1-\sqrt c)^2,(1+\sqrt c)^2]$ 上的连续分布，密度为

  \[
  f_c(x)=\frac{\sqrt{(b-x)(x-a)}}{2\pi cx},
  \qquad a=(1-\sqrt c)^2,\ b=(1+\sqrt c)^2.
  \]
\item
  若 $c>1$：$\mu_c$ 在 $0$ 处有质量 $1-1/c$（对应 $p-n$ 个零特征值），剩余质量 $1/c$ 分布在 $[a,b]$ 上。
\end{itemize}

这里只引用不证明。关键信息有三条：真实特征值全是 1，样本特征值却弥散到 $[a,b]$；弥散宽度 $b-a=4\sqrt c$ 随 $c$ 增大；$c>1$ 时必然有零特征值，因为秩不足。

$c=1$ 时支撑是 $[0,4]$，正是上一节反例的精确形状；$c=0.25$ 时支撑是 $[0.25,2.25]$，失真明显减轻但仍存在；$c\to0$ 时支撑收缩到 $\{1\}$，回到低维直觉。

\subsection{半圆律：对称随机矩阵的对应结果}

MP 律处理的是``样本协方差''型矩阵 $XX^\trans/n$。另一类基础结果是 Wigner 半圆律：设 $W$ 是 $p\times p$ 对称矩阵，对角线上方元素独立、均值为零、方差为 $1/p$，则当 $p\to\infty$ 时，$W$ 的经验谱分布收敛到半圆分布

\[
f(x)=\frac1{2\pi}\sqrt{4-x^2},
\qquad x\in[-2,2].
\]

半圆律与 MP 律共享同一个核心现象：随机矩阵的谱在高维不随机，而是收敛到确定形状。两者的差别在于矩阵结构（Wishart vs Wigner），但证明方法（矩方法、Stieltjes 变换、自由概率）是相通的。

半圆律的边界行为更干净：最大特征值收敛到 2，最小到 $-2$，没有厚尾。MP 律的边界 $(1\pm\sqrt c)^2$ 也有类似集中性，但证明更难。

\subsection{MP 律告诉我们什么}

MP 律不是``样本协方差很差''的坏消息，而是``失真可以精确计算''的好消息。它给出：

\begin{itemize}
\tightlist
\item
  \textbf{谱的支撑}：真实特征值 1 被样本谱``涂抹''到 $[a,b]$，超出这个范围的样本特征值几乎不存在；
\item
  \textbf{谱的形状}：密度在 $a$ 与 $b$ 附近呈平方根边缘（平方根消失），中间较平缓——直方图的这个形状是高维协方差的指纹；
\item
  \textbf{依赖参数}：失真只由 $c$ 决定，与数据的具体分布（在矩条件下）无关——这叫普适性（universality）。
\end{itemize}

普适性是随机矩阵理论最深刻的特征之一：Gaussian、Bernoulli、均匀分布的数据，只要满足独立同分布与矩条件，谱极限相同。这解释了为什么 MP 律能从无线通信（$X$ 是天线信号）直接搬到金融（$X$ 是股票收益）——形状与具体领域无关。

\subsection{从 MP 律到校正方法}

知道失真的精确形状，就可以设计校正。三条常用路径：

\begin{itemize}
\tightlist
\item
  \textbf{特征值收缩（shrinkage）}：把样本特征值 $\widehat\lambda_i$ 向中心收缩，例如 $\widetilde\lambda_i=\alpha\widehat\lambda_i+(1-\alpha)\bar\lambda$，其中 $\bar\lambda$ 是平均值。Ledoit--Wolf 估计给出最优 $\alpha$ 的渐近公式，它利用 MP 律预测失真。
\item
  \textbf{谱校正（spectrum denoising）}：由 MP 律反推真实谱，把样本谱``去卷积''回真实谱。这在 $\Sigma=I$ 时最直接，一般 $\Sigma$ 时更复杂但原理相同。
\item
  \textbf{截断奇异值分解（truncated SVD）}：只保留最大的 $k$ 个特征方向，承认其余方向被噪声主导。MP 律告诉我们哪些特征值还在信号范围内、哪些已经是噪声。
\end{itemize}

这些方法的共同点是把 MP 律当作``噪声模型''，而不是当作障碍。理解失真的形状，反而让校正更精确。

\subsection{MP 律的边界}

MP 律的假设是独立同分布、均值为零、协方差为 $I$、四阶矩有限。偏离这些假设时：

\begin{itemize}
\tightlist
\item
  \textbf{$\Sigma\ne I$}：谱极限由 $\Sigma$ 的谱与 MP 律的自由卷积给出，形状更复杂但仍确定；
\item
  \textbf{重尾}：四阶矩无限时，最大特征值可能不收敛到 $(1+\sqrt c)^2$，而是有厚尾；
\item
  \textbf{相关数据}：独立性失败时，MP 律不再适用，需要针对特定依赖结构的随机矩阵模型。
\end{itemize}

MP 律是高维统计的基线，不是终点。它回答的是``最干净的情况下失真长什么样''；实际数据的失真可能更严重或更复杂，但 MP 律提供了比较的基准。

\subsection{落点}

样本协方差在高维的失真不是模糊的``大样本不够大''，而是一个形状确定、只依赖 $c$ 的谱弥散。MP 律把这个形状写成公式，使校正方法有据可依。下一节看这种失真怎样影响两个最常见的下游任务：PCA 与随机投影。
