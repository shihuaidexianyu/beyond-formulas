\section{Conformal prediction 用可交换性换有限样本覆盖}

普通回归模型给出点预测 $\widehat f(x)$，却不说明新结果可能偏离多远。参数置信区间针对未知参数，不能直接当作新观测的预测区间；Bayesian posterior predictive interval 又依赖 prior 与 likelihood。Conformal prediction 采取另一条路线：不要求模型正确，用样本之间的可交换性校准预测集合的有限样本覆盖。

\subsection{Split conformal 把拟合与校准分开}

把数据分成 training set 和含 $m$ 个样本的 calibration set。先只用 training data 拟合任意预测器 $\widehat f$，再在 calibration data 上计算 nonconformity scores

\[
R_i=\abs{Y_i-\widehat f(X_i)},
\qquad i=1,\ldots,m.
\]

令排序后分数为

\[
R_{(1)}\le\cdots\le R_{(m)},
\]

并取

\[
k=\left\lceil(m+1)(1-\alpha)\right\rceil.
\]

当 $k\le m$ 时令 $q=R_{(k)}$；若 $k=m+1$，可把 $q$ 视为无穷大。对新输入 $x$ 输出区间

\[
C(x)
=
\left[
\widehat f(x)-q,
\widehat f(x)+q
\right].
\]

这里的有限样本修正是 $m+1$，不是随手取 calibration scores 的经验 $1-\alpha$ 分位数。它把新样本也计入可能的秩位置。

\subsection{覆盖保证来自秩的对称性}

条件于 training data 后，若 $m$ 个 calibration examples 与新样本可交换，则它们的 $m+1$ 个 scores 也可交换。忽略 ties 时，新 score 的秩在 $1,ldots,m+1$ 上均匀；有 ties 时采用相应保守处理仍得到

\[
\Prob\bigl(Y_{\mathrm{new}}\in C(X_{\mathrm{new}})\bigr)
\ge
1-\alpha.
\]

保证对任意样本量成立，不依赖线性、Gaussian 噪声或 $\widehat f$ 一致。模型越好，residual scores 通常越小，区间越窄；模型很差时 coverage 仍可能成立，只是集合宽到没有用途。Conformal 校准覆盖，不制造预测信息。

\subsection{分类输出集合而不是强迫单标签}

若分类器给出类别概率 $\widehat p_y(x)$，可用

\[
R_i=1-\widehat p_{Y_i}(X_i)
\]

作为 score。以 calibration quantile $q$ 构造

\[
C(x)=
\set{y:1-\widehat p_y(x)\le q}.
\]

容易样本可能只保留一个类别，模糊样本可能保留多个，极端情况下也可能为空或包含全部类别，具体取决于 score 与构造方式。集合大小反映当前程序为维持覆盖所付出的信息代价，不应只汇报平均 coverage 而隐藏集合是否几乎总是过大。

更精细的 adaptive scores、class-conditional calibration 或 normalized residuals 可以让集合适应异方差与类别不平衡，但每种修改都要重新说明交换性单位和保证对象。

\subsection{边际覆盖不是每个子群的条件覆盖}

上式平均于新样本的整体分布。它不推出

\[
\Prob\bigl(Y\in C(X)\mid X=x\bigr)
\ge1-\alpha
\]

对每个 $x$ 成立，也不保证每个年龄、地区或风险组都达到名义覆盖。无分布假设下要求所有点都具有非平凡的精确条件覆盖，一般不可实现；局部适应需要平滑性、分组样本量或模型结构等额外条件。

总体 coverage 可以由容易的大群体补偿困难小群体。因而应同时检查有实际意义的预设子群、集合宽度和 coverage 的不确定性，但不能看完结果后无限搜索最差子群而忽略多重选择。

\subsection{分布漂移与数据复用会破坏秩论证}

若 calibration data 来自旧医院而新样本来自新设备，scores 不再可交换，名义 coverage 可能失效。某些 covariate shift 可用加权 conformal 处理，但需要可靠密度比与支持覆盖；concept drift 不能由普通权重自动修复。

同一 calibration set 若被反复用于选择模型、score、超参数和最终 quantile，新样本 rank 不再与一个预先固定程序直接对称。应把模型拟合、方法选择和最终校准的数据角色分开，或使用能重现完整选择流程的有效方案。

Conformal 的核心价值不是“无需假设”，而是把主要假设集中到可交换性，并给出一个清楚、有限样本、可被分布变化击穿的覆盖承诺。
