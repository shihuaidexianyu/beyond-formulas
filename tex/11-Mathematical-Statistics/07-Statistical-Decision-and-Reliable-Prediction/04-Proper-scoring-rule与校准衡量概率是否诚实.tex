\section{Proper scoring rule 与校准衡量概率是否诚实}
\label{sec:11-Mathematical-Statistics/07-Statistical-Decision-and-Reliable-Prediction/04}

\begin{quote}
\textbf{先抓住这一点：}proper scoring rule 让如实报告自己的概率分布在期望上最划算。校准只检查报出七成概率的事件是否约有七成发生，还不能单独保证预测足够尖锐或有区分力。
\end{quote}

分类器输出 \(0.8\) 时，它可能只是一个排序分数，也可能声称事件发生概率为 \(80\%\)。若要把输出用于代价敏感决策，就需要评价整项概率声明，而不只是最终类别是否正确。

设 \(Y\in\{0,1\}\)，真实条件概率为

\[
p(x)=P(Y=1\mid X=x),
\]

模型报告 \(q(x)\in[0,1]\)。Scoring rule 是对概率报告与实际结果给出损失的函数 \(S(q,Y)\)。

\subsection{Brier score 的最优报告是真实概率}

Brier loss 为

\[
S_{\mathrm{Brier}}(q,Y)=(Y-q)^2.
\]

给定 \(X=x\) 后，条件风险可分解为

\[
\E[(Y-q)^2\mid X=x]
=
p(x)(1-p(x))
+\bigl(p(x)-q\bigr)^2.
\]

第一项是给定信息后仍不可消除的 Bernoulli 波动，第二项是错误概率报告的代价。因此唯一最优报告是 \(q=p(x)\)。这种让诚实概率在期望上最优的规则称为 strictly proper scoring rule。

Log loss

\[
S_{\log}(q,Y)
=
-Y\log q-(1-Y)\log(1-q)
\]

也严格 proper，其 excess risk 是真实 Bernoulli 分布到报告分布的 KL divergence。它对接近 \(0\) 或 \(1\) 的错误自信惩罚无界；Brier score 有界，对极端错误更温和。两者都鼓励诚实，却对尾部置信度采用不同尺度。

\subsection{校准只检查同一预测值上的长期频率}

令随机预测为 \(Q=q(X)\)。完美 calibration 要求

\[
\E[Y\mid Q]=Q
\]

几乎处处成立。也就是说，在所有预测约为 \(0.7\) 的对象中，事件长期约有 \(70\%\) 发生。

例如急诊模型把一组患者的二十四小时恶化概率报告为 \(0.2\)，校准要求的是：在足够多次、采用同一预测时点和结果定义的类似部署中，这组患者约有五分之一恶化。它不要求某一天恰好五位患者中出现一位，也不能只在当前医院总体上检查后便推断老年患者、夜班患者或新设备采集的患者同样校准。样本分组方式改变了条件事件，校准声明的对象也随之改变。

若模型输出还会触发提前治疗，随后观察到的恶化率已经受行动影响。此时必须说明校准针对``当前分诊政策下的结果''，还是试图预测``不采取额外干预时的自然风险''；二者的数据生成机制不同。一个概率数字只有连同预测时点、目标人群和后续政策一起声明，才知道所谓 \(20\%\) 在现实中指什么。

校准不要求 \(Q=p(X)\)。始终输出总体发生率 \(P(Y=1)\) 的常数模型是校准的，却没有区分任何对象。一个有用概率模型还需要 resolution：不同对象得到不同风险，并且这些分组的实际发生率确实不同。

反过来，排序极好的模型可能严重失准。对所有 logits 乘一个很大的正数通常保持排序和 AUC，却让概率过度接近 \(0\) 或 \(1\)。Discrimination、calibration 和 sharpness 是不同性质，不能由一个指标相互推出。

\subsection{总体评分同时受到不可约噪声和信息质量影响}

Proper score 的总体平均值可以比较完整概率预测，但一个更低的 Brier 或 log loss 可能来自更好校准、更强区分能力，或测试总体本身更容易。跨数据集直接比较分数时，基准率和条件噪声也参与结果。

可靠报告可以同时给出 proper score、calibration curve、按风险分组的样本量，以及与简单基线的比较。只报告 expected calibration error 会受到分箱边界、样本量和权重方式影响；样本内重校准后再在同一数据上评价也会乐观。

\subsection{重校准改变概率映射，不创造缺失信息}

Platt scaling、isotonic regression 或 temperature scaling 使用独立 calibration data 学习从原分数到概率的映射。它们能修复某些系统性过度自信或低估，却不能恢复模型从未捕捉的区分信息，也不能保证分布变化后的校准。

若训练时使用 class weighting、过采样或 label smoothing，最优输出可能不再等于自然部署人群中的条件概率。此时应先说清训练损失对应哪个目标分布，再在真实目标分布上单独校准和验证。

\subsection{概率诚实仍不等于行动合理}

一个完美校准的概率只是决策输入。行动还需要错误代价、容量和干预后果；预测 \(Y\) 的条件概率也不一定等于采取行动后 \(Y\) 的概率。下一节把概率输出扩展为具有覆盖保证的集合，最后一节再把两类输出接到实际行动。
