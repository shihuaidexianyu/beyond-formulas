\section{置换检验依赖可交换性}

一个两组比较实验记录了响应

\[
Y_1,\ldots,Y_n
\]

和处理标签 \(Z_i\in\{0,1\}\)。若原假设说处理没有作用，并且实验随机分配标签，那么在给定全部响应的条件下，哪些单位被标为处理组应当由随机化机制决定。置换标签就能重现原假设下的检验统计量分布。

\subsection{两样本均值差}

选择统计量

\[
T(Y,Z) = \bar Y_{\mathrm{treated}}
-\bar Y_{\mathrm{control}}.
\]
在所有保持处理组大小不变的标签排列中计算 \(T^\pi\)。双侧置换 \(p\) 值可以按

\[
p = \frac{
\#\set{\pi:|T^\pi|\ge|T_{\mathrm{obs}}|}
}{
\#\set{\text{all valid permutations}}
}
\]
计算。若枚举全部随机分配，这在 sharp null（每个单位的处理效应都为零，见本节稍后）和已知随机化机制下是精确检验，不依赖正态近似。

排列太多时随机抽取 \(B\) 个。常用 Monte Carlo 修正为

\[
\hat p = \frac{
1+\#\set{b:|T^{\pi_b}|\ge|T_{\mathrm{obs}}|}
}{
B+1
}.
\]
分子分母各加一，把原始排列也纳入，避免报告不可能被有限模拟支持的零 \(p\) 值，并保持适当校准。

\subsection{可交换性为随机打乱提供依据}

置换合法的原因是：在 \(H_0\) 下，允许的排列不会改变数据联合分布，或者恰好对应实验随机化分配。若两组方差不同、时间存在趋势、单位成簇或标签分配概率不同，任意打乱所有行可能破坏联合结构。

可交换性必须与设计相匹配：

\begin{itemize}
\tightlist
\item
  配对实验应在每对内部交换标签或做符号翻转；
\item
  分层随机化应在层内置换；
\item
  cluster-randomized trial 应以簇为单位置换；
\item
  时间序列不能任意打乱时序；
\item
  unequal probability design 需要按实际随机化机制生成排列。
\end{itemize}

``非参数''不等于``无假设''。置换检验把分布形状假设换成了对称性或随机化假设。

\subsection{Sharp null 与平均效应原假设不同}

随机化检验最直接检验 sharp null：每个单位的处理效应都为零。它允许为每种未观察分配推断完整潜在结果（每个单位在处理与对照下各自的响应）。

``平均处理效应为零''允许个体效应正负抵消，不是同一个原假设；简单置换分布未必精确。可以使用学生化统计量和渐近理论改善，但必须说明检验对象。

这里的潜在结果只是为区分两类原假设服务。若要系统理解平均处理效应怎样定义、随机分配为何能够识别它，以及观察研究还需要哪些额外假设，可继续读``因果推断：预测世界不等于改变世界''。

类似地，两样本标签置换精确检验的是两组完整分布相同；若只想检验均值相等而允许方差不同，未学生化均值差置换可能失准。

\subsection{Bootstrap 与 permutation 回答不同问题}

Bootstrap 从经验分布有放回抽样，近似估计量在未知总体下的抽样波动；permutation 在原假设对称性下重新分配标签，构造检验统计量的 null distribution。

Bootstrap 样本会重复观测，permutation 通常只重排现有标签；bootstrap 用于标准误和区间，permutation 主要用于检验。虽然两者都可用于更广泛任务，但不能仅替换一行抽样代码就保留解释。

\subsection{统计量选择仍然重要}

置换检验能精确控制第一类错误，不保证对所有备择都有高功效。均值差擅长位置变化，对尾部或分布形状变化可能不敏感；rank、energy distance、maximum mean discrepancy 等统计量针对不同备择。

统计量若在看数据后从许多候选中挑选，置换过程必须重现整个选择步骤，或使用独立数据。只在最终选中统计量上置换会忽略选择多重性。
