从一份样本输出一个参数值很容易；困难在于选择什么规则，以及怎样比较两个规则。点估计必须同时回答``怎样构造''和``用什么损失评价''。

本单元先建立评价语言，再比较三条构造路径：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  偏差、方差与均方误差区分有限样本精度和渐近一致性，并用收缩例子展示偏差---方差权衡。
\item
  矩估计让样本特征匹配总体特征把经验矩与理论矩对齐，说明方法简单也意味着可能忽略分布信息。
\item
  最大似然选择最能解释数据的参数从 likelihood、score 与曲率推导 Bernoulli 和正态例子，并讨论边界、错设与数值优化。
\item
  MAP 把先验与似然合在一个点上连接 Bayesian 后验、正则化与损失函数，同时解释 MAP 的参数化依赖。
\end{enumerate}

没有脱离任务的``最佳估计量''。平方误差、绝对误差、预测损失与决策代价会偏好不同的点摘要。
