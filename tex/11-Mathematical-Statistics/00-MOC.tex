\section{数理统计：从有限样本推断未知机制}

概率论通常假设分布已知，然后问随机数据会怎样；数理统计面对相反方向：

\[
\text{未知分布或参数}
\longrightarrow
\text{一次有限样本}
\]

已经发生，我们想从右侧的数据反推左侧的机制。

例如 100 件产品中有 92 件合格。样本比例 \(0.92\) 很容易计算，但真正的统计问题是：

\begin{itemize}
\tightlist
\item
  这 100 件怎样被抽到，能代表哪一个总体？
\item
  若重新抽样，\(0.92\) 会波动多少？
\item
  用样本比例、加平滑的比例还是某个模型估计更好？
\item
  区间怎样表达不确定性，概率究竟作用在参数还是抽样程序上？
\item
  若要判断合格率是否超过标准，怎样控制误报与漏报？
\item
  若先试了许多子群和模型，最终结论是否仍保持原来的错误率？
\end{itemize}

数理统计不是在数据上套一个公式，而是为``从样本到结论''的整个推理过程建立可检验的概率保证。

\subsection{推断成立需要四层共同支撑}

第一层是\textbf{数据生成与抽样设计}。iid、随机分配、缺失机制、测量过程和目标总体决定样本能代表什么。增加样本量可以减少随机波动，却不会自动消除选择偏差和系统测量误差。

第二层是\textbf{统计模型与目标量}。参数、分布函数、预测风险或因果效应是不同目标；模型必须让目标可识别。一个优化器能精确找到最大似然点，并不说明数据能够唯一识别参数。

第三层是\textbf{推断程序的重复抽样或后验性质}。偏差、方差、覆盖率、第一类错误和功效都是规则在假设机制下的长期表现；Bayesian 后验则在给定先验与数据后为参数赋予概率。两套概率对象必须明确区分。

第四层是\textbf{数据分析流程本身}。调参、选择终点、预处理、可选停止和多重比较都会使用随机数据；若推断只分析最终公式而忽略选择过程，名义保证会失效。

\subsection{先理解数据为何是随机的}

从一次数据到重复抽样从数据生成机制开始。总体被理解为目标分布，样本在观测前是随机变量，在观测后才成为具体数值。参数固定未知、统计量随机、估计值是这次实现；这三层分清后，抽样分布和标准误才不会被误解。

本单元还刻意讨论 iid 假设的两个部分、选择偏差、模型错设与可识别性。统计结论永远相对于某个观测机制；数据表本身无法证明它来自简单随机样本。

\subsection{点估计既要构造，也要评价}

点估计先用偏差、方差、MSE、风险与一致性建立评价语言，再比较矩估计、最大似然和 MAP。

矩估计让样本矩匹配总体矩，计算直接但可能丢弃分布信息；MLE 选择最能解释已观测数据的参数，在正则条件下具有良好渐近性质，却可能位于边界、无界或因模型错设收敛到伪真参数；MAP 把先验与似然合为一个后验众数，连接 \(L_2/L_1\) 正则，但不具有一般参数化不变性。

没有脱离损失的最佳点估计。后验均值在平方损失下自然，后验中位数对应绝对误差，预测任务又可能偏好完全不同的规则。

\subsection{数据压缩与精度极限}

数据压缩与可估计信息问两个更深的问题。

充分统计量说明何时可以压缩样本而不损失关于参数的模型信息；因子分解定理提供判据，Rao--Blackwell 化则把充分性变成降低估计方差的实际方法。

Fisher 信息使用 score 的方差或 log-likelihood 的平均负曲率，衡量相邻参数分布有多容易区分。Cramér--Rao 界进一步说明，在正则无偏估计类中，信息量为方差设置了下限。它不是所有估计器的普遍禁令：带偏收缩、边界模型和非正则问题需要不同分析。

\subsection{从一个点扩展到不确定性集合}

区间估计首先纠正一个关键误读：频率学派 95\% 置信度属于随机区间生成程序的长期覆盖率，不是观测后固定参数落入这次区间的概率。

枢轴量通过构造分布不含未知参数的统计函数，给出正态均值的精确 \(z\) 与 \(t\) 区间；没有精确枢轴时，Wald、delta method、profile likelihood 和 bootstrap 提供渐近或重采样近似。

Bayesian 可信区间则在给定数据与先验后，确实为参数集合赋予后验概率。两类区间可以数值接近，却回答不同概率问题；大样本一致并不能成为小样本混用解释的理由。

\subsection{检验是一条预先设计的决策规则}

假设检验从两类错误开始。显著性水平控制原假设成立时的误拒绝，功效描述具体备择下发现效应的概率；两者与样本量、噪声和效应大小共同决定设计。

\(p\) 值是原假设下``至少同样极端数据''的尾概率，不是原假设为真的概率。Neyman--Pearson 引理说明简单假设间的似然比检验为何在固定第一类错误下最有力；广义似然比与 Wilks 定理把思路扩展到复合模型，但边界、混合模型和弱识别会破坏普通卡方近似。

多重比较单元把错误计量从单个假设扩展到整个假设家族。Bonferroni/Holm 控制任何假阳性的概率，Benjamini--Hochberg 在相应依赖条件下控制平均假发现比例。亚组、模型规格、可选停止和只报告成功实验都属于多重选择。

\subsection{回归与方差分析把变异拆成可检验结构}

经典线性回归把最小二乘接到抽样分布：在设计矩阵满秩、误差均值为零且方差结构正确时，系数的标准误、(t) 检验和整体 (F) 检验都从同一 Gaussian 模型推出。若只保留条件均值正确而允许异方差，点估计与普通标准误的命运不同，必须分别判断。

ANOVA 则把总平方和拆成组间与组内变异，用一个 (F) 比较``组均值差异''和``同组自然波动''。它不是许多两两 (t) 检验的简称，也不说明究竟哪几组不同；多重比较、交互作用、平衡设计和随机化假设仍需单独处理。这一单元把前面的估计、检验和平方和几何合成一个可完整推导的经典样板，也为后续广义线性模型提供参照。

\subsection{重采样必须模拟正确的随机机制}

重采样与模型选择区分三种表面相似、目的不同的方法：

\begin{itemize}
\tightlist
\item
  Bootstrap 从经验分布有放回抽样，近似估计量的抽样波动；
\item
  permutation 在原假设的可交换性或随机化设计下重排标签，构造 null distribution；
\item
  cross-validation 让每个评价样本在相应训练过程中保持未见，估计完整 pipeline 的泛化风险。
\end{itemize}

预处理、特征选择和调参都必须在训练折内部完成；调参后的无偏评估需要 nested CV 或真正独立测试集。

最后的偏差---方差、正则化、AIC 与 BIC 章节说明``模型选择''没有单一目标。AIC 更接近正则条件下的 out-of-sample KL 风险，BIC 在另一组条件下近似 marginal likelihood 并追求模型识别，CV 则直接模拟指定预测任务。选择标准必须服务于问题，而不是挑一个数值最小的缩写。

\subsection{从推断结果走到行动}

点估计、区间、检验以及回归模型仍不是现实流程的终点。``统计决策与可信预测：模型给出概率之后怎样行动''把状态、行动、损失与风险写成统一问题，再用 Bayes rule、minimax 与可容许性解释不同最优准则。它还把概率校准、proper scoring rule、conformal prediction 和拒绝预测分开：概率诚实、集合覆盖、区分能力和行动价值是四种不同证据。

这一步也明确连接后续内容。机器学习负责从特征产生预测，因果推断判断行动会怎样改变结果，控制与强化学习再处理行动改变未来数据的序贯反馈。数理统计提供的不只是误差条，而是比较整条决策程序的语言。

\subsection{一套可复用的推断顺序}

面对新数据，先写清目标总体、抽样单位、观测机制和目标量。然后说明模型族、参数空间、独立性与可识别假设；只有这一步完成后，才选择估计量与损失。

构造结果时应同时说明有限样本性质与渐近性质，给出标准误或区间，并检查边界、偏斜和模型错设。若做检验，要预先定义原假设、备择、统计量、显著性水平和目标功效；若有多重选择，把完整 family 和停止规则纳入校准。

模型与超参数选择应嵌在合法的数据切分中。最终报告不只包含点估计与 \(p\) 值，还应包括效应尺度、区间、假设、缺失与选择过程、稳健性分析以及结论不能推广到哪里。

\subsection{前置、后续与参考主线}

本部分依赖 概率论 的随机变量、条件分布、大数定律和中心极限定理，也使用 线性代数 处理协方差、回归和信息矩阵。矩阵微积分 支撑 score 与 Hessian，凸优化 解释似然和正则化怎样被可靠求解。

后续 机器学习 会把这些推断原则用于线性模型、降维、概率模型和学习理论；深度学习再把它们用于高维复合模型。模型最优性、数值精度和统计不确定性会在那里同时出现，不能互相替代。

讲授主线参考 \emph{All of Statistics}、Stanford STATS 200、UC Berkeley EECS 126 与 MacKay 的 Bayesian 推断章节；具体有限样本例子可结合 Penn State STAT 415 阅读。各章以生成机制、推断对象和错误控制为中心重新组织，而不是按术语表逐项扩写。
