\section{偏序与可比较性}

\subsection{``有先后''不代表任何两个对象都能比较}

日常说``大小关系''时，我们容易默认任意两个对象总能分出先后。但集合包含、任务依赖、整数整除都不是这样：\(\{1\}\) 与 \(\{2\}\) 谁也不包含谁；任务 A 与任务 B 可能互不依赖；2 不整除 3，3 也不整除 2。

集合 \(P\) 上的关系 \(\preceq\) 若满足：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  自反：\(x\preceq x\)；
\item
  反对称：\(x\preceq y\) 且 \(y\preceq x\) 时，\(x=y\)；
\item
  传递：\(x\preceq y\) 且 \(y\preceq z\) 时，\(x\preceq z\)，
\end{enumerate}

就称为偏序，\((P,\preceq)\) 称为偏序集。

反对称不等于``不对称''。它允许 \(x\preceq x\)；只是禁止两个不同对象同时排在彼此前面。等价关系使用对称性把对象聚成类，偏序使用反对称性把对象排出结构。

\subsection{两个典型例子}

在 \(\mathcal P(S)\) 上，包含关系 \(\subseteq\) 是偏序。若 \(A\subseteq B\) 且 \(B\subseteq A\)，外延性给出 \(A=B\)。但一般 \(A,B\) 不必可比较。

在正整数上，整除关系 \(a\mid b\) 也是偏序：\(a\mid a\)；若 \(a\mid b\) 且 \(b\mid a\)，正性保证 \(a=b\)；传递性来自乘法。若把论域改成所有整数，反对称性会失败，因为 \(1\mid-1\) 且 \(-1\mid1\)，但 \(1\ne-1\)。这说明偏序性质依赖论域，而不只依赖符号。

若偏序中每两个元素都可比较，即总有 \(x\preceq y\) 或 \(y\preceq x\)，则称为全序或线性序。实数上的 \(\le\) 是全序，集合包含通常只是偏序。

\subsection{最大、极大、上界不是同一个概念}

设 \(A\) 是偏序集中的一个子集。

\begin{itemize}
\tightlist
\item
  最大元 \(m\in A\) 满足对每个 \(a\in A\) 都有 \(a\preceq m\)；若存在则唯一——两个最大元 \(m_1,m_2\) 会同时满足 \(m_1\preceq m_2\) 与 \(m_2\preceq m_1\)，由反对称性二者相等。
\item
  极大元 \(m\in A\) 表示 \(A\) 中没有严格比它更大的元素；可能有多个互不可比的极大元。
\item
  上界 \(u\) 不必属于 \(A\)，只需所有 \(a\in A\) 都满足 \(a\preceq u\)。
\item
  最小上界是所有上界中最小的一个，也称上确界。
\end{itemize}

在正整数按整除排序的子集 \(A=\{2,3,6\}\) 中，6 是最大元；2 和 3 都不是最大元，因为它们各自不在所有元素之上。若改成 \(A=\{2,3\}\)，2 与 3 都是极大元，却没有最大元。数字大小在这里无关，顺序由``整除''决定。

对集合族按包含排序，并集常扮演最小上界，交集常扮演最大下界。实分析中的上确界、算法中的依赖图和优化中的可行集包含，都在使用同一套偏序语言。

\subsection{Hasse 图只画必要的比较}

有限偏序可以用 Hasse 图表示：记 \(x\prec y\) 为``\(x\preceq y\) 且 \(x\ne y\)''的严格情形；若 \(x\prec y\) 且中间没有 \(z\) 满足 \(x\prec z\prec y\)，就把 \(y\) 画在 \(x\) 上方并连接。自反边和由传递性可推出的长边省略。

Hasse 图不是一般有向图的随意简写。读图时，沿向上路径表示偏序关系；两个点若没有上下路径，便不可比较。它使``多个极大元但没有最大元''一眼可见。
