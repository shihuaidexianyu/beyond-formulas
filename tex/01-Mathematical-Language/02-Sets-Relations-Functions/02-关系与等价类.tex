\section{关系与等价类}

\subsection{关系只是被选中的有序对}

从集合 \(A\) 到集合 \(B\) 的二元关系 \(R\) 是 \(A\times B\) 的一个子集。若 \((a,b)\in R\)，常写成 \(aRb\)。

例如在学生集合 \(S\) 与课程集合 \(C\) 上，``学生 \(s\) 选了课程 \(c\)''构成 \(S\times C\) 的子集；在整数上，``\(a\) 整除 \(b\)''构成 \(\Z\times\Z\) 的子集。关系可以画成箭头、列成表格或写成条件，但这些只是同一个有序对集合的不同表示。

当关系从 \(A\) 到自身时，可以研究三种重要性质：

\begin{itemize}
\tightlist
\item
  自反：对每个 \(a\in A\)，有 \(aRa\)；
\item
  对称：若 \(aRb\)，则 \(bRa\)；
\item
  传递：若 \(aRb\) 且 \(bRc\)，则 \(aRc\)。
\end{itemize}

这些性质彼此独立。``认识''常近似对称却未必传递；``小于''传递却不自反也不对称；``与\ldots\ldots 相等''三者都满足。

\subsection{等价关系是在选择一种``忽略差异''的方式}

同时满足自反、对称和传递的关系称为等价关系。它未必表示对象完全相等，而是表示在当前问题关心的标准下可以视为同类。

固定正整数 \(m\)，在整数上定义

\[
a\sim b
\quad\Longleftrightarrow\quad
m\mid(a-b).
\]

这表示 \(a,b\) 除以 \(m\) 有相同余数；其中 \(m\mid(a-b)\) 读作``\(m\) 整除 \(a-b\)''，即 \(a-b\) 是 \(m\) 的倍数，整除关系在下一节还会作为偏序再次出现。上面定义的 \(\sim\) 是等价关系：\(m\mid0\) 给出自反性；若 \(m\mid(a-b)\)，也有 \(m\mid(b-a)\)；若 \(m\) 同时整除 \(a-b\) 与 \(b-c\)，便整除两者之和 \(a-c\)。

对任意 \(a\)，其等价类是

\[
[a]=\{x\in A:x\sim a\}.
\]

模 3 时，整数被分成

\[
[0],\quad[1],\quad[2],
\]

例如 \([1]=\{\ldots,-5,-2,1,4,7,\ldots\}\)。\([1]\) 与 \([4]\) 不是两个重叠的类，而是同一个类。

\subsection{为什么等价类只能相等或不交}

假设 \([a]\) 与 \([b]\) 有共同元素 \(x\)。那么 \(x\sim a\) 且 \(x\sim b\)。由对称性 \(a\sim x\)，再由传递性得 \(a\sim b\)。于是任何与 \(a\) 等价的元素也与 \(b\) 等价，所以 \([a]\subseteq[b]\)；反向同理，故两类相等。

因此等价类满足：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  每个元素至少属于自己的等价类；
\item
  两个等价类要么完全相同，要么互不相交；
\item
  所有等价类的并是原集合。
\end{enumerate}

这正是集合的一个划分。反过来，任何划分也会定义等价关系：两个元素等价，当且仅当它们位于同一块。于是``等价关系''与``把集合分组''是同一结构的两种语言。

所有等价类组成的集合记作商集 \(A/{\sim}\)。商集的元素不是原来的点，而是整类对象。例如有理数可以从整数对 \((p,q)\)（\(q\ne0\)）出发构造：规定 \((p,q)\sim(r,s)\) 当且仅当 \(ps=qr\)，把表示同一比值的对归入同一等价类——判定用乘法而不用除法，正是为了不预先假设``比值''已经存在。构造新对象时，等价关系负责说明哪些不同表示应当视为同一个对象。

\subsection{传递性是最常被漏掉的条件}

在实数上定义 \(xRy\) 当 \(|x-y|\le1\)。它自反、对称，却不传递：\(0R0.8\) 且 \(0.8R1.6\)，但 \(0\) 与 \(1.6\) 不相关。因此``距离足够近''通常不能直接形成等价类，因为一连串局部接近可能跨越很远。

类似地，``拥有同一位朋友''也不传递。看到``相似''\,``接近''\,``相关''一类词时，不能凭语感称其为等价关系，必须逐项检查三条性质。
