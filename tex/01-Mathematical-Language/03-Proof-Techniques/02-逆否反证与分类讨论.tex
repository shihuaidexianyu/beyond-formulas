\section{逆否、反证与分类讨论}

\subsection{直接路线受阻时，先改写逻辑而不是乱试技巧}

有些蕴含 \(P\to Q\) 从 \(P\) 出发很难触碰 \(Q\)，而 \(Q\) 的否定却会立刻暴露结构。因为

\[
P\to Q\equiv\neg Q\to\neg P,
\]

我们可以证明逆否命题。

例如证明``若整数 \(n^2\) 为偶数，则 \(n\) 为偶数''。直接从 \(n^2=2k\) 很难把平方根整理成整数倍；逆否方向是``若 \(n\) 为奇数，则 \(n^2\) 为奇数''。写 \(n=2m+1\)，便有

\[
n^2=4m^2+4m+1=2(2m^2+2m)+1,
\]

所以结论成立。

逆否证明没有改变命题内容，只换了更容易使用定义的入口。不要把它与证明逆命题 \(Q\to P\) 混淆。

\subsection{反证法：让否定假设制造不可能}

要证明命题 \(P\)，反证法先假设 \(\neg P\)，再推出矛盾。矛盾可以是 \(R\land\neg R\)，也可以违反已知定理、最简性或对象定义。由于 \(\neg P\) 不可能成立，只能接受 \(P\)。

经典例子是证明 \(\sqrt2\) 不是有理数。假设相反，写成最简分数

\[
\sqrt2=\frac pq,\qquad p,q\in\Z, q\ne0, \gcd(p,q)=1.
\]

平方得 \(p^2=2q^2\)，所以 \(p^2\) 为偶数，由上一节结论知 \(p\) 为偶数，写 \(p=2r\)。代回得

\[
4r^2=2q^2,\qquad q^2=2r^2,
\]

所以 \(q\) 也为偶数。这与 \(p,q\) 互素矛盾。因此 \(\sqrt2\) 无法写成有理数。

``写成最简分数''不是装饰；没有它，推出 \(p,q\) 都偶数并不矛盾。反证法的力量来自某个明确不可同时满足的条件，必须指出矛盾落在哪里。

\subsection{逆否法与反证法怎样区分}

证明 \(P\to Q\) 时：

\begin{itemize}
\tightlist
\item
  逆否法假设 \(\neg Q\)，目标是推出 \(\neg P\)；
\item
  反证法假设 \(P\land\neg Q\)，目标是推出任意矛盾。
\end{itemize}

两者在经典逻辑中都有效，有时写出来非常相似。选择时看哪种叙述能让读者最早知道目标。若能明确推出 \(\neg P\)，逆否法通常更具信息；若否定结论会与全局结构冲突，反证法更自然。

\subsection{分类讨论：把所有可能切成可处理的块}

若对象天然有少数互斥或至少穷尽的类型，可以分别证明。例如证明对任意整数 \(n\)，\(n(n+1)\) 为偶数。每个整数不是偶数就是奇数：

\begin{itemize}
\tightlist
\item
  若 \(n\) 偶，则乘积含偶因子 \(n\)；
\item
  若 \(n\) 奇，则 \(n+1\) 偶，乘积仍含偶因子。
\end{itemize}

两种情况穷尽所有整数，所以结论成立。

分类不必互斥，但必须穷尽；重叠只会产生重复工作，遗漏则使证明不完整。按余数分类时，要明确使用除法算法保证每个整数恰落入某个余数类。

有时应选择更贴合结论的分类。证明绝对值命题时按 \(x\ge0\) 与 \(x<0\) 分；研究奇偶性时按模 2；研究平方末位时可能按模 10。分类的价值不是把证明变长，而是让每一块中关键定义变简单。

\subsection{不要把``没有想到反例''当成反证}

反证法必须从否定假设逻辑推出矛盾。``假设结论不成立，显然不可能''只是重复结论；``我们尝试了很多数都不行''也不是矛盾。真正的反证应能指出哪两条已建立事实不能同时成立。
